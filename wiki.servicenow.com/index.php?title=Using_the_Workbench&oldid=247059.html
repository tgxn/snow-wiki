Financial Management

1 Overview

The Financial Management workbench provides financial administrators with a graphical interface to allocate expenses. When you use the workbench to allocate expenses, you pass through the following stages:

Stages in the allocation process with the workbench

  • Data Definition: Select a fiscal period to work with and make changes to the segments in the IT chart of accounts. You can also perform maintenance activities, such as deleting old data and backing up your cost model.
  • Data Cleansing: Remove unnecessary financial expenses and put similar expenses in the same segment. This cleansing stage gives you the opportunity to tidy up expenses in the general ledger.
  • Bucketing: Group related expenses into customizable buckets.
  • Allocation Setup: Assign the buckets to accounts and segments that comprise the IT chart of accounts, and create rules that govern how segments relate to each other.
  • Review: Review the expense assignments you made, and then run the allocation engine to allocate expenses.

2 Browser Requirements

If you are using Internet Explorer, use version 11 or later to use the workbench. You can also use any of the other generally supported web browsers.
