1 Domain visibility performance improvement

Performance was improved when determining domain visibility for a user.

2 Property to prevent record updates when using blue arrows

A property, glide.ui.update_on_iterate, can be created and set to false to prevent records from being updated when the blue navigation arrows are clicked.

3 Property to specify width of main content frame

A property, css.nav.menu.width.right, can be created to specify the width of the main content frame. Value should be a valid CSS for width (e.g., 82% or 980px).

4 The Database Images plugin is installed

All instances receive the Database Images plugin to store images in the database.

5 Department dropdown not visible by default for Knowledge Management Internationalization plugin

The Department dropdown can be set visible by the glide.knowman.search.show_department property with the following description: 

Display department input in Knowledge search header

6 Condition and Roles fields added to kb_navons table

Navigation Add-ons can now be restricted by conditions and roles, to control when they are displayed.

7 Navigation Add-ons moved to Advanced Search options

Knowledge Base Navigation Add-ons have been moved to the Advanced Search options of the Knowledge search header.

8 report_home.do converted to a UI Page

The report portal, report_home.do, has been converted to a UI Page and can thus be secured with an ACL.

9 Property to specify valid attachment file types

The glide.attachment.extensions property specifies valid attachment file types. If empty, all file types can be uploaded as attachments.

10 Property to allow extended fields in base lists

The glide.ui.list.allow_extended_fields property allows fields from extended tables to be added to (and filtered on) base table lists. e.g., it will allow the caller_id field from the incident table to be added to a task list, like the My Work list. 

Allow base table lists (task, cmdb_ci, etc.) to include extended table fields (incident_state, os_version, etc.), and allow filtering on extended table fields

11 Table cleaner entry added for ecc_agent_metrics table

A table cleaner entry added for ecc_agent_metrics table, with a default value of 30 days.

12 Property to allow URL images to be added to an HTML field

The glide.ui.html.image.allow_url property, with a default value of false, specifies whether the URL option appears when adding an image to an HTML field. 

Show URL option for HTML field images

13 Improved look of Project Management Gantt charts

As part of an ongoing effort to improve the look and feel of Project Management Gantt charts, Gantt chart labels were moved from the left side of the chart to under each bar, resulting in less horizontal scrolling.
