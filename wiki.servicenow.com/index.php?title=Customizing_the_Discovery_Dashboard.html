Customizing the Discovery Dashboard

1 Overview

You can customize the Discovery dashboard by adding or removing gauges. You can also change the behavior of some existing gauges.

2 Adding a Gauge to the Discovery Dashboard

You must have the discovery_admin role to edit the Discovery dashboard

  1. Navigate to Discovery > Dashboard.
  2. Click the + icon in the upper left corner of the dashboard.
  3. Select a gauge that you want to add to the dashboard.
  4. Select a location to place the gauge using one of the Add here choices.
  5. Click the X in the upper right corner of the gauge box to close the gauge choices pane.
The default gauges are found in the following locations:
Gauge Gauge Path
Active Discovery Status Gauges > Discovery Status
Newly Discovered Devices (Last 7 Days) Gauges > Hardware > Newly Discovered Devices (Last 7 Days)
Total Discovered Devices (Last 30 Days) Gauges > Hardware > Total Discovered Devices (Last 30 Days)
Unrefreshed Devices (Beyond Last 30 Days) Gauges > Hardware > Unrefreshed Devices (Beyond Last 30 Days)
Newly Discovered Applications (Last 7 Days) Gauges > Application > Newly Discovered Applications (Last 7 Days)
Total Discovered Applications (Last 30 Days) Gauges > Application > Total Discovered Applications (Last 30 Days)
Unrefreshed Applications (Beyond Last 30 Days) Gauges > Application > Unrefreshed Applications (Beyond Last 30 Days)
Discovery Errors (7 Days) Gauges > Discovery Log > Discovery Error
Unused Credentials Gauges > Credentials > Unused Credentials
Upcoming Discovery Schedules Timeline > Upcoming Discovery Schedules

3 Customizing a Gauge on the Discovery Dashboard

You can customize the gauges on the dashboard if you have the discovery_admin role.

  1. Click anywhere inside the gauge.
  2. Click the pencil icon
    A pencil icon appears in the upper right corner of the gauge if you have permission to change the gauge.
  3. Customize the gauge on this report page.
  4. Click Save.
  5. Click the Discovery > Dashboard to see the changes.
