Scripting in Scoped Applications

1 Overview

The scoped GlideElementDescriptor class provides information about individual fields.

The scoped GlideElementDescriptor API is available starting with the Fuji patch 2 release.

2 Method Summary

Return Type Method
String getInternalType()
Returns the field's data type.
String getLabel()
Returns the label.
int getLength()
Returns the field length.
String getName()
Returns the field name.

3 Constructors

There is no constructor for this class. Use the GlideElement getED() method to obtain a GlideElementDescriptor object.

4 Method Details

4.1 getInternalType()

Returns the field's data type.

Parameters:

  • None

Returns:

String - the field's data type.

Example: 

var grInc = new GlideRecord('incident');
grInc.query('priority', '1');

var field = grInc.getElement('priority');
var ed = field.getED();

var isEdge = ed.getInternalType();
gs.info(isEdge);

Output: 

integer
