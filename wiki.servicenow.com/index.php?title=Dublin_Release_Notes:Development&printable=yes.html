Development & Integrations Enhancements

The Dublin release provides features for extending your ServiceNow instance, building custom applications, and taking advantage of advanced functionality.


Click to sort the features list:

Team Development

Enables teams to work in parallel on separate ServiceNow instances while sharing code and resolving collisions throughout the development life cycle.


Automatic for all instances.

Application Creation

Organizes activated plugins by application for better visibility into installed components. Includes choice lists and homepages in applications. Enhances ability to restore deleted records.


Automatic for all instances.

JavaScript Debugger

Provides troubleshooting and debugging information on business rules and script includes.


Admin can activate the JavaScript Debugging plugin.

Field Watcher

Provides troubleshooting and debugging information for actions on form fields.


Automatic for all instances.

Mobile Client APIs

Enhances Client APIs to support scripting for mobile devices.


Automatic for all instances.

Update Sets

Improves usability and enhances error messaging for update set transfer, preview, and commit. Provides ability to revert database changes. Provides new capabilities for merging update sets.


Automatic for all instances.

Table Administration

Simplifies table administration options, such as creating service catalog record producers and enabling document feeds for a table.


Automatic for all instances.

Microsoft SCCM 2012

Provides support for integration with Microsoft System Center 2012 Configuration Manager.


Admin can activate the Integration - Microsoft SCCM 2012 plugin.

Versions

Improves performance and enhances usability for determining the current version and the origin of versions. Supports team development.


Automatic for all instances.

Nested Interval Utilities

Provides an API for managing the tree structure of the Planned task table.


Automatic for all instances.

JSONv2 Web Service

Improves overall JSON performance with a new JSONv2 Web Service processor that utilizes streaming and Java libraries for JSON parsing.


Automatic for all instances.

Web Services Security

Supports multiple security profiles and the ability to exclude service accounts from security requirements. Restricts access to certain protocols.


Admin can activate the Enhanced Web Service Provider - Common plugin.

LDAP

Tests LDAP connectivity automatically. Allows LDAP connections through a MID server.


Automatic for all instances.

Single Sign-On

Provides ability to force the identity provider's single sign-on service to receive a signed AuthnRequest.


Automatic for all instances.

Web Proxy

Allows administrators to specify a list of URLs or URL patterns that bypass the proxy server.


Automatic for all instances.

GlideDateTime API

Provides the ability to set the month and the year in the GlideDateTime class.


Automatic for all instances.

ODBC Driver

Improves performance for large query results and respects character limits set in the dictionary.


Automatic for all instances.
