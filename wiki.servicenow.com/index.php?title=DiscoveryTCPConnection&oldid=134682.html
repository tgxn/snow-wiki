DiscoveryTCPConnection

From ServiceNow Wiki
DiscoveryTCPConnection

Related Topics

1 Overview

Represents a TCP connection belonging to a process, providing methods for managing Discovery TCP/IP connections.

1.1 Where To Use

Use this script include wherever you need to manage Discovery TCP connections.

2 Method Summary

Method Summary Description
initialize() Called by the Prototype JavaScript Framework during object creation.
setConnection(connectStr) Sets up the Discovery TCP connection.
getHostPort(hostPort) Gets the IP address and port number for the specified port.
setPID(pid) Sets the process identifier (PID) for the application.
getPID() Gets the process identifier (PID) of the current application.
setCommand(cmd) Sets the connection command for the current process.
getCommand() Gets the connection command of the current process.
setFrom(hostname, portname) Sets the hostname and port of the From Host.
getFromHost() Gets the hostname of the From Host.
getFromPort() Gets the port number of the From Host.
setTo(hostname, portname) Sets the hostname and port number of the To Host.
getToHost() Gets the hostname of the To Host.
getToPort() Gets the port number of the To Host.
isListening() Determines if the current device is listening.
setListening(b) Sets a listener for the current device.
isValid() Determines if there is a from host and from port.
isValidConnection() Determines if there is a valid connection with the From host and port.
toString() Gets the current connection command and connection string in string format.

3.3 getHostPort(hostPort)

Gets the IP address and port number for the specified port by parsing the input string and returning a String[] array with 0 as the host and 1 as the port. The host/port combo is returned in this format: host:port

3.3.1 Input Fields

Parameters:

  • hostPort - The port of the host.

3.3.2 Output Fields

Returns: The host IP address and port number.
