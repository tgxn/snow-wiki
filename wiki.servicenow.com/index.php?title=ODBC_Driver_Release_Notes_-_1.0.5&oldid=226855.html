1 Overview

This release of the ODBC driver provides the following fixes and is recommended for all versions of the ServiceNow platform. The previous version of the ODBC driver was 2013-09-06-1413.

2 Fixed in this Release

The following issues were fixed in this version of the ODBC driver since the last release.


Number Details
PRB582356 Joins fail using OR (AND) statements, but works correctly when queried in MySQL.
PRB589370 When the Aggregate Web Services plugin is activated, queries with a scalar function in the WHERE clause return an error.
PRB590797 When querying in ODBC for count(*), you get the error [DataDirect][ODBC OpenAccess SDK driver][OpenAccess SDK SQL Engine]Integrity constraint violation. NULL value specified for NON-NULLABLE column: COUNT(*).
PRB579764 Select statement fails with Java errors.
PRB586776 Using the "greater than" character in queries returns incorrect results when using ODBC.
Click the plus for steps to reproduce
  1. Hook up an ODBC driver to connect to an OOB instance running on glide-berlin-05-06-2012__patch8-hotfix1-04-22-2013
  2. Call the following query:
    SELECT count(*) FROM metric_instance WHERE start > end

Expected Outcome: the ODBC results should match when the query is ran from scripts background. Actual Outcome: the ODBC results do not match when the query is ran from scrips background.
