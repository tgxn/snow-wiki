1 Overview

This page describes methods for displaying a dialog in the current window and frame. For information on how to use GlideDialogWindow, including use cases, see Displaying a Custom Dialog.

1.1 Where To Use

Use these methods in scripts anywhere that you can use a client-side JavaScript. These methods are most often called from a UI Action with the "Client" check box selected.


2 Method Summary

Return Type Method
void destroy()
Closes the dialog window.
void setTitle(String title)
Sets the title of the dialog window.
void setSize(int w, int h)
Sets the size of the dialog window.
void setPreference(String name, String value)
Sets a given dialog window property to a specified value.
void render()
Renders the dialog window.

3 Constructor

3.1 GlideDialogWindow(String id)

Constructor to create a new dialog window object.

Parameters:

  • id - (String) the name of the UI page to load into the dialog window.

Example: 

var gdw = new GlideDialogWindow('show_list');

4 Method Detail

4.1 destroy()

Closes the dialog window.

Parameters:

  • None

Returns:

void

Example: 

//Destroy the current dialog window.
GlideDialogWindow.get().destroy();

4.2 setTitle(String title)

Sets the title of the dialog window.

Parameters:

  • title - (String) the title for the current window.

Returns:

void

Example: 

var gdw = new GlideDialogWindow('show_list');
gdw.setTitle('test');

4.3 setSize(int w, int h)

Sets the size of the dialog window. If you do not pass width and height parameters, a default size is used.

Parameters:

  • w - (int) the width of the dialog window.
  • h - (int) the height of the dialog window.
void

Example: 

var gdw = new GlideDialogWindow('show_list');
gdw.setSize(750,300);

4.4 setPreference(String name, String value)

Sets a given window property to a specified value. Any window property can be set using this method.

Parameters:

  • name - (String) the window property to set.
  • value - (String) the value for the window property.

Returns:

void

Example: 

var gdw = new GlideDialogWindow('show_list');
gdw.setTitle('Test');
gdw.setSize(750,300);
gdw.setPreference('table', 'u_test_list');
gdw.setPreference('title', 'A New Title');

4.5 render()

Renders the dialog window.

Parameters:

  • None

Returns:

void

Example: 

var gdw = new GlideDialogWindow('show_list');
gdw.setTitle('Test');
gdw.setSize(750,300);
gdw.setPreference('table', 'u_test_list');
gdw.setPreference('title', 'A New Title');
gdw.render();
