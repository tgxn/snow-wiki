Overview

ServiceNow provides the ability to customize your instance using JavaScript based on Mozilla Rhino and the ServiceNow Glide classes. Jelly is also available to turn XML into HTML. It is recommended that you be familiar with JavaScript coding before you begin customizing ServiceNow, and with Jelly if you intend to deploy Jelly scripts. The following topics provide general information, procedures, and contexts for scripting in ServiceNow.
