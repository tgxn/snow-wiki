Overview

This page lists release notes for software versions. To sort a table in descending or ascending order, click the sorting icon in the column heading.

Feature releases contain new functionality and fixes to existing functionality. Patch releases and hot fixes provide problem fixes and are released as needed. For more information about the ServiceNow release cycle, see Upgrades and the Release Cycle. For more information about how to upgrade your instance, see Upgrade to Istanbul or Upgrade to Jakarta.
