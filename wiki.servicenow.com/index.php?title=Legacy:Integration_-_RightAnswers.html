Legacy:Integration - RightAnswers

 This plugin has been retired and is no longer available for new activations (since Eureka). It has been replaced by a certified partner integration.

This page applies only to existing users of the RightAnswers plugin.

1 Overview

Integrate RightAnswers knowledge searches from within an Incident. Integration is via a Search RightAnswers application module that links to your RightAnswers hosted instance for searches. A link is also provided for the Incident Short Description field to search the current short description text using RightAnswers. This integration requires a subscription to RightAnswers.

2 Requirements

  • Activate the RightAnswers (com.snc.integration.right_answers) plugin (prior to the Eureka release only).
Click the plus to expand instructions for requesting a plugin.
  1. Navigate to [HI].
  2. Click Service Catalog.
  3. Click Request Plugin Activation.
    • [Required] In Target Instance, select the instance on which to activate the plugin.
    • [Required] In Plugin Name, enter the name of the plugin to activate.
    • [Optional] In Date and time would you like the plugin to be enabled?, specify a date and time at least 12 hours in the future. Leave this field empty if you want the plugin activated as soon as possible.
    Note: Plugins are generally activated during business hours in the Pacific time zone, but can be scheduled for a different time with advance notice.
    • [Optional] In Reason/Comments, add any information that would be helpful for the ServiceNow technical support engineer activating the plugin.
  4. Click Submit.
  • Enable the "Search RightAnswers" application module
  • Enable custom search of Task Short Description to use RightAnswers

2.1 Search RightAnswers Application Module

After the plugin is activated, edit the Service Desk application and activate the Search For Help module. You will see the following module under the Service Desk application.


Edit the URL in this module to refer to the URL linking to your hosted RightAnswers instance.

3 Search RightAnswers using text from Incident Short Description

Enable searching RightAnswers using text from Task (Incident, Problem, Change, Ticket) short_description field.

Edit the dictionary definition of the Task table's short_description field and add the following attribute: 

knowledge_custom=customKnowledgeSearch

Next, modify the Client Script RightAnswers Search and update the URL to point to your RightAnswers search URL.

The knowledge search button to the right of the short_description field will now enable you to search in RightAnswers.



Clicking on the button pulls up the RightAnswers search screen, or login.
