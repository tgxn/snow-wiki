Winter 2009 Description of Functionality

From ServiceNow Wiki
Home > Release Notes > Legacy Versions > Winter 2009 Description of Functionality
Jump to: navigation, search

Our Winter 2009 release contains the following new or improved functionality:

  • Project Management - plan, organize and execute projects and resources
  • Administration enhancements
    • JavaScript debug window
    • Global UI Scripts
    • List Calculations enhanced to work with duration fields
  • UI enhancements
    • Definable Homepage Layouts - Layout control of homepages
    • Editing Homepage Content
    • Global Text Search
    • Tabbed Forms
    • Encrypted Fields and Attachments
    • Speedier list rendering
  • Service Catalog enhancements
    • Service Catalog variable pricing
    • Optional price display for catalog items
    • Intermediate checkout form (for 2 step checkout) customizable with a UI page
    • Advanced reference qualifier support for Lookup Select Box variables
    • Consultants or customers can extend the catalog pricing model
    • Automatically display your most popular catalog requests
    • Display a catalog item in more than one category
    • Informational (as opposed to orderable) catalog items
    • Simpler content listing on catalog homepage
