Process Application Enhancements

The Dublin release provides new and enhanced process applications for delivering business services.


Click to sort the features list:

Vendor Performance

Drives value from vendors by evaluating and comparing them across common criteria. Provides unique graphical formats for viewing key performance information.


Admin can activate the Vendor Performance plugin.

Resource Management

Enables reliable resource forecasting through a real-time, single system of record for all work and resources.


Admin can activate the Resource Management plugin.

HR Service Automation

Tracks and manages interactions and requests between employees and human resource staff.


Available by request.

Incident Alert Management

Manages crises by consolidating all communications in a dashboard, bringing together all involved users for quick resolutions. Integrates with Notify for conference call and SMS support.


Admin can activate the Incident Alert Management plugin.

Vendor Ticketing

Tracks vendor tickets with new fields on incidents. Tracks credit owed to an organization by a vendor from a breached business service contract.


Admin can activate the Vendor Ticketing plugin.

Compliance

Enables administrators to certify CMDB data for correctness and then fix any deficiencies in the data. Offers several levels of certification to meet business needs.


Admin can activate any of these plugins:

  • Data Certification
  • Desired State
  • Architecture Compliance
  • IT Governance Risk and Compliance

Architecture Compliance

Examines the physical attributes of configuration items to determine if they match the expected state. Assigns tasks to fix any discrepancies found.


Admin can activate the Architecture Compliance plugin.

Data Certification

Includes certification filter version control. Restricts user selection in schedule definitions by role. Offers a new Overview module. Allows certification admins to see audit results.


Automatic for instances running Data Certification.

Desired State

Examines the physical attributes and relationships of configuration items to determine if they match a desired state. Assigns tasks to fix any discrepancies found.


Admin can activate the Desired State plugin.

IT Governance Risk and Compliance

Supports the use of certification filters and templates in control test definitions. Enables customers to audit vendors for non-disclosure agreements.


Automatic for instances running IT Governance Risk and Compliance.

Structured Problem Analysis

Supports the Kepner-Tregoe problem solving methodology to help investigate the root cause of a problem.


Admin can activate the Kepner Tregoe - Structured Problem Analysis plugin.

Personal Tasks

Provides the capability to create, view, and manage your own tasks.


Admin can activate the Personal Tasks plugin.

Service Desk Call

Allows service desk staff to quickly collect basic information before transferring call data to incidents, problems, changes, or service requests.


Admin can activate the Service Desk Call plugin.

Contract Management

Enables creation of contracts for purchase agreements and non-disclosure agreements (NDAs).


Automatic for all instances.

Procurement

Adds a procurement administrator role. Improves the receiving process. Populates additional fields on asset records created from purchase order items.


Automatic for instances running Procurement.

Asset Management

Enables manual creation of an asset from a model category record. Provides several new asset reports. Adds ability to set asset depreciation to start at future dates.


Automatic for all instances.

Product Catalog

Enables verification of configuration items before adding them as assets. Enables direct linking of software vendor items to the software catalog.


Automatic for all instances.

Project Management

Links change requests to project records, provides a default schedule for projects, rolls down state changes to child tasks, and makes rollup tasks read only.


Admin can activate the Project Management plugin.

Service Catalog

Simplifies creation of record producers for a table. Adds support for access from mobile devices.


Automatic for all instances.

Software Asset Management

Enhances software count functionality. Improves usability and controls for creating new models and counters.


Automatic for instances running Software Asset Management.

Cost Management

Sets contract rate cards to end when contracts end. Enhances security for the Overview module.


Automatic for instances running IT Cost Management.

Knowledge Management

Adds support for knowledge base access from mobile devices.


Automatic for all instances.

Work Management

Allows drag and drop assignment of work order tasks in visual dispatch. Enhances security for the Overview module.


Automatic for instances running Work Management.
