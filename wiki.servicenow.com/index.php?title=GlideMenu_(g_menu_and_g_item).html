1 Overview

GlideMenu.js is the JavaScript class used to customize UI Context Menu items.

  • g_menu is the UI Context Menu that is about to be shown. The onShow script can make changes to the appearance of the menu before it is displayed using these methods.
  • g_item is the current UI Context Menu item that is about to be shown. It is used in several of the g_menu methods to specify an item.


1.1 Where To Use

These methods are used in UI Context Menus, in the onShow scripts.

3 Methods

3.1 clearImage

void clearImage(GlideMenu item)

Clears the image for an item.
Parameters:
GlideMenu item - specifies which item will have its image removed from display.
Returns:
void
Example: 
g_menu.clearImage(g_item);

3.2 clearSelected

void clearSelected()

Clears any selection images (e.g. a check mark) from the items in the menu.
Returns:
void

3.3 getItem

GlideMenu item getItem(String itemID)

Returns a menu item by item ID. It can be necessary to find an item in a menu so that it can be changed before being displayed. Each menu item may be assigned a unique ID when the menu item is created (either from a UI Context Menu entry or from the addAction method in the Dynamic Script Action).
Parameters:
String itemID - specifies the item to be returned.
Returns:
GlideMenu item - menu item.

3.4 setDisabled

void setDisabled(GlideMenu item)

Disables a menu item so that it cannot be selected. The disabled menu item is displayed in a lighter color (grayed out) to indicate it is disabled.
Parameters:
GlideMenu item - specifies the item to be disabled.
Returns:
void
Example: 
g_menu.setDisabled(g_item);


3.5 setEnabled

void setEnabled(GlideMenu item)

Enables a menu item.
Parameters:
GlideMenu item - specifies the item to be enabled.
Returns:
void
Example: 
g_menu.setEnabled(g_item);


3.6 setHidden

void setHidden(GlideMenu item)

Hides the item.
Returns:
void
Example: 
g_menu.setHidden(g_item);
Note: When hiding menu items, the separator bars are not adjusted so it is possible to end up with the menu showing two separators in a row.


3.7 setImage

void setImage(GlideMenu item, String imgSrc)

Sets an image for an item.
Parameters:
GlideMenu item - specifies the item to have the image displayed by it.
String imgSrc - the image to attach to the menu item.
Returns:
void
Example: 
g_menu.setImage(g_item, 'images/checked.gifx');


3.8 setLabel

void setLabel(GlideMenu item, HTMLString label)

Sets the display label for a menu item, which may contain HTML.
Parameters:
GlideMenu item - specifies the item to add the label.
HTMLString label - label to display for a menu item (may contain HTML).
Returns:
void
Example: 
g_menu.setLabel(g_item, "This is a new label");


3.9 setVisible

void setVisible(GlideMenu item)

Displays the item.
Parameters:
GlideMenu item - specifies the item to be displayed.
Returns:
void
Example: 
g_menu.setVisible(g_item);
