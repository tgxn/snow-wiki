1 Overview

The following updates fix issues after the Spring 2010 release and are available on an as-needed basis.


2 PRB278188

  • Problem: An illegal XML character, such as an ampersand (&), in the title of a piece of Welcome Page Content caused an error during rendering of the login page and prevented users from logging into the system until the welcome page content was corrected.
  • Symptom: The login page failed to render, preventing users from logging in.


3 PRB278231

  • Problem: On a Catalog Item with a price of type Multi, deleting all child prices resulted in a server side-error (null pointer exception (NPE)) and the inability to render that catalog item or any catalog home page containing it.
  • Symptom: Items in the Service Catalog might not render properly.


4 PRB270118

  • Problem: The CI identification process failed to stop Discovery when it found duplicate CIs for a device in the CMDB.
  • Symptom: Discovery created a new CI, although a duplicate of the device existed in the CMDB.


5 PRB278187

  • Problem: Context menu code could not process a single quote in the arguments list. For example, if you had a related list labeled Day's Ago and clicked Personalize list control, the list control would not function.
  • Symptom: The Personalize > List Control command on a Related List produced a browser error.


6 PRB278866

  • Problem: The Add New Item UI Action button in an existing catalog Request produced an error.
  • Symptom: When a user clicked the Add New Item button, the wrong screen was displayed. When the user tried to cancel the operation, the following error appeared: Glide.Window.locate:window not found.


7 PRB278915

  • Problem: The link to Portfolio Management was not removed from the Project form in Project Management v2 when Portfolio Management was removed from the Spring 2010 release.
  • Symptom: The Portfolio Management Related Link that appeared in the Project form was broken.


8 PRB278916

  • Problem: The SLA Plugin contained obsolete SLA timeline functionality.
  • Symptom: The link to the SLA Timeline module under the Service Level Management application did not work.


9 PRB281033

  • Problem: The Personalize Dictionary operation failed on an extended task table field if a Documentation entry existed for that field.
  • Symptom: When a user selected Personalize Dictionary on a field such as Number in an Incident record, the operation failed to find the dictionary item.


10 PRB279107

  • Problem: Derived, multi-line text fields did not save their contents properly, due to an error in the change detection logic.
  • Symptom: The text disappeared from a derived multi-line text field, such as Close notes in a task record, when the record was saved.


11 PRB279044

  • Problem: The WSDL definition did not match the output of the SOAP response.
  • Symptom: The order of the attributes in the SOAP response was not consistent with the WSDL definition.


12 PRB282194

  • Problem: When using the displayvalue=all URL parameter, dv_ fields were not always output in the response, and were not always specified in the WSDL.
  • Symptom: The WSDL response was sometimes missing dv_ fields.


13 PRB284351

  • Problem: A localized language was cached incorrectly for Related List labels.
  • Symptom: Related List labels might be in the wrong language for the user.
