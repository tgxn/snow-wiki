1 Overview

This optional plugin contains Software Asset Management improvements that include:

  • Checking contracts to determine if they are approaching expiration
  • Generating events when a contract is approaching expiration
  • Checking software licenses for near and out of compliance conditions
  • Automatic management of install counts from links to the CMDB
  • Ability to group several software packages under a single license

2 Enabling the Software Asset Management Improvements

The Software Asset Management Extensions plugin is enabled automatically on all new instances of ServiceNow.

3 Contract Expiration Notifications

Detailed information about checking and notification of approaching contract expiration can be found at the topic Managing Contract Expirations.

4 Software License Counting, Package Grouping, and Compliance Checking

See Software License Counts and Compliance Checking for detailed information about:

  • Defining and tracking software license counts.
  • Checking and notification of software license compliance conditions

4.1 Scheduled Job

The plugin installs a scheduled job called Check contracts and license compliance that uses the ConditionChecks script include. This job checks for matching conditions, such as for contract expirations and license compliance, defined in the Condition Check [cmn_condition_check] table.
