1 Overview

Zing is the text indexing and search engine that performs all searches in ServiceNow. It also provides extended administration options and configuration capabilities.

1.1 Basics

  • Created entirely with ServiceNow code
  • Implemented purely within the relational model
  • Sharded Index, Parallel Query

1.2 Features

  • Fast results with search engine logic
  • Results scoring system yields relevant results
  • Usable for search users
    • Type-ahead suggestions
      Type-ahead search

    • "Did you mean?" suggestions for the global search
      Did you mean? suggestions

  • Administrative options
    • Control field level weighting
    • Configure the global stop words
    • Configure stop words for each index

1.3 Search Quality

Emphasis is on scoring:

  • Frequency points
  • Proximity points
  • Field-level scoring (title, metadata, and short description, weighting fields)

1.4 Administration

Administrators and users with the ts_admin role can configure Zing. For more guidance on administration, see Administering Zing Text Search.
