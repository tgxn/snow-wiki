Application Development & Integrations Enhancements

The Eureka release provides features for extending an instance, building custom applications, and taking advantage of advanced functionality.


Click to sort the features list:

Service Creator

Empowers line-of-business service owners to build lightweight, custom request-fulfill applications within an IT-governed framework.


Requires the Service Creator plugin.

Catalog Item Designer

Provides a simplified interface for creating, maintaining, and publishing items, allowing these functions to be delegated to non-admin users.


Automatic for all instances.

Multiple Catalogs

Allows users to group and administer catalog items into multiple service catalogs by business area, organization, geography, or target user base.


Automatic for all instances.

Form Designer

Allows administrators to create forms and tables with simple, visual controls.


Automatic for new instances. Requires the NG form designer plugin on upgraded instances.

Team Development

Enables administrators to require code review of all pushed changes and to create exclusion policies that ignore changes to specific records. Allows developers to back out local changes.


Automatic for all instances.

REST API

Provides REST APIs with detailed response codes, header information, pagination support, and streaming data on requests.


Automatic for all instances.

Email Notifications

Offers a rich HTML editor for creating notifications with simplified, reusable mail scripts. Provides easier debugging of complex notifications and filtering of notification devices using a schedule.


Automatic for all instances.

LDAP

Allows the LDAP listener to monitor LDAP changes directly from a MID Server. Allows the LDAP monitor to report on the health of LDAP servers and listeners to improve diagnostics and maintenance.


Automatic for all instances.

Multi-SSO and InCommon Federated Authentication

Allows organizations to use several SSO Identity Providers (IdPs) to manage external authentication requirements. Supports InCommon federated identity management.


Requires the Integration - Multiple Provider Single Sign-On Installer plugin.

Okta SSO Integration

Integrates Okta Single-Sign-on with ServiceNow.


Requires the SSO provided by Okta, Inc. plugin.

GlideDateTime

Offers new time zone-specific methods for modifying or retrieving data from GlideDateTime objects. Deprecates methods that do not handle time zones.


Automatic for all instances.

Scriptable Order Guides

Executes catalog order guides directly from server scripts or workflow activities.


Automatic for all instances.

ACL Debugger

Provides a redesigned ACL debug output form that allows administrators to see which ACL checks passed or failed.


Automatic for all instances.

Application Creation

Stores a user-defined version number for use when creating and publishing applications.


Automatic for all instances.

JavaScript Debug Window

Makes the JavaScript debug icon and window available to an administrator who is impersonating an ITIL or self-service user.


Automatic for all instances.

Update Sets

Allows update sets to store choice list options as changes to an extended table rather than changes to a parent table.


Automatic for all instances.
