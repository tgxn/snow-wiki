1 Overview

To improve the usability of the Wiki, we've updated the content organization and provided new navigational tools and information to help you find what you're looking for. As always, feedback is appreciated. To discuss or rate a page, use the Wiki feedback tools.

2 Categories

Documentation topics have been organized into categories. A category is a way to group pages that provides several navigational tools, including breadcrumbs and category pages.

2.1 Breadcrumbs

Categories generate breadcrumbs at the bottom of each page for every place in the structure that a page is listed. For example, if you're looking for an email notification about an incident, you'll find it in the Notification Administration and the Incident Management categories.

Category Breadcrumbs

 Note: Breadcrumbs at the top of the page represent one category to which a page belongs. To see all category paths, scroll to the bottom of the page.

2.2 Category Page

A category page lists the subcategories and pages in a particular category and provides a tree to aid in topic navigation. To view a category page, click the category name at the bottom of a page. To view the main category page, navigate to Category:Contents.
