Creating Marketing Groups

1 Overview

If you have the admin or user_admin role, you can set up your groups and assign the necessary roles and users. The users in the group inherit the roles of the group, so you do not have to assign roles to each user separately.

A good practice is to create as many marketing groups as needed in your organization. For example, you can create a group for each geographic location or for each function, assign the necessary users to those groups, and then assign the marketing_agent role to those groups. It is also a good practice to create one group for marketing administrators and assign the marketing admin role to this group only.

2 Creating a Marketing Group

To create a marketing group:

  1. Navigate to User Administration > Groups.
  2. Click New.
  3. Fill in the fields. See Creating Groups for an explanation of each field.
  4. Click the lock icon beside the Type field. If the field is not visible, configure the form to add it.
    The Type field expands.
  5. Click the reference lookup icon (), and select the Marketing type.
  6. Right-click the form header and select Save.
  7. Add the marketing_admin or marketing_agent role to the Roles related list.
  8. Add users to the Group Members related list.
  9. Click Update.
