Service Management & Business Management Enhancements

The Fuji release provides new and enhanced features for delivering and managing business services.


Click to sort the features list:

Creating Service Management Applications from a Template

Provides application developers with basic, intermediate, and advanced templates to quickly create service management applications of varying complexity. Each template offers default processes for request fulfillment but can be configured to provide additional options, such as managing documents and assigning work based on location or skill. Each template also creates a service catalog and knowledge base in the application.


Requires the Service Management Core plugin.

Demand Management - Demand Workbench

Provides an interactive page for assessing and adjusting demands from the Demands list or a bubble chart indicating combinations of risk and value levels.


Requires the Demand Management plugin.

Facilities Service Management

Allows users to create trackable and auditable facilities requests. Enables administrators to configure a request-driven or task-driven workflow to handle jobs that require one or multiple tasks. An optional plugin allows users to launch the Facilities application from a CMS portal.


Requires the Service Management Core plugin.

Finance Service Management

Allows users to create trackable and auditable finance requests. Enables administrators to configure a request-driven or task-driven workflow to handle jobs that require one or multiple tasks. An optional plugin allows users to launch the Finance application from a CMS portal.


Requires the Service Management Core plugin.

IT Financial Management

Enables IT managers to perform IT costing across the service hierarchy and view reports on allocated expenses.


Requires the IT Finance plugin.

IT Financial Management - CIO Dashboard

Provides IT management with an overview of IT-related finances, operations, and projects.


Requires the IT Finance plugin.

IT Financial Management - Workbench

Provides a user-friendly interface to set up an IT cost model and allocate expenses across the service hierarchy.


Requires the IT Finance plugin.

IT GRC - Attestation

Allows administrators to create and administer surveys to assess how well governance, risk, and compliance (GRC) policies are enforced.


Requires the IT Governance, Risk and Compliance plugin.

IT GRC - Import UCF Content

Enables administrators to compare updated UCF documents to currently used documents, and require approval for importing selected documents into the GRC application for use in control tests.


Requires the IT Governance, Risk and Compliance plugin.

IT GRC - Reports

Provides portals for reports on GRC compliance, controls, and audits that are viewable by users with compliance roles. Includes reports about UCF import activity, remediation tasks, approval requests, and past due attestations.


Requires the IT Governance, Risk and Compliance plugin.

Knowledge Management - Contextual Search

Adds a configurable search mechanism on record producers and forms, presenting users with existing knowledge that may resolve issues. Provided by default for incidents to help incident deflection.


Automatic for all instances.

Knowledge Management - Multiple Knowledge Bases

Allows administrators to create multiple knowledge bases with unique life cycle workflows, user criteria, category structures, and management assignments.


Automatic for all instances.

Knowledge Management - User Criteria

Enables administrators to create flexible and reusable user criteria records that control access to knowledge bases.


Automatic for all instances.

Legal Service Management

Allows users to create trackable legal requests. Enables administrators to configure a request-driven or task-driven workflow to handle jobs that require one or multiple tasks. An optional plugin allows users to launch the Legal application from a CMS portal.


Requires the Service Management Core plugin.

Marketing Service Management

Allows users to create trackable marketing requests. Consolidates information gathered for requests. Enables administrators to configure a request-driven or task-driven workflow to handle jobs that require one or multiple tasks. An optional plugin allows users to launch the Marketing application from a CMS portal.


Requires the Service Management Core plugin.

Performance Analytics - Breakdown Matrix

Allows users to configure a breakdown matrix that groups information by user-selected categories for more granular analysis.


Automatic for new instances. Requires the Performance Analytics plugin on upgraded instances.

Performance Analytics - New Chart Types and Features

Adds new chart types, settings to configure elements such as the title and border, and the ability to link and share tabs across dashboards.


Automatic for new instances. Requires the Performance Analytics plugin on upgraded instances.

Performance Analytics - Scorecard Features

Provides options in the detailed scorecard to select the chart type and show or hide chart elements such as the trendline and confidence band. Adds the More Info section to provide the description, last update, update frequency, direction, and formula (if applicable) of the chart.


Automatic for new instances. Requires the Performance Analytics plugin on upgraded instances.

Performance Analytics - Target and Threshold Shortcuts

Provides shortcuts to configure targets or add thresholds to indicators within detailed scorecards.


Automatic for new instances. Requires the Performance Analytics plugin on upgraded instances.

Planned Maintenance

Allows field and facilities teams to attach maintenance plans to product models, generating maintenance tasks either on schedule or by usage metric.


Requires the Service Management Core plugin.

Project Portfolio Management - Agile Project Management

Integrates components of Project Portfolio Management (PPM) with SDLC - Scrum Process Pack to enable a project management approach that combines the Waterfall and Scrum methodologies.


Requires the Project Portfolio Suite plugin.

Project Portfolio Management - Project Templates

Allows project managers to create new project templates, save existing projects as templates, and reuse project structure.


Requires the Project Management plugin.

Project Portfolio Management - Project Workbench

Enables project managers and teams to create, plan, monitor, and manage projects on a single page.


Requires the Project Management plugin.

SDLC - Scrum Process Pack - Burndown chart update

Provides an updated look for burn down charts, and adds the ability to drill down to sprint burn down charts from release burn down charts.


Requires the SDLC - Scrum Process Pack plugin.

Service Catalog - Access Controls

Enables administrators to create flexible and reusable user criteria records that restrict access to service catalog items.


Automatic for all instances.

Service Catalog - Configurable Cart Layouts

Enables administrators to configure the layout and behavior of the service catalog cart and order status screens and still receive future updates to these features.


Automatic for all instances.

Test Management

Provides an integrated application for creating test plans and test cases, guiding testers through test execution, monitoring testing processes, evaluating test results, and completing testing sign-off.


Requires the Test Management plugin.

Visual Task Boards

Provides a more streamlined interface and a simplified configuration menu. Enables switching between lane and list views of tasks. Integrates with Collaboration.


Automatic for new instances. Requires the Visual Task Boards plugin on upgraded instances.
