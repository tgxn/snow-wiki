Platform Enhancements

The Eureka release provides new and enhanced features for configuring and managing a ServiceNow instance.


Click to sort the features list:

User Interface

Improves usability and adds live streaming, document tagging, form pane tabbing, and column header filtering with UI14. Automatically enables UI11 for upgrades.


Automatic for all instances.

Surveys

Provides a new survey engine with capabilities to create and send surveys in minutes, add conditional questions, schedule surveys, and limit access to specific users. Displays results on scorecards.


Automatic for all instances.

Assessments

Displays results in bubble charts. Allows users to send triggered assessments, create conditional questions, customize assessment appearance, and retire assessments. Supports additional question types.


Automatic for all instances.

Notify

Delivers bi-directional notifications over SMS, Voice, and Conference Bridge, and is tightly integrated with workflows and business rules.


Requires the Notify plugin.

Business Rules

Offers a more user-friendly form with choice lists for building filter conditions and setting field values and an HTML field for composing information messages without writing a script.


Automatic for all instances.

Reference Qualifiers

Offers new methods for building reference qualifiers, including a simple condition builder, an advanced query or script, and dynamic filtering options.


Automatic for all instances.

Domain Separation

Provides improvements to the domain framework including domain assignment and validation options.


Requires the Domain Support plugin.

Geolocation

Pushes latitude and longitude from mobile devices to user records. Plans routes and finds accurate travel times using a Google Maps API key. Improves routing and dispatching in Work Management.


Requires the Geolocation plugin.

Mobile UI

Provides related lists, map pages, and UI action redirects in the Smartphone interface.


Automatic for new instances. Requires the Mobile UI plugin on upgraded instances.

State Flows

Offers a new option for controlling task transitions between states, setting field behavior and visibility, and triggering events when states change. Provides default state flows for Work Management.


Requires the State Flows plugin.

Homepages

Offers a new default administrator homepage to provide quick access to core system settings. Allows users to add service catalog categories to homepages.


Automatic for all instances.

HTML Sanitizer

Cleans up HTML markup in HTML fields to remove unwanted code. Allows administrators to configure standard policies for which tags are allowed at an instance or individual field level.


Automatic for new instances. Requires the HTML Sanitizer plugin on upgraded instances.

Usage Analytics

Provides an updated report for application usage and license compliance, classifying users as requesters, fulfillers, and approvers. Allows administrators to specify a reporting period.


Automatic for all instances.

Context-Sensitive Help

Searches for relevant content on the ServiceNow wiki when no help context is specified.


Automatic for all instances.

Live Feed

Allows the administrator to maintain shared content within a specific domain and establish parent and child relationships to define the content that can be viewed and shared with other users.


Automatic for all instances.

Reporting

Provides a consistent look for on-demand and exported reports. Provides the ability to save reports as PNG or JPG using the report context menu. Enables Report Charting V2 by default.


Automatic for all instances.

Available Plugins

Changes how some plugins are activated, and retires plugins that are no longer supported.


Automatic for all instances.

Company record

Stores values in the latitude and longitude fields as a floating point data type rather than as a string. Preserves existing data in new fields and converts to new data type where possible.


Automatic for all instances.

Location record

Stores latitude and longitude as a floating point data type rather than as a string. Preserves existing data in new fields and converts to new data type where possible.


Automatic for all instances.

On-Form Help

Provides on-form user assistance and improved form design for many important platform features.


Automatic for all instances.

Zing Text Search

Indexes and searches elements and attributes of certain HTML tags, such as link targets and alt text for images.


Automatic for all instances.
