1 Overview

The Assessment application prepares printable scorecards for GRC attestations. A scorecard displays easy-to-interpret results, in which category and question responses are analyzed and current ratings are compared with previous ratings. Users can examine ratings over time, compare question ratings, or compare the ratings of all categories. All ratings are averages for the time range selected. The system dynamically updates a scorecard each time you view it, so the ratings reflect recently completed attestations. Only scored results are displayed in the attestation scorecard.

2 Accessing Scorecards

You can access scorecards from links in a GRC control test instance record or an assessment metric type record. The scorecard link is hidden if there are no results to report. The scorecard contains a header that displays the name of the attestation and a section that displays results or compares the ratings.

2.1 GRC Control Test Instance

Navigate to a control test instance record, using one of the following methods:

  • GRC > Control Tests > All: Open a record from the list.
  • GRC > Administration > Control Test Definitions: Open a control test definition in the list, and then select a record from the Control Test Instance related list.

In the control test instance record, click View Scorecard under Related Links.

2.2 Metric Type Link

  1. Navigate to Assessments > Metric Definition > Types.
  2. In the list of Assessment Metric Types, remove the Evaluation method = Assessment condition.


  3. Select an attestation metric type.
  4. Under Related Links, click View Scorecard.

3 Scorecard Views

The attestation scorecard presents results in the following views:

  • Current Results
  • Current Category Results
  • Current Question Results
  • Current Question Ratings
  • Category Results
  • Question Results
  • Average Ratings
  • History

3.1 Current Results

The Current Results view is a bar chart showing the correct responses by percentage for each recipient in the current control test. If the attestation is scored as a Percentage, the normalized value can be any percentage. If the attestation is scored as All or nothing, scores for each recipient are either 0 or 100. The Results view is only available on scorecards accessed from a control test instance.



To view each recipient's correct responses by percentage, point to the colored bar.


3.2 Current Category Results

The Current Category Results view is a stacked bar chart showing attestation results for all questions in a category by respondent count for the current control test. Select a category from the list to display the questions from that category in the chart. This view displays scored responses that use the following data types:

  • Checkbox
  • Choice
  • Likert Scale
  • Numeric scale
  • Template
  • Yes/No



To view details about a specific response, point to the colored bar to display the evaluation, the number of responses, and the percentage it represents of all the responses to that question.


3.3 Current Question Results

The Current Question Results view shows the results for each question in a category for the current control test. You can select to display the results in either a pie chart or a bar chart, depending on the data type. To view the total number of responses for each answer, point to the colored bar or wedge.

3.3.1 Pie chart

The pie chart shows question results for these data types:

  • Checkbox
  • Choice
  • Likert Scale
  • Numeric Scale
  • Number
  • Template
  • Yes/No


3.3.2 Bar chart

A bar chart appears when question results use the Percentage data type. By default, all results for percentage questions use a report range of 20% segments. To configure a report range, navigate to Reports > Administration > Report Ranges.

3.4 Current Question Ratings

The Current Question Ratings view displays the weighted average rating for each scored question in a category for the current control test. Use this view to learn how individual questions affect the overall rating for the category. Select an attestation category from the choice list to display the chart.



To view the effect of each question's ratings on the entire category's ratings, point to the colored bar. The pop-up box shows the percentage of the total ratings represented by each individual question's weighted average.


3.5 Category Results

The Category Results view is a stacked bar chart showing results for all questions in a category by respondent count. The chart shows results for all attestations completed by a specific control test instance. Select a category from the list to display the questions from that category in the chart. This view displays scored responses that use the following data types:

  • Checkbox
  • Choice
  • Likert Scale
  • Numeric Scale
  • Template
  • Yes/No



To view details about a specific response, point to the colored bar to display the evaluation, the number of responses, and the percentage it represents of all the responses to that question.


3.6 Question Results

The Question Results view shows the results for each question in a category for all attestations completed for a specific control test instance. You can select to display the results in either a pie chart or a bar chart, depending on the data type. To view the total number of responses for each answer, point to the colored bar or wedge.

3.6.1 Pie chart

The pie chart shows question results for these data types:

  • Checkbox
  • Choice
  • Likert Scale
  • Numeric Scale
  • Number
  • Template
  • Yes/No


3.6.2 Bar chart

A bar chart appears when question results use the Percentage data type. By default, all results for percentage questions use a report range of 20% segments. To configure a report range, navigate to Reports > Administration > Report Ranges.

3.7 Average Ratings

The Average Ratings view displays the weighted average rating for each scored question in a category. Use this view to learn how individual questions affect the overall rating for the category. Select an attestation category from the choice list to display the chart.



To view the effect of each question's ratings on the entire category's ratings, point to the colored bar. The pop-up box shows the percentage of the total ratings represented by each individual question's weighted average.


3.8 History

The History view compares the current ratings for the categories and their questions with ratings from the previous three years or four quarters. Ratings that have declined are highlighted in red and display negative numbers. Ratings that have improved are highlighted in green with positive numbers. Arrow icons beside the values in the Diff column indicate the trend of the current control test against the previous control test.

Point to a category to display a line chart that shows the rating trend for that category. Click a category to view the Attestation Category form containing the questions.

  • 3 Years: To calculate the current ratings, the system averages the ratings from the trailing twelve month (TTM) period. The Diff column shows the discrepancy between the current ratings and the previous calendar year's ratings.



  • 4 Quarters: Quarterly attestations compare the average rating for each question and category in the current quarter against the average ratings from the previous four quarters. The Diff column shows the discrepancy between the current ratings and the previous quarter's ratings. The column labels count backward, by quarter from the current quarter. For example, if the current quarter is the 3rd quarter of 2013, then the previous quarters appear as 2nd [2013], 1st [2013], 4th [2012], and 3rd [2012]. All four of the previous quarters appear, whether or not there was any data for those quarters.


4 Exporting Scorecards as Images

Click the menu icon () and select either PNG or JPEG as the image download format. When prompted, select Download to save the scorecard image to your local drive.
