1 Overview

UI11 is a new user interface for ServiceNow. It offers more user control for better usability and simplified navigation. UI11 features include:

  • Split screen: view list and form panes at the same time, similar to an email client.
  • Bookmarks: customize the Edge, which is a tool bar with quick links to commonly used features. Users can choose to hide the application navigator and use their bookmarks instead.
  • Flyout windows: open informational windows without navigating away from your working panes.

UI11 is automatically installed for all instances.

UI11

2 Getting Started with UI11

To get started with UI11:

  1. Install the UI11 plugin and ensure the glide.ui11.show_switch_link is set to true (default value). UI11 is automatically installed on new instances. See Administering UI11.
  2. Inform users that a new interface option is available to them.
    All users configure their own UI11 settings. See Using UI11.
