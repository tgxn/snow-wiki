1 Overview

The ServiceNow user interface uses a number of icons in the following sections of ServiceNow screens:

  • Banner Frame
  • Application Navigator
  • The Edge
  • Form View
  • List View
 Note: Common browser icons, such as the Print icon, are omitted unless they have a functionality that is unique in the ServiceNow interface.

Icon Name Description
Search Performs a global text search on the terms entered into the search box.

You can click the down arrow to view a list of recent searches.

For more information, see Using Global Text Search.

Collapse/Expand Buttons These buttons show and hide adjacent areas of the user interface, depending on their location. The and on the right side show and hide part of the banner frame.

Impersonate (Requires admin or impersonator role) Presents the Impersonate User dialog box, which allows the administrator to select a user to impersonate, either from a list of recent impersonations or through a reference to the User [sys_user] table.

For more information, see Impersonating a User.

Elevate Privileges (Requires admin or security_admin role) Presents the Activate an Elevated Privilege dialog box, which allows the administrator to select an elevated role, which will expire at the end of the session.

For more information, see Elevated Privileges.

For more information, see Customizing Homepages.

Refresh picker (Requires admin role) Refreshes the associated picker list. Located under the gear icon.

This icon is displayed for the language picker if a language plugin has been activated. For more information, see Internationalization Support.

This icon is displayed for the theme picker if the ui_theme_changer UI macro is active. For more information, see Enabling the Theme Picker.

Debug (Requires admin role) Opens the JavaScript debug window at the bottom of the screen with the JavaScript Log tab and the Field Watcher and JavaScript Debugger. Located under the gear icon.

3 Application Navigator

3.1 UI15

UI15 Application Navigator

Icon Name Description
Favorites Filter Shows either all modules or selected favorites.

For more information, see Using Application Navigator Favorites.

Menu Lists the following menu options:
  • Roles: similar to the Switch Perspective option in UI11; when a user selects a role, the application navigator is filtered to show only the applications and modules included in that role.
  • Automatically Add Favorites: when enabled, any module that the user selects is automatically marked as a favorite.
  • Refresh Navigator: refreshes the list of applications and modules.
  • Collapse All Applications: collapses all applications so that only application labels appear.
  • Expand All Applications: expands all applications so that all available modules appear.

For more information, see Navigating Applications.

Note: To increase or decrease the font size on pages and menus in UI15, use your browser's zoom controls.

3.3 UI11

UI11 Application Navigator

Icon Name Description
Text Smaller and Larger Increases or decreases the size of the text in the instance. The font size displays in the header after the icon is clicked.

For more information, see Navigating Applications.

Refresh Refreshes all of the applications and modules in the application navigator. A loading message appears briefly.

For more information, see Navigating Applications.

/ Collapse/Expand All Applications Collapses or expands all of the applications in the navigator.

For more information, see Navigating Applications.

Switch Perspective Presents a list of Perspectives to choose from.


When a perspective is selected, the application navigator is filtered to show only the applications and modules included in that perspective.

For more information, see Navigating Applications.

/ Collapse/Expand Application Collapses or expands the application, which displays or hides the modules associated with it.

For more information, see Navigating Applications.

4 The Edge

4.1 UI15

Icon Name Description
Toggle Navigator Shows or hides the application navigator.
List and Form View Splits the content frame into two vertical panes, with the list pane on the left and the form pane on the right by default. To remove the split, click the button again.

To split the content pane horizontally, click the gear icon () and then click Split Layout Horizontal.

Tagged Documents Displays tagged documents and recently viewed documents in the content frame. For more information, see Tagging Documents.
All Bookmarks Displays a list of all bookmarks in a flyout window.
Home Opens the last visited homepage.

4.2 UI14

Icon Name Description
Toggle Navigator Shows or hides the application navigator.
List and Form View Splits the content frame into two vertical panes, with the list pane on the left and the form pane on the right by default. To remove the split, click the button again.

To split the content pane horizontally, click the gear icon () and then click Split Layout Horizontal.

Tagged Documents Displays tagged documents and recently viewed documents in the content frame. For more information, see Tagging Documents.
All Bookmarks Displays a list of all bookmarks in a flyout window.
Bookmarks and pane-based UI help Opens a window with Edge help topics.

4.3 UI11

Icon Name Description
Toggle Navigator Shows or hides the application navigator.
Toggle Banner Shows or hides the banner frame.
Toggle Vertical Split Splits the screen vertically. The left pane becomes the list pane and the right pane becomes the form pane. To remove the split, click the button again.
Toggle Horizontal Split Splits the screen horizontally. The top pane becomes the list pane and the bottom pane becomes the form pane. To remove the split, click the button again.
Bookmarks Displays a list of all bookmarks.
Bookmarks and pane-based UI help Opens a window with Edge help topics.

5 Form View

5.1 UI15

Icon Name Description
Back Button Navigates to the previous page.
Menu Displays the form context menu.


For more information, see Form Context Menu.

Edit Tags Expands the form header and displays the Add tag field. Use this field to create a new document tag or select an existing tag. For more information, see Tagging Documents.
Attachments Opens the Attachments dialog box for adding, viewing, or removing attachments.

For more information, see Uploading Attachments.

Toggle Annotations On/Off Shows or hides on-form annotations. When this icon is light grey, no annotations are available. For more information, see Annotating Forms.
Mail Opens the mail client to send an email from this record.

For more information, see Enabling the Email Client.

Show Live Feed Shows live feed information specific to this record.

For more information, see Live Feed.

/ Previous/Next If the form was accessed from a list view, shows the next or previous record according to the previous list. If inactive ( / ), there is no previous or next record. Point to the icon to display what position the current record is in the list.

For more information, see Using Forms.

Reference Lookup Opens the Reference Lookup dialog box for finding a record to reference.

For more information, see Reference Fields.

Related Incidents Displays incidents related to the current record. Available for some reference fields.

For information on how to configure this icon, see Related Incidents Icon.

Show Workflow Displays the workflow for the current record. Available for some reference fields.

For more information on how to configure this icon, see Show Workflow Icon

Reference Icon Displays information regarding the record referenced in the field.

If a record is specified in the reference field, point to the reference icon to display the referenced record. Hold down the Shift key to move your cursor without dismissing the pop-up window. Click the icon to navigate to the referenced record.


If you set set the system property glide.ui.clickthrough.popup to true, the reference icon changes. Clicking the icon opens the record in a new window.


For more information, see Reference Fields.

Calendar Displays the date or date-time picker for date or date-time fields.


For more information, see Using Date and Time Fields.

Show BSM Map For reference fields that specify a CI, opens the BSM map view of the CI in another window or tab.

For more information, see Business Service Management Map.

Suggestion Available on some string fields, presents a list of suggested values.


Select a suggested value to populate the field.

For more information, see Suggestion Fields.

Knowledge Searches the knowledge base for the value of the field.

For more information, see Searching Knowledge from Forms.

Hide/Show Section Hides or shows a form section.
Unlock Expands the watch list.

For more information, see Using Watch Lists.

Add me Adds the current user to the watch list.

For more information, see Using Watch Lists.

Fewer/More Lines Decreases or increases the number of lines a text field displays.
Hide/Show Related List Hides or shows a related list.
Field Watcher (Requires admin role) Appears on any form field when the user right-clicks the field and selects Watch - <field name>. It also displays the Field Watcher tab at the bottom of the screen.

5.4 Activity Formatter

5.4.1 UI15

Icon Name Description
/ Collapse/Expand Activities Collapses or expands all activities in the formatter. The same icons appear next to each activity entry and collapse or expand individual activities.
Show/Hide Activity Filters Shows or hides the activity filters.

5.4.2 UI14 and UI11

Icon Name Description
/ Collapse/Expand Activities Collapses or expands all activities in the formatter.
/ Show/Hide Activity Filters Shows or hides the activity filters.
Toggle Activity Log Shows or hides the activity log.
/ Expand/Collapse Activity Expands or collapses an individual activity.
Email Indicates an email sent from by the platform to a user.
Comment Indicates a message left in a journal field, such as Additional comments.
Worknote Indicates a message left in a protected journal field, such as Work notes.
Assignment Indicates a change in assignment.
