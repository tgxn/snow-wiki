Workflow Validators

From ServiceNow Wiki
Home > Administer > Workflow > Workflow Validation > Workflow Validators
Jump to: navigation, search


Overview

ServiceNow offers several workflow validators for workflow designers to test their workflows. This page lists all available workflow validators. See Workflow Validation for information on using workflow validators and Workflow Validation Report to see the type of information that is returned.

Hanging Workflows

Identify workflow design decisions that can result in a hanging workflow

  • ValidateTransitionOut
  • ValidateTransitionIn
  • ValidateDanglingTransition *
  • ValidateSubflows *
  • ValidateScriptForCurrentDotUpdate
*A critical error has occurred that prevents you from publishing the workflow.

Update Sets

Identify related artifacts being moved in different update sets

  • ValidateUpdateSetDependencies
  • ValidateUpdateSetParentDependencies
  • ValidateInputVarUpdateSetDependencies

Unexpected Workflow Termination

Identify workflows that can end unexpectedly

  • ValidateSingleEnd

External Dependencies

Identify external ServiceNow artifacts that have potential workflow dependencies

  • ValidateParentFlow

Workflow Properties Conflicts

Identify workflow design decisions that conflict with workflow properties

  • ValidateLowestCommonTable
  • ValidateTableChange
