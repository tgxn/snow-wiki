1 Overview

The Desired State application includes reports to assess your audit results.

These reports are available to all users whose role gives them access to the Reporting application. Users with the admin role can share these reports with specific users of groups or change the display options.

2 Viewing the Desired State Reports

Navigate to Reports > View / Run. In the Reports search field, enter all or part of the report name. You may also scroll to the designated category and select one of the reports.

In addition to these reports, you can also generate other reports.

Sample Desired State Threshold Report

Report Description Category
Desired State Discrepancies This report displays all desired state audit results that have a follow-on task that is not yet in the Closed Complete state. The report is grouped by column name.
  • Type: bar chart
  • Table: Audit Result [cert_audit_result]
 Audit Result
Desired State Result with Stability Unstable This report displays all audit results where the Stability field has the value Unstable. The report is grouped by CI and stacked by audit. This report is available starting with the Eureka release.
  • Type: bar chart
  • Table: Audit Result [cert_audit_result]
 Audit Result
Desired State Result with Threshold Exceeded This report displays all audit results where the Threshold field has the value Exceeded. The report is grouped by CI and stacked by each audit. This report is available starting with the Eureka release.
  • Type: bar chart
  • Table: Audit Result [cert_audit_result]
 Audit Result
Upcoming Desired State Audits This report displays the desired state audits that are scheduled to run in the next two quarters.
  • Type: List (tabular) report
  • Table: Audit [cert_audit]
 Audit
30/60/90 Day Desired State Task Aging This report displays the number of follow-on tasks that are not Closed Complete for desired state audit types. The report is grouped by aging level.

Desired State Hierarchical Task Roll Up This report displays similar data to the Task Aging report, but groups the results by manager.

Outstanding Desired State Tasks This report displays similar data to Task Aging report, but groups the results by short description.
  • Type: List (tabular) report
