Overview

Data management functions include importing, exporting, and archiving database data and configuring fields and tables. Although the actual data stored in the database is managed by ServiceNow, the instance provides a user-friendly interface for managing the data. An overview of ideas and definitions for managing data can be found on the Introduction to Managing Data page.
