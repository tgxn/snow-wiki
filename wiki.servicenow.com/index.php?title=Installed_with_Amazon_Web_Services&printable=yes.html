1 Overview

The following components are installed with the Amazon Web Services plugin starting with the Fuji release.

Demo data is available with Amazon Web Services (AWS). The demo data provides sample budgets, distribution costs, expense allocations, contract rate cards, and rate card costs. For details, see Amazon Web Services Demo Data.

2 Tables

AWS adds the following tables.

Table Description
Tables related to tags
AWS Tags [aws_tag] Defines the tags used for tagging Amazon resources.
AWS Tag Rules [aws_tag_rule] Defines the rules for automatically attaching values to tags.
AWS CI Tag Value [aws_ci_tag_value] Maps a CMDCI to a tag name and tag value.
AWS Custom Tag Value [aws_custom_tag_value] Maps a tag to a static string value.
AWS Resource Tag History [aws_rsrc_tag_history] Maps an Amazon resource ID to an Amazon tag.
AWS Resource Tag History Mapping [aws_rsrc_tag_history_mapping] Maps an an aws_rsrc_tag_history reference to a tag name and tag value.
Tables related to downloads
AWS Job Configuration [aws_job_config] Represents an AWS bill download.
AWS Billing Report Type [aws_billing_report_type] Defines how to parse a downloaded Amazon billing .csv file.
AWS Report Download Log [aws_report_download_log] Represents a log item created during an AWS billing download.
AWS Billing Rollup Aggregate [aws_billing_rollup_aggregate] Defines which field to sum from on the hourly and daily billing tables.
AWS Billing Rollup Pivot Point [aws_billing_rollup_pivot] Defines which field to rollup on the hour and daily billing tables.
AWS Billing Report Filter Point [aws_billing_report_filter] Inserts all unique resource IDs that are used in Amazon cost reports.
Tables related to Amazon Cost Reports
AWS Billing Hourly with Tags [aws_billing_hourly_with_tags] Contains records corresponding to an Amazon line item with hourly data and tag mapping relationship. Keeps data less than 30 days old.
AWS Billing Chart Definition [aws_billing_chart_definition] Contains cost and usage report selections.
AWS Billing Rollup Daily [aws_billing_rollup_daily] Created from aws_billing_hourly_with_tags records table, for a 1 day duration. Only keeps data less than 180 days old.
AWS Billing Rollup Monthly [aws_billing_rollup_monthly] Created from the aws_billing_hourly_with_tags table, for a 1 month duration. Only keeps data less than than 1 year old.
Tables related to resource optimization
AWS Optimization Rule EBS [aws_opt_rule_ebs] Contains rules for Amazon Elastic Block Store resources. This table is a child of the opt_rule_attached_storage table.
AWS Optimization Rule EC2 [aws_opt_rule_ec2] Contains rules for Amazon EC2 resources. This table is a child of the opt_rule_attached_storage table.
AWS Optimization Rule ELB [aws_opt_rule_elb] Contains rules for Amazon Elastic Load Balancing. This table is a child of the opt_rule_attached_storage table.
Tables related to provisioning and discovery
CloudFormation Provisioning Rules [aws_cldfrm_provisioning_rules] Contains rules for Amazon Elastic Load Balancing. This table is a child of the opt_rule_attached_storage table.
AWS CloudFormation Stack Change Definition [aws_cldfrm_stack_change_definition] Contains an editable draft of stack changes. The records are created when modifying a stack.
AWS CloudFormation Stack Change Parameters [aws_cldfrm_stack_change_params] Contains an editable draft of stack parameters. The records are created when modifying a stack.
AWS CloudFormation Stack Resource [aws_cldfrm_stack_change_definition] Contains a list of stack resources.
AWS CloudFormation Script Mapping [aws_cloudformation_script_mapping] Contains a script mapping table used to resolve scripts dynamically.
AWS CloudFormation Stack Output [aws_cloudformation_stack_output] Contains a list of stack outputs.
AWS CloudFormation Stack Parameter [aws_cloudformation_stack_parameter] Contains a list of stack parameters.
AWS Console Access Request [aws_console_access_request] Contains either a URL or link to generate tokens for Amazon console access.
CloudFormation Parameters Reference Table Exclusions List [cldfrm_param_ref_table_excl] Contains a list tables that are not applicable for CloudFormation Parameters Reference Table lookup.
CloudFormation Template Parameters [cloudformation_template_parameter] Contains a list of CloudFormation template parameters. The records are created when a template is saved.
CloudFormation Variable Set for Template Parameters [cloudformation_template_to_variableset] Contains mapping for CloudFormation template and variable sets.
AWS CloudFormation Stack [cmdb_ci_cloudformation_stack] Contains a list of CloudFormation stacks from all Amazon regions.
CloudFormation Template [cmdb_ci_cloudformation_template] Contains a list of CloudFormation templates.
AWS Security Token Catalog Item [sc_aws_security_token_cat_item] Contains the catalog item for AWS security tokens.
CloudFormation Catalog Item [sc_cloudformation_cat_item ] Contains a catalog item for a CloudFormation stack.
AWS Auto Scaling Group Launch Config [aws_asgrp_launch_cfg] Contains a list of launch configurations from all regions.
AWS Policy [aws_policy] Contains a list of policies that can be applied to security tokens for federated access.
AWS Security Token [aws_token] Contains security tokens for federated access.
AWS Security Token for User [aws_user_token] Contains the mapping for a user and a security token.
AWS Auto Scaling Group [cmdb_ci_aws_asgrp ] Contains a list of autoscaling groups from all regions.
Amazon Web Service [amazon_web_service] Contains algorithms, api version, and other meta data needed to make API call for different Amazon Web Services.
AWS Datasource Type Mapping [aws_datasource_type_mapping] Contains mappings between discovered attributes and a variable in a normalized object created on the fly.
Normalized Object to DB Mapping [normalizedobject_db_mapping] Contains mapping between variables in normalized object to field in tables.
AWS Region Endpoint [aws_region_endpoint] Contains a list of endpoints for all regions for different amazon web services.
AWS Region [aws_region] Contains a list of all amazon regions.
AWS Resource Type [aws_resource_type] Contains a list of amazon resource type.
AWS Service Endpoint [aws_service_endpoint] Contains metadata to calculate a service endpoint.
AWS Resource [cmdb_ci_aws_resource] Contains a reference to the specific resource record.
AWS Account [aws_account_admin] Contains account details for Amazon accounts such as username and ID.
AWS Actions [aws_action] Contains actions for Amazon Web Services API calls.
AWS Availability Zone [aws_availability_zone] Contains a list of availability zone from all regions.
AWS Credentials [aws_credentials] Contains access credentials for amazon account like security key and access key.
AWS Elastic Block Store Snapshot [aws_ebs_snapshot] Contains a list of snapshot from all regions.
AWS Elastic Load Balancer Backend Server Description) [aws_elb_backend_server] Contains a list of back-end servers.
AWS Elastic Load Balancer Policy [aws_elb_policy] Contains a list of load balancer policies from all regions.
AWS Noun [aws_noun] Contains nouns to create probe actions such as image and instances.
AWS Verb [aws_verb] Contains verbs to create probe actions such as get and describe.
AWS Resource to Availability Zone Mapping [aws_resource_to_availzone] Contains a mapping between a generic AWS resource and an availability zone.
AWS EBS Volume [cmdb_ci_aws_ebs_volume] Contains a list of volumes from all regions.
AWS Elastic Load Balancer Listener [cmdb_ci_aws_elb_listener] Contains a list of ELB listners from all regions.
AWS Subnet [cmdb_ci_aws_subnet] Contains a list of subnets from all regions.
AWS VPCs [cmdb_ci_aws_vpc] Contains a list of VPCs from all regions.
EC2 Virtual Machine Instance [cmdb_ci_ec2_instance] Contains a list of virtual machine instances from all regions.
Subnet [cmdb_ci_subnet] Contains a list of ELBs from all regions.
Virtual Private Cloud [cmdb_ci_vpc] Parent table for cmdb_ci_aws_vpc.
AWS Probe [discovery_aws_probe] Contains a list of AWS probes.
Probe Parameter Override [dscy_probe_parameter_override] Contains overrides for request parameter of probes.
AWS VPC Security Group Rule [aws_vpc_security_group_rule] Contains a list of security group rules.
AWS VPC Security Group [aws_vpc_security_group] Contains a list of security groups.
AWS VPC Security Group Rule Traffic Type [aws_vpc_sg_rule_traffic_type] Contains a list of security group rule traffic types (name, protocol number, from_port, and to_port).
EC2 Approved Image [ec2_approved_images] Contains images approved by a cloud_administrator for provisioning user requests.
EC2 Image Attribute [ec2_image_attribute] Contains list of attributes specifying properties of images for spinning up EC2 instances.
EC2 Image Criteria [ec2_image_critera] Contains a list of filter names used to filter EC2 instance types and sizes.
EC2 Criteria Attribute [ec2_criteria_attribute_m2m] Contains list of mapping between criteria and image attribute used for filtering EC2 instances types and sizes.
EC2 Image [ec2_image] Contains a list of all EC2 images from all regions.
EC2 Keypairs [ec2_keypairs] Contains the keypairs for all the regions for the account.
EC2 Provision Rules [ec2_provision_rules] Contains rules to auto-select provisioning variables such as account, vpc, and subnet.
EC2 Region Size Exclude [ec2_region_size_exclude_m2m] Contains a list of mappings between region and sizes that are not supported in that given region.
EC2 Shared Image Account [ec2_shared_image_account] Contains the names and numbers of the accounts that provide shared EC2 images.
EC2 Size Criteria [ec2_size_criteria_m2m] Contains a list of mappings between image criteria and EC2 sizes used for filtering EC2 instances types and sizes.
EC2 Size [ec2_size] Contains list of supported EC2 sizes from all regions.
Internet Protocol [internet_protocol] A list of standard internet protocol numbers and names.
ICMP Message [ip_icmp_message] A list of ICMP control messages
IP Message Mapping [ip_message_mapping] Contains mappings between an IP protocol number and a translator script include.
EC2 Catalog Item [sc_ec2_cat_item] Contains catalog items for EC2 instances.
Catalog EC2 Element Price [sc_ec2_element_price] Contains mappings between an EC2 element and the catalog price.
EC2 Key Pair Catalog Item [sc_ec2_key_pair_cat_item] Contains catalog item for Amazon keypair.
EC2 Catalog Offering [sc_ec2_os_selection] Contains the OS types that appear in the service catalog, as defined by the cloud administrator.

3 Plugins

AWS adds the following plugins.

Table Description
com.snc.aws The primary Amazon Web Services plugin.
com.snc.aws.activities Amazon Web Services activities.
com.snc.aws.common Amazon common.
com.snc.aws.core Components common to Amazon Web Services.
com.snc.discovery.aws Probes and sensors for discovering infrastructure components in Amazon Web Services, including VPCs, subnets, EC2 instances, availability zones, regions, volumes, and so on.
com.snc.ec2_v2 Support for provisioning Amazon EC2 instances from images (AMIs). Includes a service catalog entry for ordering an instance running Windows or Linux.
com.snc.orchestration.activities.ec2 Uses workflow technology to issue Amazon EC2 activities that will execute the commands on Amazon cloud.

4 Properties

AWS adds the following properties.

Name Description
aws.token.provisioning.duration Provisioning token duration (in minutes).
aws.cldfrm.template.s3_bucket The S3 bucket for large CloudFormation templates.
aws.cldfrm.template.size_threshold Threshold (in bytes) for large CloudFormation templates.
aws.rerc.opt.enabled Status of resource optimization report scheduled job.
aws.rerc.opt.wf_time_out The maximum time (in seconds) a resource optimization analysis is allowed to run before self-termination.

5 User Roles

AWS adds the following user roles.

Role Contains Roles Description
aws_admin aws_user, discovery_admin Has full access to common AWS configurations.
aws_operator aws_user, itil, ec2_operator Responsible for the technical administration and operation of the all AWS provisioning environment.
aws_user none Has access to the AWS application menu.
cloud_admin workflow_admin, credential_admin, aws_admin, itil, cloud_user Has full access to the overall operation of the virtual provisioning cloud environment.
cloud_user aws_user Can request and control individual virtual assets, but has no control over how/where those assets are provisioned
cloud_operator cloud_user, aws_operator, ec2_operator Performs technical administration and operation of the cloud provisioning environments utilized by the Virtual Provisioning application.
ec2_operator itil Performs technical administration and operation of the EC2 provisioning environment.
