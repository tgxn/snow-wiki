Field Administration

1 Overview

The Condition Count Widget can be activated on condition fields to display a preview of the records that would meet the current set of conditions.

2 Using the Condition Count Widget

For fields where the condition count widget is activated, the number of records that match the conditions will automatically display. The count refreshes if the field the condition field depends on, such as Table, is changed; if the Table field is left blank, the widget is hidden.

In this example, a condition is run on the Incident table to look for incidents where Category is Network:

The list shows that there are 112 records with Network as its category.

To refresh the preview, click the update count icon ().

To view details of the results, click on the number of records to open the list view of the results:

3 Adding the Condition Count Widget

To add the condition count widget to a condition field:

  1. Right-click the field label and select Configure Dictionary (Personalize Dictionary in versions prior to Fuji).
  2. Add show_condition_count=true to the Attributes field.
  3. Submit.
