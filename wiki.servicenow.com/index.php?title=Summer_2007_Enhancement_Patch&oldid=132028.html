The following issues were addressed by targeted fixes in a patch deployed 29 July 2007 to instances running our Summer 2007 Enhancement.

  • Dot-walked queries against roles sometimes do not display correctly
  • Grouped list display issue in IE
  • Case-sensitive role compares
  • Possible duplicate .xml updates
  • Remove hard-coded test for "itil" role for report creation
  • Use of equals operator to set journal fields can fail
  • Do not allow domain enabling to change to false during an upgrade
  • Title nulls out when paging through a calendar report
  • Dependent choice list issue
  • Allow view setting for Search results
  • CMDB class cannot be changed
  • Apply defaults during imports
  • Autocomplete on Search views
  • Null choice values during import can break import
  • Answer cascade into Catalog items doesn't work properly
  • Enhancement to navigation buttons
  • Properly write out an update record if manipulating the attachments on an update synched record
  • Ignore baseline entries during upgrade
  • Read-only inputs still need style
  • Include SQL Server in the JDBC loader
  • AddressException errors when receiving a POP3 message can fail
  • Issue with related lists and domains
  • Grouped list report enhancements/fixes
  • Seek to correct position in a list when duplicate values exist
  • IRRegen sometimes will not work
  • Import enhancements
  • Date queries with an OR can return incorrect results
  • Resolve incident email activity by name causing long delays when LDAP is used
  • Properly save variables on action scripts even if parent record is unchanged
  • Consistent enter key behavior between FF and IE
  • Choice Lists not saving on extended tables
  • Honor order field on Service Catalog subcategories
  • LDAP may change values in a group when it should not
  • Print mode enhancements/fixes
  • Disable a flawed business rule script debugger

The following issues are addressed by targeted fixes in an unreleased patch to our Summer 2007 Enhancement. All fixes were included in our Fall 2007 Enhancement.

  • Changes being reported as Problems in ECMDB
  • Chart URL not generated correctly if choice field values on are extended vs. base table
  • Check advanced conditions if simple conditions are not provided (enhancement)
  • Apply reference_qual to question-type references
  • Fix variable sets in Wizards
  • Honor trusted domains when creating incidents from incoming emails
  • Reference autocomplete fixes
  • Oracle long query fix
  • MS SQL Server issues fixed with reporting, date/time functions
  • Serialize reference display so can be used during deserialization (enhancement)
  • Enforce mandatory state on wizard variables and catalog variables
  • Pending nodes not clearing properly on rejection
  • Resolve references in filters, conditions when moved between systems (enhancement)
  • Escape characters in user display name
  • Restrict "export XML" to admins in UI Macro
  • Don't show a broken image link on an unknown status
  • Fix overflow issue with Service Catalog task calculation
  • Fix for date/time imports
  • Fix for custom state defaulting in Service Catalog
  • If fields change on both main form and variable set, save both when action button clicked
  • Fix for type 3 reference lists and short sys_ids
  • Fix for mailing large reports from the export dialog
  • fix attachment icon in wizards
  • Add older Excel support to imports
  • Don't set g_user or g_form twice inside of a wizard
  • Fix for Calendar not rendering properly on the home page
  • Fix for Update Set retrieval when no update set selected
  • Fix for Update Sets being applied twice
  • Fix advanced delivery plan logic
  • Fix homepage escaping issues
  • Fix problem where cache is getting flushed on updates that do not warrant it
  • Fix for prepared queries not being issued
  • Remember the fact that a table is valid
  • Fix IP authentication test logic
  • Sort Action input names so that the parent is processed before the child
  • Turn separation back on after an import
  • Fix issue with display values for reference fields
  • Printing not honoring section collapse
  • Monthly scheduled report protection against bad request
  • Allow exceptions per table to company separation (Enhancement)
