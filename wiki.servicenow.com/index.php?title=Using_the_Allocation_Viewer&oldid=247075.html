1 Overview

The allocation viewer is a graphical display that illustrates how rules allocate expenses. It displays a hierarchy of nodes with the general ledger expense at the top, allocation rules that process the expense in the middle, and each allocation segment at the bottom. You can preview allocations before they are made or view allocations after they are processed.

Note: If you are using Internet Explorer, version 8 or later is required to use the allocation viewer.


An example allocation that shows how a general ledger expense is allocated over the segments specified in the three methods.


When you preview allocations, records are created in the Cost Allocation Preview [itfm_cost_allocation_preview] table, rather than the Cost Allocation [itfm_cost_allocation] table, where records are created when you allocate expenses.

2 Previewing an Allocation for an Expense

To see what an allocation would look like for an unallocated expense:

  1. Navigate to IT Finance > General Ledger > Groomed Expenses.
  2. Click an expense that is not allocated.
  3. Click Preview Allocation under Related Links.
    The allocation viewer appears in a new tab or browser window. The allocation viewer shows the expense line with the rule that first matches the expense and how it breaks down the expenses.
  4. To view any other rules of a higher order number that match the expense, right-click the expense and select Show all rules matching this expense. All the rules that match appear in the Cost Allocation Rules list.

You can also preview allocations from the workbench. See Previewing Allocations from the Workbench for more information.

See [[#Allocation Viewer Options|Allocation Viewer Options] for more information about using the viewer.

3 Previewing an Allocation for a Rule

You can preview which general ledger expenses would be processed by a rule and how the rule would break down the expenses. You can preview allocations for rules that:

  • Are active
  • Run on the Groomed General Ledger Data table, not the Cost Allocation table
  • Did not process any allocation lines yet

When you preview an allocation for a rule, the application finds an unallocated expense that matches the rule and runs a mock allocation as if the rule were the only one to run, ignoring its Order value. When you actually perform allocations, the system does consider the Order value, and the rule that you are previewing might not actually be run. See Administering Allocations for more information.

To see what general ledger expenses can be processed by a particular rule:

  1. Navigate to IT Finance > Cost Model > Allocation Rules.
  2. Click an active rule to open the record.
  3. Click Preview Rule under Related Links.
    The allocation viewer appears in a new tab or browser window and shows the allocation for the primary segment. You can select other segments or the complete allocation from the View choice list.
  4. To view a rule record, double-click the rule icon.
  5. To view other rules that match the expense, click View [x] rules matching this expense at the top of the viewer.

See Allocation Viewer Options for more information about using the viewer.

4 Visualizing an Allocation

You can also use the allocation viewer to view an allocation after it has been made. You can do this from the groomed general ledger expense or from the allocation line.

To visualize an allocation from the general ledger expense:

  1. Navigate to IT Finance > General Ledger > Groomed Expenses.
  2. Click an expense to open the record.
  3. Click Visualize Allocation under Related Links.

To visualize an allocation from an allocation line:

  1. Navigate to IT Finance > Cost Model > Allocation Lines.
  2. Click an allocation line to open the record.
  3. Click Visualize Allocation under Related Links.

See Allocation Viewer Options for more information about using the viewer.

5 Allocation Viewer Options

You can modify the View to show the complete allocation viewer or the allocation lines based on a segments involved in the allocation. Select a new view to redraw the allocation viewer. If the rules involved in the method that you are previewing are unallocated, segment values are not populated and the general ledger expense is shown as unallocated.

You can also view record information for the nodes:

  • To view summary information about the allocation segment or the allocation rule, point to the node.
  • To open the record for a general ledger expense, allocation rule, method, or a processed allocation, double-click the node.
  • To collapse or expand the allocation line, left-click the top-most expense node.
