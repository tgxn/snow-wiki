1 Overview

Discovery script includes available for accomplishing Discovery tasks are listed here for reference purposes. For examples of how to use Discovery script includes and use cases, see Using Discovery Script Includes. For general information on how to use script includes, see Script Includes.

2 General Discovery and Utility

The following script includes accomplish general Discovery tasks and provide utility functions.

Script Include Description
AbstractDBObject Provides common methods for classes based on records in the database.
AbstractReconciler Provides an abstract base class for reconciling newly discovered data in an XMLMemoryTable to a related list in the database.
AutomationException Provides generic exception handling for automation.
CriteriaEvaluator Implements a general-purpose criteria evaluator.
DiscoveryAjax Provides client-callable Ajax functions for Discovery.
DiscoveryAttachments Implements methods related to attachments used in Discovery.
DiscoveryClearAllSoftwareManufacturers Removes all software manufacturer relations from the Software [cmdb_ci_spkg] table.
DiscoveryException Throws a runtime automation exception in the course of a discovery.
DiscoveryFunctions Provides a set of miscellaneous functions for Discovery, including functions for managing PIDs and virtual machines.
DiscoveryIssuesURLBuilder Builds a URL for the calculated issues_link field in the Device [discovery_device_history] table.
DiscoveryLogger Provides methods for convenient logging of Discovery events.
DiscoveryTimeline Generates a view that shows Discovery probes and sensors on the timeline. See the TimelineAbstractSchedulePage script include description for correct implementation and API usage.
DiscoveryUIActionUtils Provides utility methods for the discovery of actions.
GlideRecordUtil Provides utility methods for working with GlideRecords.
HostnameJS Formats host names according to property settings.
IPAddressFixup Ensures that no other device has the same IP address after a device has been successfully discovered.
NameValuePairs Converts between maps and strings representing the name/value pairs in the map.
RoundingInterval Handles all the details of rounding intervals for CPU speed and RAM size.
VMUtils Provides utility functions for Discovery and provisioning of virtual machines (cloud provisioning).
WindowsOSNameHelper Handles the formatting of Windows OS names.
XMLUtilJS Provides XML utilities for JavaScript.

3 Schedule

The following script includes accomplish Discovery schedule tasks.

Script Include Description
DiscoveryBehavior Provides a set of functions used in the Discovery behavior functionality.
DiscoveryBehaviorRecord Encapsulates a Discovery behavior.
DiscoveryCreateRangeSets Creates range sets from IP networks, transforming the results of a network discovery into range sets that may be incorporated into Discovery schedules.
DiscoveryFunctionality Encapsulates a Discovery functionality record.
DiscoveryFunctionalityCriterion Encapsulates a Discovery functionality criterion.
DiscoveryFunctionalityDefinition Encapsulates a Discovery functionality definition.
DiscoveryJob Encapsulates a scheduled job that kicks off a discovery.
DiscoverySchedule Encapsulates a Discovery schedule.
DiscoveryStatus Encapsulates a Discovery status record.
QuickRanges Generates IP network, range, and address entries from a convenient comma-separated input field using conventional CIDR network notation, hyphenated range entries, or individual IP addresses.

4 CI Manipulation

The following script includes manage Discovery CIs.

Script Include Description
CIData Provides utility methods for working with CI data structures in JavaScript.
CimCIData Provides a wrapper for CIM CI data manipulation.
CimIDSensor Performs identification for CIM probe results.
CimInstanceToken Represents the unique query identity of a CIM instance.
CimProbeResult Processes CIM probe results.
CIRelatedList Provides utility methods for working with CI-related lists.
DiscoveryCIRelationReconciler Reconciles records that are related by CI relations.

5 CI Identification

The following script includes accomplish tasks related to CI identification.

Script Include Description
CIIdentification Provides the main class for Discovery CI identification.
CIIdentifierResult Gets the result returned by an identifier.
IDResult Provides a simple container class for the result of a CI identifier.

6 Port Scanning (Shazzam)

The following script includes accomplish port scanning tasks.

Script Include Description
Discovery Provides various JavaScripts for use in Discovery.
DiscoveryPhaser Encapsulates Discovery phases using Shazzam.
DiscoveryPortProbe Encapsulates a Discovery port probe.
DiscoveryPortScanner Provides a set of functions used in Discovery port scanning.
IPService Encapsulates an IP service.
ServiceRegistryQuery Maps Shazzam service registry query results against a port probe.
ShazzamLaunch Handles all the details of launching a Shazzam probe for a given discovery.
ShazzamResult Encapsulates a Shazzam result.
ShazzamScanner Encapsulates a Shazzam scanner.
SLPServiceRegistryQuery Maps Shazzam SLP query results to port probes.
StartDiscovery Starts Discovery jobs.


7 SNMP

The following script includes accomplish SNMP-related tasks.

Script Include Description
APCHandler Handles SNMP classification and identification for UPSs, environmental monitors, and PDUs made by APC.
ApcPduOutletReconciler Reconciles outlets for power distribution units.
GenericUPSAlarmsReconciler Provides a concrete reconciler for generic UPS alarms.
GenericUPSBypassReconciler Provides a concrete reconciler for generic UPS bypasses.
GenericUPSInputsReconciler Provides a concrete reconciler for generic UPS inputs.
GenericUPSOutputsReconciler Provides a concrete reconciler for generic UPS outputs.
NetwareHandler Handles SNMP classification and identification for Netware OS.
SNMPTable Wraps an SNMP table, mainly for identification. The elements of the table are all set into properties whose name is the table element's index.
UPSHandler Handles SNMP classification and identification for UPSs.

8 Probes

The following script includes accomplish Discovery probe-related tasks.

Script Include Description
CimProbe Maintains CIM probe metadata.
ClassifierProbes Provides a classifier probe launch facility.
ProbeHandlerCim Configures a CIM probe.
ProbeHandlerMulti Provides a Discovery probe handler for MultiProbes.
ProbeHandlerSNMP Provides a Discovery probe handler for SNMP probes.
ProbeHandlerWMI Provides a Discovery probe handler for WMI probes.

9 Sensors

The following script includes accomplish Discovery sensor-related tasks.

Script Include Description
DiscoveryAPDEnvSensor Processes the application discovery environment file.
DiscoveryAPDSensor Provides the abstract sensor for parsing application files.
DiscoveryAPDVersionSensor Processes the application discovery version file.
DiscoveryApplicationDiscoMatcher Determines how to match an application during application classification.
DiscoveryClusteredProcessHelper Provides a mechanism for rewiring classified process associations to their cluster rather than a cluster node.
DiscoveryIncludes Provides standard Include declarations for a Discovery sensor.
DiscoveryMacProfileUtils Provides utilities for parsing XML files returned by Mac OS.
DiscoveryMSSQLSensor Provides a sensor that processes MSSQL data fetched from either the PowershellMSSQL probe or the RemoteRunner probe.
DiscoveryMultiSensor Implements multisensors by running the multisensor scripts to process the information for results (called once for each probe).
DiscoveryProcessAppMatch Determines how to match an application during process classification.
DiscoveryProcessConnectionsSensor Provides the abstract sensor for running processes.
DiscoveryPsSensor Provides tools for working with UNIX process status information returned by the ps command. Most UNIX platforms share a common output for the ps command. This object is shared among the various ps sensors in Discovery.
DiscoveryRunningProcessSensor Provides the abstract sensor for running processes.
DiscoverySensor Provides the default sensor object. All sensors should extend this object.
DiscoveryTCPConnection Represents a TCP connect belonging to a process.
DiscoveryVMWareSensor Provides the sensor for processing VMware information from UNIX and Windows.
DiscoveryWindowsClusterBaseSensor Provides the base class for Windows - Classify and Windows - Cluster sensors.
LSOFParser Parses the output of the lsof command.
SerialNumberManager Manages the serial numbers for Discovery devices.
SNMPGatherDocParts Handles gathering the parts of an SNMP response document for JavaScript SNMP sensors.
SNMPNetworkInterfaces Handles creating network interfaces for JavaScript SNMP sensors.
SNMPResponse Wraps an SNMP payload response instance with methods to safely and easily retrieve SNMP singleton fields or tables.
