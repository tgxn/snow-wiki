The Spring 2010 Stable 2 release was released August 6, 2010, and includes the following enhancements and new functionality. This release is no longer recommended for upgrades. If you want to upgrade an instance, consider a current version.

A manual detailing all the new functionality can be found here.

2 Description of Functionality

Our Spring 2010 Stable 2 release contains the following new or improved functionality:

  • Overriding dictionary definitions on extended tables: If a field appears on both a base table and an extended table, administrators can override parts of the field's definition on the extended table.
  • Updated timelines using Timeline Pages: The Timeline Plugin was superseded by Timeline Pages, which delivers improved editing functionality and range controls in timelines.
  • Project Management user resource timeline improvements: The resource timeline rendering performance was improved, as were the range selection controls. New features include multiple frames, an auto refresh feature, and a current date/time indicator that sweeps across the timeline automatically.
  • Expanded auto-completion behavior for reference fields: A new attribute was added to enable auto-completion for any column appearing in a reference field pick list. In previous versions, auto-completion was only applied to the display value of the current field.
  • JSON and SOAP multiple insert support: ServiceNow now supports the creation of multiple new records for JSON and SOAP.


3 Other Notable Changes

See Spring 2010 Stable 2 Notable Changes for differences you may encounter with this build. Not necessarily problems, not necessarily major features, just things you may notice behaving different than before.
