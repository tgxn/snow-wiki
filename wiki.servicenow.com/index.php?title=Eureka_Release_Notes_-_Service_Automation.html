Service Automation & Business Management Enhancements

The Eureka release provides new and enhanced features for delivering and managing business services.


Click to sort the features list:

Facilities Service Management

Manages facilities requests throughout the organization and enables requests to be tagged to specific locations on a floor plan.


Requires the Facilities Service Management plugin.

Visual Task Boards

Allows users to organize, modify, and track progress of multiple tasks from an intuitive, Kanban-inspired interface.


Automatic for new instances. Requires the Visual Task Boards plugin on upgraded instances.

Demand Management

Consolidates strategic requests from the business and facilitates the IT investment decision process.


Requires the Demand Management plugin.

Visualizations

Displays animated 2D and 3D visual representations on a timeline for any logical data relationships within an instance.


Requires the Timeline Visualizations plugin.

On-Call Scheduling

Enables creation of on-call schedules and escalation rosters. Provides escalation rotations, assignments based on rules, and reporting of on-call obligations. Integrates with Notify for SMS support.


Requires the On-Call Scheduling plugin.

Performance Analytics

Enables users to define KPIs and visualize metrics in scorecards and dashboards, compare multiple time series, perform advanced trend analysis, and compare performance to preset targets.


Automatic for new instances. Requires the Performance Analytics plugin on upgraded instances.

Work Management

Adds dispatch, task, and SLA maps that show tasks, agents, and user locations. Calculates daily work routes for agents. Creates recurring work orders, tasks, and part requirements from templates.


Requires the Work Management plugin.

Software Asset Management

Validates compliance for Oracle, Client Access, subcapacity, and custom license types. Improves the software counter. Allows configuration of start and end dates for upgrade and downgrade rights.


Requires the Software Asset Management plugin.

Asset Management

Improves the sourcing process for assets in the stockroom. Adds a new application menu to better distinguish asset inventory management capabilities.


Automatic for all instances.

Project Management

Exports entire ServiceNow projects and individual project tasks to Microsoft Project. Allows users to edit exported projects and import them back into the ServiceNow instance.


Requires the Project Management plugin.

HR Employee Change

Allows collaboration between HR, IT, and other groups to process employee change events. Supplies a new HR schema for storing people data.


Requires the Human Resources Service Management: Employee Change plugin.

Product Catalog

Tracks customizations to product catalog tables in update sets. Uses product models installed by the Work Management plugin as templates to create work orders, tasks, and parts requirements.


Automatic for all instances.

Vendor Performance

Adds a new module for vendor managers to view vendor bubble charts.


Requires the Vendor Performance plugin.
