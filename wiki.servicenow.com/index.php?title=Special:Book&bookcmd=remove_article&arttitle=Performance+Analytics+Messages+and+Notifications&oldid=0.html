Performance Analytics

1 Overview

Messages in Performance Analytics allow you to communicate observations and actions in a Twitter-like message stream. To post a message, add your text in the message box and post the message. Select tags to add tags to the message. Tagging messages groups messages.

2 Weekly email

Every Monday Performance Analytics automatically sends a weekly email to subscribed users. This weekly email contains all key indicators and their latest known scores. Also, when available, the latest messages that have been posted.

3 Thresholds messages

When thresholds were configured for indicators, the Performance Analytics system will post new messages when the thresholds are being met. Users will receive an email notification of these threshold messages when a message like that is posted.
