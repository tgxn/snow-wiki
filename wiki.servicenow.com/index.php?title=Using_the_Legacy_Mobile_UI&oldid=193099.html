1 Overview

Two smartphone interfaces are currently supported in ServiceNow: a legacy mobile UI and the updated smartphone interface (starting with the Dublin release). Click here for information on the smartphone interface.

The ServiceNow legacy mobile UI supports many of the same features as the ServiceNow browser interface, including:

  • Lists
  • Forms
  • Navigation pane
  • Filtering

The legacy mobile UI is available only on instances that do not use the smartphone interface. If an administrator has configured your instance to use the smartphone interface, see Using the Smartphone Interface.

2 Configuring Your Device to Access ServiceNow

Some supported devices require special configuration to access ServiceNow.

2.1 BlackBerry

You must configure the default browser to access ServiceNow from a BlackBerry OS 7.0 or earlier.

  1. Select the browser application.
  2. When the browser application opens, press the menu key (), and then select Options.

  3. On the Browser Options screen, select Browser Configuration.
  4. Ensure the following options are selected:
    • Support JavaScript
    • Use Background Images
    • Show Images: On WML & HTML Pages
    • Browser Identification: BlackBerry

  5. Press the back key .
  6. At the confirmation prompt, select Save.
    The device is ready to access your ServiceNow instance.

  7. [Optional] You may want to set the default font size and minimum font size to a smaller size (available from the General Properties browser submenu).
    Smaller text allows more content to be displayed but may be difficult to read.

2.2 Opera Mini

To configure an Opera Mini device for ServiceNow access:

  1. Navigate to Menu > Tools > Settings.
  2. Ensure the Mobile View option is cleared.
    When this option is selected, the browser screen is formatted as one column and does not correctly display ServiceNow.

2.3 Windows Mobile

To configure a Windows mobile device for ServiceNow access:

  1. Open the browser application.
  2. Open the menu and select Tools > Options > General.
  3. Ensure the Allow cookies option is selected.

3 Accessing ServiceNow

To access ServiceNow from a supported mobile device:

  1. Open the browser application.
  2. In the address bar, enter the URL of your ServiceNow instance.
    The login screen appears.

  3. Enter your user name and password.
    The mobile menu appears.

  4. [Optional] You may want to bookmark the page so you do not need to enter the address again.

3.1 Adding ServiceNow on Apple Home Screens

You can add a link to the ServiceNow site on the home screen of your Apple iOS device.

  1. Access ServiceNow on your iPhone.
  2. Tap the Safari share button at the bottom of the screen and follow the on-screen instructions.
    A link to the ServiceNow site appears on the home screen.
    ServiceNow on the iPhone home screen

3.2 Selecting the UI on iPhone & Android Devices

When you access ServiceNow from an iPhone or Android device, you can choose whether to use the standard browser UI or the mobile UI. Your most recent selection is saved as a user preference so the desired UI is loaded whenever you log in with an iPhone or Android.

  • To use the standard browser UI, press the gear button on the top of the mobile home page, and select Desktop version. The screen reloads with the standard browser UI.
  • To switch back to the mobile version, press the cell phone button beside your name on the welcome bar. The screen reloads with the mobile UI.
