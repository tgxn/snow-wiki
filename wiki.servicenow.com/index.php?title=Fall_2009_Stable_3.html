 Note: Stable builds are deployed automatically only to demo instances. Customers may also request an upgrade to a stable build.

The Fall 2009 Stable 3 build was completed December 18, 2009, and includes the following enhancements and new functionality in addition to everything in Fall 2009 Stable 1 and Fall 2009 Stable 2. All content in a Stable build is included in the subsequent Release.

2 Description of Functionality

Our Fall 2009 Stable 3 build contains the following new or improved functionality:

  • Enhanced Device Identification for Discovery in DHCP Environments
  • Enhanced Security options
  • Activity Formatter automatic for all users
  • Discovery Multiprobes and Multisensors

3 Other Notable Changes

See Fall 2009 Stable 3 Notable Changes for differences you may encounter with this build. Not necessarily problems, not necessarily major features, just things you may notice behaving different than before.

4 Fixes

See Fall 2009 Stable 3 Fixes for a list of issues fixed in the Fall 2009 Stable 3 build (logged in users only).
