Overview

To allow users to change their own password, you can create a simple module in Self-Service. Create a module in the Self-Service application called "Change Password", give it a Type of "URL", and give it an Argument of "login_cpw.do". The module will present the user with a password change form and have them validate the new password by typing it twice.
