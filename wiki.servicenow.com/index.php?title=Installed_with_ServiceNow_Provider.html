1 Overview

Activating the ServiceNow Configuration Automation plugin adds or modifies tables, user roles, properties, and other components.

2 Tables

Name Description
ServiceNow Config Object [snc_config_object] A base class that some of the ServiceNow Provider tables extend.
Configuration Template [snc_config_cfg_template] A collection of node information to be applied to a node.
Configuration Template Attribute [snc_config_cfg_template_attr] The declaration of an attribute from within a configuration template.
Configuration Workflow [snc_config_workflow] A component of a configuration, identifying an orchestration workflow.
Configuration Workflow Attribute [snc_config_wf_attribute] The declaration of an attribute from within a configuration workflow that maps to an orchestration workflow input.
Configuration Workflows [snc_config_wf_to_cfg_template] The declaration of configuration workflows mapped to the configuration templates.

3 Properties

Property Name Description Default
Enable change management to control any updates to configuration templates (or node definitions)

[glide.config_auto.node_definition.use_change]

Enables change management to control any updates to configuration templates or node definitions. Yes
Enable change management to control any updates to CI

[glide.config_auto.ci_assignment.use_change]

Enables change management to control any updates to CI Yes
Allow multiple matches per node when running assignment rules

[glide.config_auto.node_groups.rules.execute]

Allows multiple matches per node when running assignment rules. No
Enable automatic change management for auto assignment

[glide.config_auto.node_groups.rules.auto_change_request]

Enables automatic change management for auto assignment. Yes

4 Script Includes

Script Name Description
SNCConfigAttributeResolver Provides utilities to handle configuration attributes.
SNCConfigHelper Provides utilities to handle configuration templates and workflows.
SNConfigTemplateChangeManager Manages changes to configuration templates.
SNConfigCIChangeManager Manages changes to a CI.
ServiceNowDiffByCI Generates the differences between the current and upcoming changes on a CI.
ServiceNowTemplateDiff Generates the differences between the current and upcoming changes on a configuration template.

5 User Roles

Role Contains Roles Description
Configuration Automation user config_auto_user A basic role with access to Self Service application for requesting items from the Service Catalog.
Configuration Automation administrator config_auto_admin Can create and modify configuration templates, configuration workflows, and node groups. Can perform all configuration actions using the ServiceNow Provider.
Note: Because this role also grants workflow_admin privileges, users with the role can edit any workflow, not just configuration workflows.

6 User Groups

Group Description
Configuration Automation Users Grants the Configuration Automation user role.
Configuration Automation Administrators Grants the Configuration Automation administrator role.
