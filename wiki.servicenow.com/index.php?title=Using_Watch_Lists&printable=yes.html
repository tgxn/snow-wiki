1 Overview

Watch lists allow multiple users to subscribe to notifications of a task. To receive these notifications, users must have an email address defined in their user record or enter an email address into the watch list email field.

Note: Administrators configure email notifications for watch lists (see Configuring Watch Lists).


2 Using Watch Lists

To use a watch list:

  1. Expand the watch list by clicking the lock icon ( / / ).
  2. Select users with the glide list controls (see table).
UI15 Icon UI14 Icon UI11 Icon Function
Add the current user.
Remove the highlighted user.
Open the highlighted user's record (active only when a user record exists).
Open a slushbucket to add or remove multiple users (not available until a record has initially been saved).
Collapse or expand the watch list.
Open a reference list to select a single user.
Enter an email address for users who are not in the User [sys_user] table or do not have an email address defined in their user record.
