1 Overview

This release of the ODBC driver provides the following fixes and is recommended for all versions of the ServiceNow platform. The previous version of the ODBC driver was 1.0.10.

Version 1.0.11 of the ODBC Driver uses Java 1.8. You may need to configure your Java path after installing this version. See KB0597981 for more information.

2 Fixed in this Release

The following issues were fixed in this version of the ODBC driver since the last release.


Number Details
PRB711960 When querying a table that has a string field whose value exceeds 65535 kb, the following error occurs:

[SN][ODBC ServiceNow driver][OpenAccess SDK SQL Engine]Disk cache error. Field length:93238 exceeds maximum limit of 65535

PRB691056 ODBC query on Display Value for a state does not return the entire Display Value
PRB716125 iSQL returns only 16383 bytes even if a field contains more data
