1 Overview

As soon as you activate work management on an instance, it is available on mobile devices for viewing information and tracking the time spent on work order tasks. The features available with the mobile interface vary depending on configuration and version.

For general information about how ServiceNow works with mobile devices, see Mobile Device Support.

Note: References to Work Management indicate that the information on this page is valid for ServiceNow versions prior to Fuji. In the Fuji release, Work Management was renamed Field Service Management.


2 Features for Work Agents

The table below displays features available to users with the wm_agent role.

Features listed for the smartphone interface are available on instances running Dublin or later with the smartphone interface enabled. The smartphone interface is enabled by default for new instances starting with the Dublin release.

Features listed for the legacy mobile interface are available on Dublin instances that have not enabled the smartphone interface and on Calgary instances.

Smartphone Interface Legacy Mobile Interface
Create work orders. Create work orders.
View all open work order tasks. View all open work order tasks.
No ability to view part requirements. Dublin or later: No ability to view part requirements.

Calgary: View part requirements.

View all "assigned to me" work order tasks.
  • Accept or reject a work order task.
  • Start work and travel on a work order task.
  • Close a task as Closed Complete or Closed Incomplete.
  • Add work notes.
 View all "assigned to me" work order tasks.
  • Accept or reject a work order task.
  • Set the date and time of travel start and completion.
  • Set the date and time of work start and completion.
View agent task map (starting with the Eureka release).
  • View work order tasks that are unassigned or assigned to you for a specific day.
  • Assign yourself nearby tasks that are unassigned.
  • Use auto-routing to show suggested order for tasks.
 No ability to view agent task map.

3 Features for Dispatchers

Smartphone interface work management features are available for users with the wm_dispatcher role starting with the Eureka release. Dispatchers can view modules for the dispatch map and dispatch queue. Both modules display work order tasks to be dispatched. The dispatch map also displays work agent locations. For general information about using these modules, see Using the Dispatch Map and Dispatching Tasks from the Queue. Note that some functionality is different in the smartphone interface.
