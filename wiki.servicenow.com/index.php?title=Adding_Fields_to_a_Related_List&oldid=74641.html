 Functionality described here requires the Admin role.

Overview

If you click the Edit button on a related list or if you are using a list collector in the service catalog, sometimes you may only see one field below for information:

For instance on the Groups related list, out of the box you only see the group name, maybe you need to see more fields from the Group record

To add more fields, browse to System UI - Views and the view you are looking for is sys_ref_list


Once looking at the form, look under the 'Sys ui lists' list for the table that you're looking to manipulate the view for (in this case, sys_user_group)


Go into the record and you will see under list elements, the fields that will be shown, add some more fields/elements as desired (you need to know the field/element name exactly)


Now you can see more fields when you look up the sys_user_group table
