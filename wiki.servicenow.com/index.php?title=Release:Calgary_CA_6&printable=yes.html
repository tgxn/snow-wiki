1 Overview

Calgary CA 6 is the first version in the Calgary release family to enter general availability (see release terminology).

For the Calgary CA 6 release:

Build date: 05-30-2013_1423
Build tag: glide-calgary-02-15-2013__cca6-05-23-2013
 Note: See how to determine the version on your instance.


2 Previous CA Releases

These releases provided fixes while the Calgary release was in a state of controlled availability (CA). They are no longer available for upgrade. Calgary CA 6 provides all fixes from the previous CA releases.

3 Fixed in This Release

To sort the table in ascending or descending order, click the sorting icon in the column heading.

Some fix descriptions provide steps to reproduce. Click the plus (+) icon to view a procedure that you can use to test the fixes on your instance.

CA Release Problem Description
CA 6 PRB584667 ServiceNow may skip an email coming from a Lotus Notes client if it contains an attachment, inline image, or signature image that has no file name. The email does not appear in the error log nor in any mailbox.
CA 6 Security Implemented security enhancements.
CA 5 PRB582357 Knowledge searches would not display results in weight order.
Click the plus for steps to reproduce
  1. Log in to an instance containing Knowledge Management demo data with a user who can search knowledge base articles.
  2. Navigate to Knowledge Base > View.
  3. In Search, enter the string date.
  4. The search results display the knowledge articles in the following weight order:
    1. KB0000022: One hit in the text. Weight: 1
    2. KB0000007: One hit in the text and one hit in short_description. Weight: 11
    3. KB0000024: One hit in the text. Weight: 1
CA 5 PRB581514 The slow query log would cause an instance to hang while processing extremely large SQL queries. For example, a SQL query containing 850,000 characters can cause an instance to hang for approximately one minute.
CA 4 PRB582728 Making the jquery.min UI script global would prevent the UI11 frameset from rendering correctly.
CA 4 PRB583740 Reporting on a database view could produce the error Syntax Error or Access Rule Violation detected by database.
CA 3 Reporting Homepage performance and pivot table problems are fixed.
CA 3 Service Catalog Problems with catalog UI policies are fixed.
CA 2 PRB583284 Instances that activated the Data Archiving plugin would see SQL syntax errors.
CA 1 PRB582766 Instances that activated the Data Archiving plugin could encounter an error when upgrading to Calgary.
