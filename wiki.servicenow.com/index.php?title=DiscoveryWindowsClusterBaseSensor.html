DiscoveryWindowsClusterBaseSensor

1 Overview

Base class for Windows Classify and Windows Cluster sensors.

1.1 Where To Use

Use with any server-side Discovery script.


2 Method Summary

Method Summary Description
prepare() Called by the Prototype JavaScript Framework during object processing.
findClusters(result) Finds the Windows clusters in the result returned by the sensor.
findNodes(result) Finds the Windows cluster nodes in the result returned by the sensor and creates an object for each with information about the node.
findResources(result) Finds the Windows cluster resources and puts them into an array of resource information.
associateClustersToNodes(result) Associates Windows clusters with specific nodes in the result returned by the sensor.
associateClustersToResources(result) Associates Windows clusters with resources in the result returned by the sensor.
associateResourcesToNodes(result) Associates resources with specific nodes in the result returned by the sensor.
propertiesToHumanReadable(properties) Converts cluster properties to a readable format.
changeCI(newCIGR) Changes the CI sys_id for the current device to the specified CI sys_id.
findCluster(name) Finds the specified cluster record.
findOrCreateCluster(name) Finds the requested cluster; if it does not exist, creates a new cluster record under the specified name.
queryForCluster(name) Queries the Windows Cluster (cmdb_ci_win_cluster) table for the specified cluster.
getPropValue(name, source) Gets the value of a specified property.
primStr(value) Converts the specified value to a string.


3 Method Detail

3.1 prepare()

Method called by the Prototype JavaScript Framework during object processing. Do not directly call this method.

3.1.1 Input Fields

Parameters:

  • None

3.1.2 Output Fields

Returns: void

3.2 findClusters(result)

Finds the Windows clusters in the result returned by the sensor.

3.2.1 Input Fields

Parameters:

  • result - The result from the sensor.

3.2.2 Output Fields

Returns: void


3.3 findNodes(result)

Finds the Windows cluster nodes in the result returned by the sensor, and creates an object for each with information about the node, including name, status, description, state, caption, and server.

3.3.1 Input Fields

Parameters:

  • result - The result from the sensor.

3.3.2 Output Fields

Returns: void


3.4 findResources(result)

Finds the Windows cluster resources and puts them into an array of resource information.

3.4.1 Input Fields

Parameters:

  • result - The result from the sensor.

3.4.2 Output Fields

Returns: void


3.5 associateClustersToResources(result)

Associates Windows clusters with resources in the result returned by the sensor.

3.5.1 Input Fields

Parameters:

  • result - The result from the sensor.

3.5.2 Output Fields

Returns: void


3.6 associateResourcesToNodes(result)

Associates resources with specific nodes in the result returned by the sensor.

3.6.1 Input Fields

Parameters:

  • result - The result from the sensor.

3.6.2 Output Fields

Returns: void


3.7 associateClustersToNodes(result)

Associates Windows clusters with specific nodes in the result returned by the sensor.

3.7.1 Input Fields

Parameters:

  • result - The result from the sensor.

3.7.2 Output Fields

Returns: void


3.8 propertiesToHumanReadable(properties)

Converts cluster properties to a readable format.

3.8.1 Input Fields

Parameters:

  • properties - The properties to convert.

3.8.2 Output Fields

Returns: the converted properties in the format [property name:property value], with one property:value pair per line.


3.9 changeCI(newCIGR)

Changes the CI sys_id for the current device to the specified CI sys_id.

3.9.1 Input Fields

Parameters:

  • newCIGR - The new CI sys_id for the device.

3.9.2 Output Fields

Returns: void


3.10 findCluster(name)

Finds the specified cluster record.

3.10.1 Input Fields

Parameters:

  • name - The cluster record to find.

3.10.2 Output Fields

Returns: (GlideRecord) the requested cluster record.


3.11 findOrCreateCluster(name)

Finds the requested cluster; if it does not exist, creates a new cluster record under the specified name.

3.11.1 Input Fields

Parameters:

  • name - The cluster name to find or, if it does not exist, the name for the new cluster record.

3.11.2 Output Fields

Returns: (GlideRecord) the found, or newly created, cluster record.


3.12 queryForCluster(name)

Queries the Windows Cluster (cmdb_ci_win_cluster) table for the specified cluster.

3.12.1 Input Fields

Parameters:

  • name - The cluster to look for.

3.12.2 Output Fields

Returns: (GlideRecord) the record of the requested cluster.


3.13 getPropValue(name, source)

Gets the value of a specified property.

3.13.1 Input Fields

Parameters:

  • name - The property to get.
  • source - The properties source.

3.13.2 Output Fields

Returns: the requested name/value pair.


3.14 primStr(value)

Converts the specified value to a string.

3.14.1 Input Fields

Parameters:

  • value - The value to convert.

3.14.2 Output Fields

Returns: the value converted to a string; if the value parameter is empty, returns null.
