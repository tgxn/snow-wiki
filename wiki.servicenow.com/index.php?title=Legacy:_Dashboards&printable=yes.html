Legacy: Dashboards

From ServiceNow Wiki
Legacy: Dashboards
Jump to: navigation, search
 Functionality described here is obsolete since the Homepages release .
 
See Also
  • System Overview portal
  • Incident Management portal
  • Customizing Homepages


In ServiceNow, dashboards offer an at-a-glance view of your IT Service Management environment. Think of a dashboard as a collection of charts, graphs, lists, etc., that can be set to automatically update at predefined intervals.


To change the refresh rate for a dashboard, look for the "Refresh:" option in the upper-right corner of the dashboard's screen, and set it to what you'd like to be.


If you do not see the refresh interval you'd like, an administration of your ServiceNow system can add new refresh interval options.

  1. From the left navigation pane, select System Properties > System, and then look for the option that sets The available refresh intervals for a dashboard in seconds.
  2. Here you can add whatever interval you'd like in seconds.  Separate multiple values with commas.


Note: if you set a dashboard's refresh interval too low, and there are a large number of people who frequently use that dashboard, you may end up noticeably slowing down your ServiceNow implementation. If you want to set a very short refresh interval, it would be a good idea to keep an eye on performance. If your system slows down noticeably, try increasing the interval.
