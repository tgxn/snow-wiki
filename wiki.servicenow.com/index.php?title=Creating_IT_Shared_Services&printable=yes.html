1 Overview

IT shared services are business services that are IT-related but are still shared and used across your organization. Examples of IT shared services are user services like a help desk and voice network services like a call center. You can create business services for your organization and put them into broader, customizable categories like IT operations or security and compliance.

By default, the IT chart of accounts includes a segment for IT shared services. When expenses are associated with IT shared services, they roll up to business services by default. This allows you to allocate expenses to any IT shared service and see the affects of the expenses on business services and, ultimately, the business units that comprise your organization. See Administering the Segment Hierarchy for more information on how these segments are related in the default configuration of the Financial Management application.

2 Creating IT Shared Services

  1. Navigate to IT Finance > Chart of Accounts > IT Shared Services.
  2. Click New.
  3. Fill out the fields on the form (see table).
  4. Click Submit.
An example IT shared service.

Field Description
Name The name of the shared service.
Owner The user in the group that owns the shared service.
Description A detailed description of the shared service.
Group The group that owns this shared service.
Service type The type of service that you are creating.

3 Creating Service Types

  1. Open one of the IT shared service records.
  2. Click the lookup icon in the Service type fields.
    The IT Shared Service Types list appears.
  3. Click New.
  4. Enter a name.
  5. Click Submit.
