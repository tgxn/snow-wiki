1 Overview

The following update fixes issues after the Fall 2010 Stable 2 release.

This release is no longer recommended for upgrades. If you want to upgrade an instance, consider a current version. For the Fall 2010 Stable 2 Update release:

Build date: 12-08-2010_1114
Build tag: glide-fall2010-stable2-12-01-2010
 Note: See how to determine the version on your instance.



1.1 PRB559423: The PDF Generator failed to generate a chart if the grid was not included.

Now when a PDF report is scheduled for email, the PDF is generated, even if the grid in hot included.

1.2 PRB559429: The SOAPEnvelope script include was not properly loaded on zbooted instances.

The SOAP protocol loads properly throughout the platform.

1.3 PRB559369: A NullPointerException occurred on the server when executing wizards.

Wizards now advance to the next panel without producing an error.

1.4 PRB559392: The variable_conditions field (used in Catalog UI Policy) was not rendering properly.

The value in the variable_conditions field is not deleted when the field is updated.

1.5 PRB559434: A NullPointerException occurred on Oracle when viewing tables with at least one Roles column.=

Record lists (using List v1) display all available rows.
