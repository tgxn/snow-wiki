1 Overview

Making a report public is not always as simple as it seems. On any report there is a button that publishes that report so that it can be viewed publicly. It is also necessary that the records being displayed by the report have a public role or else the report will be empty. This wiki article steps through a basic example of how to create a public report including what is necessary in order to create an ACL rule that makes the reported on table public. For a list of roles delivered with the ServiceNow platform, see Base System Roles.

Warning: If a report is made public, this report is openly accessible on the internet.

2 Creating a Public Report

2.1 Step 1 - Publish the Report

  1. Create a new report or select an existing one.
    When creating a new report it is necessary to save the report before it can be published.
  2. Click Publish.
    You will be alerted to a URL that can be used by users to access the report.
