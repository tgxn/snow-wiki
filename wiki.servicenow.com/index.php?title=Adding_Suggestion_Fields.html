1 Overview

A suggestion field allows users to select predefined text in addition to entering text. Administrators and users with the personalize role or the personalize_dictionary, personalize_choices, and personalize_responses roles can configure suggested text for any text or journal field.

2 Adding a Suggestion Field

To add a suggestion field to a form:

  1. Open the form to which you are adding a suggestion field.
  2. Add a new string field or locate an existing string field.
  3. Right-click the field label and select the appropriate option for your version:
    • Fuji or later: Configure Dictionary
    • Eureka or earlier: Personalize Dictionary
  4. In the Choice field, select Suggestion.
  5. Click Update.

3 Configuring Suggested Text

Suggestions are configured differently depending on field type.

3.1 String Fields

To configure the suggested text options for string fields:

  1. Right-click the field label and select the appropriate option for your version:
    • Fuji or later: Configure Choices
    • Eureka or earlier: Personalize Choices
  2. Using the slushbucket, select options and the order in which you wish them to appear.
    • To create a new option, enter the suggested text in the Enter new item field and click Add.
    • The Apply to Table field is available when the current table extends another table (example, Incident extends Task). This field allows suggested text options to be configured for all tables that extend the parent or for only the current (child) table.
  3. Click Save.
  4. To edit existing options, right-click the field label and select Show Choice List. To learn more, see Customizing Choice List Options.
Suggested text. Short fields.


3.2 Journal Fields

To configure the suggested text options for journal fields (field type begins with journal):

  1. Right-click the field label and select Configure Responses (Personalize Responses in versions prior to Fuji).
  2. Click New.
  3. Enter a label (brief description) for the option.
  4. Enter the complete text in Response Text.
  5. Click Save.
Suggested text. Long fields.
