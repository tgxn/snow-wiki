Platform Enhancements

The Calgary release provides new and enhanced features for configuring and managing your ServiceNow instance.


Click to sort the features list:

Tablet Interface

New iPad interface supports nearly full-product functionality with a user-friendly interface.


Automatic for all instances.

PDF Page Header Footer Templates

Allows users with the report_admin role to create header and footer templates. Allows reporting users to apply header and footer templates to exported PDFs.


Automatic for all instances.

Field Comparison

Allows users to evaluate equality between field values using the condition builder.


Automatic for all instances.

Zing Text Search

Improves performance and replaces the text index reaper with a new text index structure.


Automatic for all instances.

Attachment Indexing

Supports Microsoft Office (2007 and greater) XML file formats and provides new debugging properties.


Automatic for all instances.

Context-Sensitive Help

Significant enhancements include the ability to define help links for specific records. Now provides predefined help contexts for several ServiceNow features.


Automatic for all instances.

Live Feed

Now supports any attachment file type.


Automatic for instances running Live Feed.

Document Feeds

Provides live feed on record forms, creates a document feed group when a user creates a new record, and adds users when they view the record.


Automatic for instances running Live Feed.

Chat

New properties to configure sounds in the chat room.


Automatic for instances running Chat.

Report Security

Administrators can create ACLs to prevent unauthorized users from modifying reports through the report UI and by using a URL construct.


Automatic for new instances. For existing instances, activate the Report Security - enforce access control checks plugin.

Reports

Users can now export a PDF directly from the report form and control what additional information appears on the report PDF. This functionality is distinct from PDF Page Header Footer Templates.


Automatic for all instances.

Fields

Streamlined list of available field types now includes document ID, percent complete, and workflow. Also, a new property for text area fields displays the number of characters left as users type.


Automatic for all instances.

Number Maintenance

Improved usability and automatic field creation for defining record numbering. Update sets now track number formats.


Automatic for all instances.

Reference Fields

Ability to specify a unique identifier for reference fields, and improvements to dynamic creation and setting the display value.


Automatic for all instances.

Journal Fields

New property prevents users from saving invalid HTML in a journal field.


Automatic for all instances.

Personalizing Lists

New option to disable the link to the record from the first column in list view.


Automatic for all instances.

Context Ranking

Offers new controls to generate indexes for ranking definitions that are inserted by an update set.


Automatic for instances running Context Ranking.

Content Management System

New option simplifies catalog pages in a site by hiding the search bar, breadcrumbs, and the results per page choice list.


Automatic for all instances.

Upgrade History

Upgrades do not skip objects that have a current version in the Customer Updates table when only excluded fields have changed.


Automatic for all instances.

Access Control Rules

ACL rules are available for processors and client-callable script includes.


Automatic for new instances.

Encryption Support

A new business rule automatically encrypts passwords in system properties.


Automatic for new instances.

High Security Settings

A new system property offers an improved way to process ACLs when high security is enabled. Some custom ACL rules may require the prior ACL processing behavior.


Automatic for all instances

Exporting Data

New system property controls the character set used to export CSV files.


Automatic for all instances.

Import Sets

New system properties are available for ignoring bad data rows in CSV files, preserving leading spaces in Excel, and controlling changes to the dictionary entries for staging tables.


Automatic for all instances.

Internationalization Support

The term Quebecois has been changed to read French - Canada.


Automatic for all instances.

Form Annotations

New annotation type is available for adding gray text to forms.


Automatic for all instances.

Client Transaction Timing

The Client Transaction Timing module now accesses the Transaction Log Entry [syslog_transaction] table. The same fields are available on this new table.


Automatic for instances running Client Transaction Timing.

Inactivity Monitors

The Wait field on the SLA inactivity monitor form is now mandatory.


Automatic for all instances.

Banner

The default product description is now Service Automation.


Automatic for all instances.
