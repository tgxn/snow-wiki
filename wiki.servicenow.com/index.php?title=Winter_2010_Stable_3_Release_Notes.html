 Note: Stable builds are deployed automatically only to demo instances. Customers may also request an upgrade to a stable build.

The Winter 2010 Stable 3 build was completed April 16, 2010 and includes the following enhancements and new functionality. This release is no longer recommended for upgrades. If you want to upgrade an instance, consider a current version. All content in a Stable build is included in the subsequent Release.

2 Description of Functionality

Our Winter 2010 Stable 3 build contains the following new or improved functionality:

  • Shopping Cart and Catalog Item Quantity Controls: New fields in the Catalog Items form can be used to hide the quantity of items being requested and the contents of the shopping cart from a user's view.
  • Simplifying Discovery: Discovery has been enhanced to simplify the selection of scan types and to eliminate inaccurate updates to the CMDB. Logging information has been improved and some functionality has been made obsolete.



3 Other Notable Changes

See Winter 2010 Stable 3 Notable Changes for differences you may encounter with this build. Not necessarily problems, not necessarily major features, just things you may notice behaving different than before.
