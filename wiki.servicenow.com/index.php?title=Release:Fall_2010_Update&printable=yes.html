1 Updates

An update to Fall 2010 was released on October 8th, 2010 to fix the following problems.

This update is automatically included for all upgrades or new instances performed after that date. If an instance was upgraded before that date and is affected by one or more of the following problems, contact ServiceNow to get the update.

This release is no longer recommended for upgrades. If you want to upgrade an instance, consider a current version. For the Fall 2010 Update release:

Build date: 10-08-2010_1406
Build tag: glide-fall2010-09-22-2010
 Note: See how to determine the version on your instance.



1.1 PRB517230 — Issues with calls to GlideSession (query condition evaluation)

gs.getSession() now returns a stub object which supports isInteractive(). Other session level activities are still disabled within the script sandbox.

1.2 PRB513624 — Issues with onSubmit Client Script

Notification of failed client scripts have been improved. Errors in the Client Script will need to be fixed.

1.3 PRB511721 -- Dependent Fields not working properly in list editor

Dependent field checking has been enabled.

1.4 PRB513719 -- Print button on home pages doesn’t work

The print button now works as expected.

1.5 PRB516927 — SDLC, Release V2 and SCRUM are not installable by an admin

The Release Management Plugins are now available for installation by administrators.

1.6 PRB516927 — SMTP Sender Job throwing repeated OOM errors post upgrade

The SMTP sender job is no longer flagged debug.
