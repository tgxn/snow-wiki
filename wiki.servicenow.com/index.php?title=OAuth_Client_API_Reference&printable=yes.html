1 Overview

The OAuth client API provides methods to request and revoke OAuth tokens. You can also customize the OAuthUtil script include to intercept the request parameters and also parse the responses from external OAuth providers.

See OAuth API Response Parameters for a list of parameters in the JSON response in name:value pair format.

2 Application Scoping

OAuth supports application scoping by providing access to all OAuth methods in the available OAuth classes using the sn_auth prefix (starting with Fuji patch 3).

For example, the following code accesses the requestToken: 

var oAuthClient = new sn_auth.GlideOAuthClient( ) ;
var tokenResponse = oAuthClient.requestTokenByRequest('TestClient',requestObject) ;

You can still access the methods in the global scope by omitting the sn_auth prefix. However, methods that access tokens on locally scoped applications must use the prefix. See Pre-Application Scoped API Reference for the methods used before application scoping was introduced.

3 Application Scoped APIs

3.1 GlideOAuthClient

Use these methods for requesting and revoking the refresh and access tokens.

3.1.1 Method Summary

Return Type Method
GlideOAuthClientResponse requestToken(String clientName, String jsonString)
Retrieves the token for the client, with the request parameters encoded in JSON format.
GlideOAuthClientResponse requestTokenByRequest(String clientName, GlideOAuthClientRequest request)
Retrieves the token for the client, with the request and optional header parameters set into a GlideOAuthClientRequest object.
GlideOAuthClientResponse revokeToken(String clientName, String accessToken, String refreshToken, GlideOAuthClientRequest request)
Revokes the access or refresh token for the client, with the request and optional header parameters set into a GlideOAuthClientRequest object.

3.1.2 Method Details

3.1.2.1 requestToken(String clientName, String jsonString)

Retrieves the token for the client, with the request parameters encoded in JSON format.

Parameters:

  • clientName - (String) The client name.
  • jsonString - (String) The JSON string for the client.

Returns:

GlideOAuthClientResponse

Example: This example shows a resource owner password grant type request, with request parameters encoded in JSON format. 

var oAuthClient = new sn_auth.GlideOAuthClient();
var params = {grant_type:"password", username:"itil", password:'itil'};
var json = new global.JSON();
var text = json.encode(params);
var tokenResponse = oAuthClient.requestToken('TestClient', text);
var token = tokenResponse.getToken();

gs.info("AccessToken:" + token.getAccessToken());
gs.info("AccessTokenExpiresIn:" + token.getExpiresIn());
gs.info(" RefreshToken:" + token.getRefreshToken());

3.1.2.2 requestTokenByRequest(String clientName, GlideOAuthClientRequest request)

Retrieves the token for the client, with the request and optional header parameters set into a GlideOAuthClientRequest object.

Parameters:

  • clientName - (String) The client name.
  • request - (GlideOAuthClientRequest) The request.

Returns:

GlideOAuthClientResponse


3.1.2.3 revokeToken(String clientName, String accessToken, String refreshToken, GlideOAuthClientRequest request)

Revokes the access or refresh token for the client, with the request and optional header parameters set into a GlideOAuthClientRequest object.

Parameters:

  • clientName - (String) The client name.
  • accessToken - (String) The access token.
  • refreshToken - (String) The refresh token.
  • request - (GlideOAuthClientRequest) The request.

Returns:

GlideOAuthClientResponse

3.2.3 Examples

This example shows a resource owner password grant type request. 

     var tokenRequest = new sn_auth.GlideOAuthClientRequest();
     tokenRequest.setGrantType("password");
     tokenRequest.setUserName("itil");
     tokenRequest.setPassword("itil");
     tokenRequest.setScope(null);

 var oAuthClient = new GlideOAuthClient();
     var tokenResponse = oAuthClient.requestToken("TestClient", tokenRequest);
     gs.info("Error:" + tokenResponse.getErrorMessage());

     var token = tokenResponse.getToken(); 
     if(token) {
       gs.info("AccessToken:" + token.getAccessToken());
       gs.info("AccessTokenExpiresIn:" + token.getExpiresIn());
       gs.info("RefreshToken:" + token.getRefreshToken());

This example shows a refresh token grant type request. 

     var tokenRequest = new sn_auth.GlideOAuthClientRequest();
     tokenRequest.setGrantType("refresh_token");
     tokenRequest.setRefreshToken("N-GtdSVLkWP_Cr-TysXdmNy59ZYafu5ZzAS4YaSluXDm0kCkInEnu-hwM5SsGYSFwKJ6xauVmoaq7xJNoalXFQ");
     tokenRequest.setScope(null);

     var oAuthClient = new sn_auth.GlideOAuthClient();
     tokenResponse = oAuthClient.requestToken("TestClient", tokenRequest);
     gs.info("Error:" + tokenResponse.getErrorMessage());
     token = tokenResponse.getToken();
     if( token) {
        gs.info("AccessToken:" + token.getAccessToken());
        gs.info("AccessTokenExpiresIn:" + token.getExpiresIn());
        gs.info("AccessTokenSysID:" + token.getAccessTokenSysID());
        gs.info("RefreshToken:" + token.getRefreshToken());
        gs.info("RefreshTokenSysID:" + token.getRefreshTokenSysID());

3.5 OAuthUtil Script Include

Use the OAuthUtil script include to modify request parameters and to parse the token response during runtime.

3.5.1 Modifying the Script Include

By default, the OAuthUtil script include parses application/JSON type responses from external OAuth providers. If the external source that you are using OAuth to connect to returns a response other than an application/JSON type response, you must customize your own version of this script include to parse responses that are in different format. Extend or copy this script include, and then reference your version from the OAuth API Script field on the Application Registry form for third-party OAuth providers. The custom script include name must start with OAuth.

3.5.2 Method Summary

     var tokenRequest = new GlideOAuthClientRequest();
     tokenRequest.setGrantType("password");
     tokenRequest.setUserName("itil");
     tokenRequest.setPassword("itil");
     tokenRequest.setScope(null);

 var oAuthClient = new GlideOAuthClient();
     var tokenResponse = oAuthClient.requestToken("TestClient", tokenRequest);
     gs.info("Error:" + tokenResponse.getErrorMessage());

     var token = tokenResponse.getToken(); 
     if(token) {
       gs.info("AccessToken:" + token.getAccessToken());
       gs.info("AccessTokenExpiresIn:" + token.getExpiresIn());
       gs.info("RefreshToken:" + token.getRefreshToken());

This example shows a refresh token grant type request.

     tokenRequest.setScope(null);
