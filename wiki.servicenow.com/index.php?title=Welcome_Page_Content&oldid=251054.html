1 Overview

ServiceNow's login page content is completely customizable by the administrator of the system. This should contain instructions and any other important information you would like to convey to your customers each time they use the system.

2 Sections

To view and configure the welcome page elements navigate to System UI > Welcome Page Content.

You may need to enable the module if it is not visible. If any of the sections are not needed, they may be removed. You can edit any item by clicking on its short description.

  • Demonstration: Information about logging in to a ServiceNow demo system
  • Welcome: General information
  • Using: Some tips and tricks for using the system
  • More Information: a link to the ServiceNow wiki
  • Login: Login information

3 Company-specific Welcome Page Content

You can make info page content that is company-specific.

 From the left navigation pane, select System Definition > Home Page Content.

To create a new section, and use the following in your Condition (where Company is the name of the company you want to see this info page section): 

 gs.getUser().getCompanyRecord().name.toString() == "Company"

4 Choosing Language on Login

Users can select their language on the login page. This enables them to start their user session in the language of their choice.
