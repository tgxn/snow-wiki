Requesting an Upgrade

From ServiceNow Wiki
Home > Customer Support > Requesting an Upgrade
Jump to: navigation, search
Technical Support

Overview

You can upgrade your instance by request.

Partners can request an upgrade for associated customers. Both the partner and the associated customer must have the version entitlement in order for the partner to submit an upgrade request on behalf of the associated customer. For example, the partner and customer would both need a version entitlement for the Eureka release before the partner could request an upgrade to Eureka for the customer.

For detailed information and steps, see How to upgrade a ServiceNow instance and manage scheduled upgrades in the HI knowledge base.
