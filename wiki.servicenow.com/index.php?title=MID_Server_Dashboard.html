1 Overview

Starting in Fuji, the MID Server dashboard provides information about each MID Server. The MID Server Dashboard displays information from the MID Server Status [ecc_agent_status] table.

2 Adding a Gauge to the MID Server Dashboard

You must have the discovery_admin role to edit the MID Server dashboard.

  1. Navigate to MID Server > Dashboard.
  2. Click the + icon in the upper left corner of the dashboard.
  3. Select a gauge that you want to add to the dashboard.
  4. Select a location to place the gauge using one of the Add here choices.
  5. Click the X in the upper right corner of the gauge box to close the gauge choices pane.
The default gauges are found in the following locations:
Gauge Description and Path
Avg Percentage of CPU Used (Last 30 days) Shows the average CPU usage on all the systems that host MID Servers in the last 30 days, by percentage.

Path: Gauges > ECC Agent Memory Metric > Avg Percentage of CPU (Last 30 days)

Avg Percentage of Max Memory Used (Last 30 days) Shows the average memory usage on all the systems that host MID Servers in the last 30 days, by percentage.

Path: Gauges > ECC Agent Scalar Metric > Avg Percentage of Max Memory Usage (Last 30 days)

MID Server Status Shows the following information about each MID Server:
  • Name: the name of the MID Server.
  • Host name: the name of the system that hosts the MID Server.
  • Status: the status of the MID Server. Valid values: Up, Down.
  • Max memory used %: the highest percentage of memory used on the system that hosts the MID Server in the last 24 hours.
  • Mean CPU used %: the average percentage of CPU usage on the system that hosts the MID Server in the last 24 hours.
  • Pending jobs: number of pending ECC queue jobs for that MID Server.
  • Processing jobs: number of processing ECC queue jobs for that MID Server.

Path: Gauges > MID Server Status > MID Server Status

3 Customizing a Gauge on the MID Server Dashboard

You can customize the gauges on the dashboard if you have the discovery_admin role.

  1. Click anywhere inside the gauge.
  2. Click the pencil icon.
    A pencil icon appears in the upper right corner of the gauge if you have permission to change the gauge.
  3. Customize the gauge on this report page.
  4. Click Save.
  5. Click the X in the upper right corner to close the report page.
  6. Navigate to the dashboard after the report has been saved.
