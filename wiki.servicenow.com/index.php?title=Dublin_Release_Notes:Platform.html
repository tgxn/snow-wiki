Platform Enhancements

The Dublin release provides new and enhanced features for configuring and managing your ServiceNow instance.


Click to sort the features list:

Smartphone Interface

Optimizes the ServiceNow mobile user experience for service catalog and knowledge base access from iPhones and Android phones. Adds productivity features such as real-time record commenting, homepages, and tagging.


Automatic for all instances.

Tablet Interface

Supports Android Ice Cream Sandwich and Jelly Bean. Adds label editing, deleting, and sorting capabilities to improve user experience.


Automatic for all instances.

Notify

Supports notifications via text messages and voicemails. Manages conference calls between participating users for quick communications.


Admin can activate the NotifyNow plugin.

Reporting Engine

Provides a new charting engine that uses the Highcharts charting library and supports client-side report generation.


Admin can active the Report Charting V2 plugin.

Email Diagnostics and Properties

Provides a new properties page layout that displays the status of your email accounts.


Automatic for all instances.

Right-to-Left Language Support

Supports right-to-left languages, such as Hebrew.


Automatic for all instances.

Assessments

Evaluates any set of records in ServiceNow with scripts and custom questionnaires. Provides unique graphical formats for viewing assessment results.


Admin can activate the Assessment plugin.

Easy Import

Imports records without having to use transform maps.


Admin can activate the Easy Import plugin.

Dynamic Filter Options

Allows the creation of conditions based on contextual data without having to use JavaScript.


Automatic for all instances.

Choice Lists

Improves creation and administration of choice lists.


Automatic for all instances.

E.164-Compliant Phone Number Fields

Allows users to properly format and validate that telephone numbers are E.164-compliant when displaying local and international numbers.


Automatic for all instances.

Email Service Availability

Ensures email service availability during an upgrade.


Automatic for all instances.

Usage Analytics

Provides data collection, auditing, and reporting for application usage and license compliance on the ServiceNow platform.


Automatic for all instances.

Content Management System

Allows administrators to add content pages to an application.


Automatic for all instances.

Homepages

Allows administrators to add homepages to an application.


Automatic for all instances.

Import Sets

Improves performance when no transform field mappings or row scripts are defined in a transform map.


Automatic for all instances.

Task Flattening

Improves performance by combining the Task table and all child tables, such as Incident, Problem, and Change, into a single physical database table.


Automatic for instances using MySQL.

Document Feeds

Allows administrators to enable document feeds from the table record.


Automatic for all instances.

Field Administration

Provides a new field type for validating phone numbers.


Automatic for all instances.

Approval with E-Signature

Supports the use of SAML 2.0 update 1 for authentication, which works without logging the user out or deleting session cookies.


Automatic for instances running Approvals with e-Signature.

Availability

Improves platform reliability by reducing the need for maintenance restarts.


Automatic for all instances.

Deadlock Detection

Detects and takes corrective action for Java deadlocks.


Automatic for all instances.

Exporting Data

Allows control of multiple aspects of data export with new system properties, such as export thresholds and line breaks in CSV files.


Automatic for all instances.

Inbound Email Actions

Matches incoming email to active users before matching to inactive users.


Automatic for all instances.

Time Zones

Requires that certain reports and queries observe daylight saving time rules.


Automatic for all instances.

Application Menus

Allows administrators to define application menus and modules that are available for browsers and mobile devices.


Automatic for all instances.

Attachments

Provides the ability to force downloads of all file types that administrators specify, rather than rendering the file types in the browser.


Automatic for all instances.

Google Maps Support

Allows administrators to add their own client ID with a new system property.


Automatic for all instances.

High Security Settings

Enforces strict validation on CSRF tokens by default.


Automatic for all instances.

Script Includes

Makes client-callable script includes private by default unless public pages need to access them.


Automatic for all instances.

Table Auditing

Audits the change history for the Password field on the User [sys_user] table.


Automatic for all instances.

UI Actions

Supports UI actions in the Dublin mobile UI.


Automatic for all instances.

User Sessions

Provides the ability to enforce the session timeout value even if the user specifies an automatic refresh value.


Automatic for all instances.

HTML Fields

Improves accessibility and support for screen readers with the addition of the Alt field for images.


Automatic for all instances.

Schedules

Provides two new default schedules: Project Management Schedule and Resource Management Schedule.


Automatic for all instances

Available Plugins

Allows administrators to install a variety of plugins that previously required a request to ServiceNow technical support.


Automatic for all instances.

Zing Text Search

Improves search results by increasing the relative importance of exact word and phrase matches.


Automatic for all instances.
