Automation Enhancements

The Calgary release provides new and enhanced features for automating business processes and services.


Click to sort the features list:

Cloud Provisioning

New application enables the provisioning and management of private (VMware) and public (Amazon EC2) virtual machines in a single application that is fully integrated with ServiceNow.


Automatic for instances running Runbook plugins for VMware or Amazon EC2.

Workflow Validation

New validation tool enables workflow designers to run validation tests on their models prior to publication and to detect potential problems in update sets containing separate workflow elements.


Automatic for all instances.

Microsoft Hyper-V Discovery

Discovery now creates CMDB records for the Hyper-V server, including the host machine, running virtual machines, stored templates, and component relationships.


Automatic for instances running Discovery.

Discovery Resource Throttling

To mitigate performance issues during large discoveries, resource throttling controls the interval between Discovery sensor processing jobs, so that no two sensors are processed simultaneously.


Activate the Discovery Throttling plugin.

Quick Discovery/DiscoverNow

Allows an administrator to discover a single configuration item from a Discovery Schedule form or a script as soon as the device is connected to the network.


Automatic for instances running Discovery.

Workflow

Now offers the ability to see activity execution order for troubleshooting and to create exits for skipped approvals. Improves subflow performance.


Automatic for all instances.

MID Server

Enhancements include new configuration parameters, discovery_admin role changes, a new configuration file, and new system requirements.


Automatic for all instances.

Discovery

Includes enhancements to configuration parameters, schedules, classifiers, and credentials. Improves access to CIM storage tables.


Automatic for instances running Discovery.

Application Profile Discovery

Now supports the use of MID Servers configured for PowerShell to discover applications on Windows machines.


Automatic for instances running Discovery.

Data Collected by Discovery

New data is collected for Windows and MSSQL.


Automatic for instances running Discovery.

Runbook Activities

Five new Runbook activities and an enhanced activity are provided for Amazon EC2 and VMware.


Automatic for all instances running Runbook Automation.

External Credential Storage

External credential storage is available for use with Runbook Automation. The Discovery plugin is not required.


Automatic for instances running Runbook Automation.

Amazon EC2 for Training Partners

A termination date and time can now be set for Amazon EC2 lab instances, preventing them from running longer than necessary.


Automatic for instances running Runbook Automation - Amazon EC2 Integration for Training Partners.
