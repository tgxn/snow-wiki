IT Operations Management Enhancements

The Fuji release provides new and enhanced features for managing IT operations.

Click to sort the features list:

Cloud Provisioning - Amazon Standard Tagging and Billing

Allows administrators to tag Amazon resources to enable usage analysis. Supports Amazon billing to let administrators analyze cost metrics across Amazon resources through dashboards and reports.


Requires the Orchestration - Amazon EC2 plugin.

Cloud Provisioning - AWS Cloud Formation Stack Provisioning and Updates

Provides support for the provisioning and management of Amazon CloudFormation stacks to create virtual datacenters using Amazon Web Services resources.


Requires the Orchestration - Amazon EC2 plugin.

Cloud Provisioning - AWS S3 Activities

Provides support for Amazon S3 storage solution activities.


Requires the Orchestration - Amazon EC2 plugin.

Cloud Provisioning - AWS Virtual Private Clouds (VPC) Provisioning and Updates

Adds support for Amazon Virtual Private Clouds (VPCs) to isolate and secure virtual datacenters.


Requires the Orchestration - Amazon EC2 plugin.

Cloud Provisioning - On Demand Discovery

Allows administrators to discover cloud resources on demand or as part of a cloud provisioning workflow. Adds support for VMware VMotion technology such as cloning.


Requires the Orchestration plugin.

Cloud Provisioning - VMware Improvements

Adds support for VMware VMotion technology such as cloning.


Requires the Orchestration - VMware Support plugin.

CMDB - Business Service Management Map

Next Generation BSM (NG-BSM). Provides an enhanced and interactive graphical interface to visualize Configuration Items (CIs) and their relationships. Built on D3 and Angular technology. Continues providing the filtering and related information capabilities available in the Legacy BSM.


Automatic for all instances.

CMDB - IT Service Baseline

Allows administrators to create a baseline for business services, view all changes from the baseline that were made outside of the change process, override changes to the baseline, and push new changes to the baseline.


Requires the Service Model plugin.

CMDB - Proposed Changes

Allows administrators to add and apply proposed changes to the relationships between configuration items.


Automatic for all instances.

Configuration Automation - Node Groups

Allows multiple nodes to be provisioned at the same time when they are defined as a node group. This is a foundation element for service provisioning.


Requires the Configuration Automation plugin.

Configuration Automation - Reporting

Provides new configuration automation dashboards to quickly assess resources.


Requires the Configuration Automation plugin.

Configuration Automation - ServiceNow as a Configuration Automation Provider

Supports the use of familiar automation tasks when enabling ServiceNow as a configuration management provider for Orchestration activities.


Requires the ServiceNow Provider Configuration Automation plugin.

Discovery

Provides a dashboard, enhanced probes, and support for SNMPv3. Added support for DAS, NAS, and SAN technologies to collect information on the storage infrastructure. MID Servers can listen for traps and events.


Requires the Discovery plugin.

Event Management - CloudWatch Integration

Supports event monitoring of Amazon Web Services.


Requires the Event Management plugin.

Event Management - Interface Performance Enhancement

Improves throughput by 100 to 200 percent through parallel event processing.


Requires the Event Management plugin.

Event Management - Monitoring Connectors

Enables the ServiceWatch Connector library to deliver integrations (NetCool, SolarWinds, SCOM, Splunk, HP Operations Manager) to the Event Management REST API.


Requires the Event Management plugin.

Event Management - ServiceWatch SNMP Collector Integration

Leverages the ServiceWatch SNMP collector to enable MID Servers to collect and forward SNMP traps to the Event Management REST API.


Requires the Event Management plugin.

Event Management - Threshold Rules

Allows event administrators to create alerts for when a threshold is exceeded.


Requires the Event Management plugin.

Orchestration - Activities

Improves how workflow designers perform create, read, update, and delete (CRUD) operations on ServiceNow tables.


Requires the Orchestration plugin.

Orchestration - Activity Designer

Enables administrators to create reusable workflow activities with minimal or no scripting, using templates that can query data from ServiceNow tables and external sources such as REST, SOAP, and Powershell. Lets administrators upload and download custom activities from the ServiceNow Store, organize activities into packs, and share data between activities.


Requires the Orchestration plugin.

Orchestration - UI Refresh

Improves the user interface with frames, redesigned search, and expanded menu options.


Requires the Orchestration plugin.

Orchestration - Workflow Dashboard

Tracks status of workflow health by identifying potential issues in aging workflows and workflows without a context. Uses historical data to identify workflows operating outside their average runtimes that might need performance improvements.


Requires the Orchestration plugin.

Password Reset - Alt+Ctrl+Del

Provides a Windows add-on to support resetting passwords on a locked PC.


Requires the Password Reset plugin.

Password Reset - Change Password

Supports changing passwords in addition to resetting passwords.


Requires the Password Reset plugin.

Password Reset - Enhanced User Experience

Provides a redesigned interface for resetting passwords.


Requires the Password Reset plugin.

ServiceWatch

Provides a dashboard to monitor business services and analyze impact of changes. Adds an event console to monitor events that affect business services. Provides an Excel-based graphical report generator. Provides a top-down discovery of business service topologies.


ServiceWatch is a separate product. Integration with the ServiceNow CMDB requires the Service Model plugin.
