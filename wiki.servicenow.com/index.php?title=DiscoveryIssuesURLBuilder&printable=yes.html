1 Overview

Builds a URL for the calculated issues_link field in discovery_device_history.

1.1 Where To Use

Use with any server-side Discovery script where you need a build a URL for a calculated issues link.


2 Method Summary

Method Summary Description
url(dh) Builds and returns the URL for the calculated issues_link field in discovery_device_history.


3 Method Detail

3.1 url(dh)

Builds and returns the URL for the calculated issues_link field in discovery_device_history.

3.1.1 Input Fields

Parameters:

  • dh - The device history record.

3.1.2 Output Fields

Returns: the URL.
