Scripting in Scoped Applications

1 Overview

The scoped XMLNode API allows you to query values from XML nodes. XMLNodes are extracted from XMLDocument2 objects, which contain XML strings.

The scoped XMLNode API is available starting with the Fuji release.

2 Method Summary

Return Type Method
String getTextContent()
Gets the node's text content.
XMLNode getLastChild()
Gets the node's last child node.
XMLNode getFirstChild()
Gets the node's first child node.
String getNodeValue()
Gets the node's value.
String getNodeName()
Gets the node's name.
Boolean hasAttribute(String attribute)
Determines if the node has the specified attribute.
String getAttribute(String attribute)
Gets the value of the specified attribute.
XMLNodeIterator getChildNodeIterator()
Gets the node's XMLNodeIterator object.
String toString()
Gets the node's string value.

3 Constructors

There are no constructors for creating a stand alone instance of an XMLNode object. Instead, use the createElement() method of XMLDocument2, which adds a node to an existing document.

4 Method Details

4.1 getTextContent()

Gets the text content of the current node. The text content of a node consists of all the node's child text nodes.

Parameters:

  • None

Returns:

String - the text content of the current node.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmldoc = new XMLDocument2();
xmldoc.parseXML(xmlString);
var node = xmldoc.getNode('//one/two');
gs.info(node.getTextContent());

Output: 

abcd1234

4.2 getLastChild()

Gets the node's last child node.

Parameters:

  • None

Returns:

XMLNode - the node's last child node.

Example: 

var xmlString = "<test>" +
                "<one>" +
                "<two att=\"xxx\">abcd1234</two>" +
                "<three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "<two>another</two>" +
                "</one>" +
                "<number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//one');

gs.info(node.getLastChild());

Output: 

<two>another</two>

4.3 getFirstChild()

Gets the node's first child node.

Parameters:

  • None

Returns:

XMLNode - the node's first child node.

Example: 

var xmlString = "<test>" +
                "<one>" +
                "<two att=\"xxx\">abcd1234</two>" +
                "<three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "<two>another</two>" +
                "</one>" +
                "<number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//one');
gs.info(node.getFirstChild());

Output: 

<two att="xxx">abcd1234</two>

4.4 getNodeValue()

Gets the node's value. A node's value is determined by the node type. Element and document-element nodes return null.

Parameters:

  • None

Returns:

String - the node's value.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//two');
gs.info(node.getNodeValue());

Output: 

null

4.5 getNodeName()

Gets the node's name. A node's name is determined by the node type. A document-element node's name is #document. A text node's name is #text. An element node's name is the element's name.

Parameters:

  • None

Returns:

String - the node's name.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//two');
gs.info(node.getNodeName());

Output: 

two

4.6 hasAttribute(String attribute)

Determines if the node has the specified attribute.

Parameters:

  • attribute - (String) name of the attribute to check.

Returns:

Boolean - true, if the node has the attribute; false, if not.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//two');
gs.info(node.hasAttribute('att'));

Output: 

true

4.7 getAttribute(String attribute)

Gets the value of the attribute.

Parameters:

  • attribute - (String) name of the attribute.

Returns:

String - the attribute's value.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//two');
gs.info(node.getAttribute('att'));

Output: 

xxx

4.8 getChildNodeIterator()

Gets a XMLNodeIterator object that can be used to walk through the list of child nodes.

Parameters:

  • None

Returns:

XMLNodeIterator - the node iterator object.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//one');
var iter= node.getChildNodeIterator();
gs.info(iter.hasNext());

Output: 

true

4.9 toString()

Returns the string value of the current node.

Parameters:

  • None

Returns:

String - the string value of the current node.

Example: 

var xmlString = "<test>" +
                "  <one>" +
                "    <two att=\"xxx\">abcd1234</two>" +
                "    <three boo=\"yah\" att=\"yyy\">1234abcd</three>" +
                "    <two>another</two>" +
                "  </one>" +
                "  <number>1234</number>" +
                "</test>";
var xmlDoc = new XMLDocument2();
xmlDoc.parseXML(xmlString);
var node = xmlDoc.getNode('//one');
gs.info(node.toString());

Output: Line breaks were added to the output. 

<one>    
<two att="xxx">abcd1234</two>    
<three att="yyy" boo="yah">1234abcd</three>    
<two>another</two>  
</one>
