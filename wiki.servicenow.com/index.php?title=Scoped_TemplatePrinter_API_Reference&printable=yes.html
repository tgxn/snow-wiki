Scripting in Scoped Applications

1 Overview

Scoped TemplatePrinter handles printing from a mail script to the email message. There is no constructor for the scoped TemplatePrinter API. The methods are called in mail scripts using the template global variable.

The Scoped TemplatePrinter API is available starting with the Fuji release.

2 Method Summary

Return Type Method
void print(String string)
Prints the string to the email body.
void space(Number spaces)
Outputs the spaces to the email body.

3 Constructors

There are no constructors for the scoped TemplatePrinter API. Access the methods in mail scripts using the template global variable.

4 Method Details

4.1 print(String string)

Prints the string to the email body.

Parameters:

  • string - (String) the string to print.

Returns:

void

Example: 

template.print("Incident number - " + current.number + "\n");

4.2 space(Number spaces)

Adds spaces to the email body.

Parameters:

  • spaces - (Number) the number of non-breaking spaces to output to the email body.

Returns:

void

Example: 

 template.space(4);
