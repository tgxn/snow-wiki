Legacy:Hiding the Subcategory Price

From ServiceNow Wiki
Legacy:Hiding the Subcategory Price
Jump to: navigation, search
 Functionality described here is obsolete since the Spring 2009 Stable 3 release and has been replaced by Hiding Prices in Cart and Subcategories.
Functionality described here requires the Admin role.

1 Overview

When viewing a category of the Service Catalog, subcategories display their price, even when the price is $0.00. Administrators and Catalog Administrators can choose either to hide all prices of subcategory entries, or to hide prices when they equal $0.00, using separate scripts. Both scripts are implemented by creating a new UI Macro named com.glideapp.servicecatalog_item, and inserting the appropriate jelly script.

2 Script Object

Paste the following script object into the new com.glideapp.servicecatalog_item UI Macro to suppress all Service Catalog subcategory prices: 

<?xml version="1.0" encoding="utf-8"?>
<j:jelly trim="true" xmlns:j="jelly:core" xmlns:g="glide">
<tr>
<td>
<table>
<tr>
<j:set var="jvar_icon" value="${jvar_item.getIconSRC()}"/>
<j:if test="${jvar_icon == ''}">
<j:set var="jvar_icon" value="images/service_catalog/generic_small.gifx"/>
</j:if>
<td valign="top">
<span onmousemove="popDiv(event, '${jvar_item.getID()}');" onmouseout="lockPopup(event);">
<img src="${jvar_icon}"/>
</span>
</td>
<td>
<g:sc_catalog_item_href />

<j:set var="jvar_target" value="${jvar_item.getTarget()}" />

<a href="${jvar_href}" target="${jvar_target}">
<strong>${jvar_item.getName()}</strong>
<div>${jvar_item.getShortDescription()}</div>
</a>
</td>
</tr>
</table>
</td>
</tr>
</j:jelly>

Paste the following script object into the new com.glideapp.servicecatalog_item UI Macro to suppress only Service Catalog subcategory prices that equal $0.00: 

<?xml version="1.0" encoding="utf-8"?>

<j:jelly trim="true" xmlns:j="jelly:core" xmlns:g="glide">
<tr>
<td>
<table>
<tr>
<j:set var="jvar_icon" value="${jvar_item.getIconSRC()}"/>
<j:if test="${jvar_icon == ''}">
<j:set var="jvar_icon" value="images/service_catalog/generic_small.gifx"/>
</j:if>
<td valign="top">
<span onmousemove="popDiv(event, '${jvar_item.getID()}');" onmouseout="lockPopup(event);">
<img src="${jvar_icon}"/>
</span>
</td>
<td>
<g:sc_catalog_item_href />


<g:set_if var="jvar_show_price" test="${jvar_item.getRecordClass() == 'sc_cat_item'}" true="true" false="false" />
<j:set var="jvar_target" value="${jvar_item.getTarget()}" />


<a href="${jvar_href}" target="${jvar_target}">
<strong>${jvar_item.getName()}</strong>
<div>${jvar_item.getShortDescription()}
<j:if test="${jvar_show_price &amp;&amp; jvar_item.getPrice() > 0}"> ${SP}
<g:currency_format value="${jvar_item.getPrice()}"/> </j:if> </div>
</a>
</td>
</tr>
</table>
</td>
</tr>
</j:jelly>
