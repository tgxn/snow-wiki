Assessments

1 Overview

The Assessment Overview module is a type of homepage that displays various assessment gauges, such as results by category and assessments by state. For details about editing gauges on homepages, see Adding Existing Gauges to a Homepage.

2 Roles

Users with the assessment_admin role can view the overview page and refresh, add, delete, and rearrange gauges.

3 Using the Assessments Overview Module

To use the Assessments Overview module, navigate to Assessments > Overview and click elements within the gauges to obtain more information.

The available gauges are:

Gauge Table
Assessable Records by Type Assessable Record
Assessment Instances by Assessment Group Assessment Instance
Assessments by State Assessment Instance
Answered Questions by Assigned User Assessment Instance Question
Average Rating by Type Assessment Category Result
Category Result Ratings by Category Assessment Category Result
Metrics by Data Type Assessment Metric
Stakeholders per Category Assessment Stakeholders
Top Rated Items Metric Result
Total Metrics by Metric Type Assessment Metric
Unanswered Questions by Assigned User Assessment Instance Question

For more information about the reports used by the gauges, see Assessment Reporting.
