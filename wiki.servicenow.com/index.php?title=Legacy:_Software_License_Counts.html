1 Overview

This page discusses managing software license counts when the Software Asset Management Extensions have been activated. Software Licenses are found under Asset Contracts > Software Licenses.



There are several different supported license types. Each type supports slightly different information regarding the license, but all software licenses support tracking license counts and install counts.

Specify the software license information by creating a new license or by editing an existing license.


2 Counting Installed Software

To count installed software, you need to specify the software packages that are part of this software license. You may specify as many software packages as you like for a software license. This allows you to group multiple software packages together into a single license, such as when you want to track multiple minor versions of a software package within a single license.

The system will count installed software in one of two ways. The counting is based on whether or not the license key information can be discovered for an installed software package.

2.1 Discoverable Key, Product Id and License Key

Several Microsoft software products will store their product key and license key in the registry of the machine that the software is installed on. This allows the system to discover where a specific licensed piece of software is installed in your environment. When the system discovers a software package installed on a machine, it determines if the license key was also discovered. If so, the machine will be attached to any software license that has its Discoverable Key flag checked and has a matching License Key specified. These machines are shown in the License Used By related list for the software license:



In the above example, the install count would be 5.

However, there are lots of other software packages where the license key cannot be determined by inspecting the machine on which the software is installed. In this case, the Discoverable Key flag should be unchecked. When this flag is unchecked, installed software is counted based on the counts associated with the software packages that you have associated with the software license:



In the above example, the install count would be 17.

2.2 Percent Used

The Percent Used field is updated when the Install Count or the License Count fields are changed. The Percent Used field may be used to support Software License Compliance Checking.

2.3 Associated Scripts

To customize the functionality, the following business rules can be altered:

  • Adjust install counts for licenses
  • Asset Adjust License Count
  • Asset Adjust Used License Count
  • Package Adjust Install Count'
  • Delete software license instance

Make sure that changes made to any one of the Adjust License Count business rules are applied to all of the license count business rules, so that difference tables are not computing license counts separately.

There is also a scheduled job which updates the license counts named Count Software Licenses.
