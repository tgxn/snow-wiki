Hiding Groups

From ServiceNow Wiki
Home > Administer > User Administration > Users and Groups > Hiding Groups
Jump to: navigation, search
User Administration

1 Overview

Groups can be hidden by introducing a hidden field to the group form and creating a business rule to filter out groups marked as "hidden." Only users in that group will be able to see it when selecting a group in a reference field.

2 Add a "hidden" field to the Group form

Create a new true/false field on the Group form labeled Hidden. This will create a new field called u_hidden on the sys_user_group table and enables use of the check box to designate a hidden group.

3 Add a new business rule to filter out hidden groups

This business rule filters out hidden groups from the system, except for users marked admin, groups_admin, and users within the hidden group.

Create a new "before query" business rule on the sys_user_group (Group) table with the following script: 

 
if (!gs.hasRole("admin") && !gs.hasRole("groups_admin") && gs.getSession().isInteractive()) {
  var qc = current.addQuery("u_hidden", "!=", "true"); //cannot see hidden groups... 
  qc.addOrCondition("sys_id", "javascript:getMyGroups()"); //...unless in the hidden group
}
