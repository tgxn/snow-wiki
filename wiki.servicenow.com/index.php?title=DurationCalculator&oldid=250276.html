1 Overview

Provides methods for calculating durations and due dates. For examples of how to use DurationCalculator, see Using DurationCalculator To Calculate a Due Date.

1.1 Where To Use

Use this script include to calculate a simple or relative duration in order to calculate a due date.


2 Method Summary

3 Method Detail

3.1 initialize()

Called by the Prototype JavaScript Framework during object construction. Do not call this method directly.

3.1.1 Input Fields

Parameters:

  • None

3.1.2 Output Fields

Returns: void


3.2 setSchedule(schedID, timezone)

Sets the schedule and time zone to use for calculating the due date.

3.2.1 Input Fields

Parameters:

  • schedID - (string) The schedule sys_id (table cmn_schedule).
  • timezone - (string) Optional time zone string value.

3.2.2 Output Fields

Returns: void

3.2.3 Example

var dc = new DurationCalculator();
dc.setSchedule('08fcd0830a0a0b2600079f56b1adb9ae', 'Los Angeles');


3.3 setTimeZone(timezone)

Sets the time zone to use for calculating the due date.

3.3.1 Input Fields

Parameters:

  • timezone - (string) Value of the time zone.

3.3.2 Output Fields

Returns: void

3.3.3 Example

var dc = new DurationCalculator();
dc.setTimeZone("Los Angeles");


3.4 setStartDateTime(start)

Sets the start datetime for the duration calculations.

3.4.1 Input Fields

Parameters:

  • start - Optional String or GlideDataTime object representing the start time in GMT for subsequent calculations. If not present then the current datetime is used.

3.4.2 Output Fields

Returns: void

3.4.3 Example

var dc = new DurationCalculator();
dc.setStartDateTime("2012-04-10 08:00:00")


3.5 getEndDateTime()

Gets the endDateTime property that was set by calcDuration/calcRelativeDuration, indicating the end date and time for the duration.

3.5.1 Input Fields

Parameters:

  • None

3.5.2 Output Fields

Returns: (GlideDateTime) The end dateTime.

3.5.3 Example

var dc = new DurationCalculator();
dc.calcDuration(52);
gs.print(dc.getEndDateTime());

Output: 

*** Script: 2012-04-17 20:57:27


3.6 getSeconds()

Gets the this.seconds property that was set by calcDuration/calcRelativeDuration, indicating the total number of seconds of work to be performed for the duration.

Note: This is the total work time, not the total time between start and end times and may be used to determine percentages of the work time.

3.6.1 Input Fields

Parameters:

  • None

3.6.2 Output Fields

Returns: Number of seconds.

3.6.3 Example

var dc = new DurationCalculator();
dc.calcDuration(52);
gs.print(dc.getSeconds());

</source> Output: 

*** Script: 52


3.7 getTotalSeconds()

Gets the this.totalSeconds property that was set by calcDuration/calcRelativeDuration, indicating the total number of seconds between the start and end times of the duration.

3.7.1 Input Fields

Parameters:

  • None

3.7.2 Output Fields

Returns: Number of total seconds.

3.7.3 Example

var dc = new DurationCalculator();
dc.calcDuration(52);
gs.print(dc.getTotalSeconds());

Output: 

*** Script: 52


3.8 calcScheduleDuration(startTime, endTime)

Gets the actual duration between startTime and endTime within the already-specified schedule and optionally overridden timezone. Sets this.endDateTime (for completeness), this.seconds, and this.totalSeconds.

3.8.1 Input Fields

Parameters:

  • startTime - (GlideDateTime or String) The start time. If not provided the current value is used (set using setStartDateTime()).
  • endTime - (GlideDateTime or String) The end time. If not provided the current value is used (set using setEndDateTime()).

3.8.2 Output Fields

Returns: The schedule duration in seconds (same as calling getSeconds()). Returns 0 if endTime is before startTime.

3.8.3 Example

var dc = new DurationCalculator();
gs.print(dc.calcScheduleDuration("2012-04-10 08:00:00","2012-04-14 06:00:00"));

Output: 

*** Script: 338400


3.9 calcDuration(seconds)

Calculates the end date and time. Upon completion, this.endDateTime, this.seconds, and this.totalSeconds properties are set to indicate the results of the calculation. These values can be accessed using the appropriate accessors. Prior to calling this method, call setStartDateTime with the start time to use in the computation.

3.9.1 Input Fields

Parameters:

  • seconds - (int) Number of seconds to add to the startDateTime to compute the other values.

3.9.2 Output Fields

Returns: False if input value is not a number.

3.9.3 Example

var dc = new DurationCalculator();
dc.calcDuration(52);


3.10 calcRelativeDuration(relativeDurationID)

Calculates the duration using the specified relative duration script. Upon completion, the this.endDateTime and this.seconds properties are set to indicate the results of the calculation.

3.10.1 Input Fields

Parameters:

  • relativeDurationID - (string) sys_id of relative duration schedule (table cmn_relative_duration).

3.10.2 Output Fields

Returns: The result of the duration script.

3.10.3 Example

var dc = new DurationCalculator();
dc.calcRelativeDuration('08fcd0830a0a1b2600074f56b1ad7cb');


3.11 isAfter(dt, tm)

Checks if a time occurs after a specified time of day.

3.11.1 Input Fields

Parameters:

  • dt - (GlideDateTime). The Datetime to compare to.
  • tm - (String in hh:mm:ss format). The test datetime.

3.11.2 Output Fields

Returns: True if tm is after dt.

3.11.3 Example

var dc = new DurationCalculator();
gs.print(dc.isAfter("2012-04-10 08:00:00", "07:00:00"));

Output: 

*** Script: true


3.12 calcRelativeDueDate(start, days, endTime)

Calculates the due date starting at 'start' and adding 'days' using the schedule and time zone. Called from relative duration definitions, initiated by calcRelativeDuration(), as calculator.calcRelativeDueDate(calculator.startDateTime, days). When the day that the work is due is found, sets the time to 'endTime' of that day. If there are not enough days left in the schedule, uses the last day in the schedule.

3.12.1 Input Fields

Parameters:

  • start - (GlideDateTime) The start datetime for the computation.
  • days - (int) The number of days to add to the start datetime.
  • endTime - (string) Time due of the computed day due in "hh:mm:ss" format, or blank to indicate the end of the work day

3.12.2 Output Fields

Returns: Undefined if no schedule was set prior to calling this method. True if the completion date is within the schedule. False if the completion date falls outside the schedule.

3.12.3 Example

var dc = new DurationCalculator();
dc.calcRelativeDueDate("2012-04-10 08:00:00", 2, "08:00:00")
