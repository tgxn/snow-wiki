1 Overview

The Work Management Overview module displays various work management gauges. These gauges help work management users track and manage aspects of the work management processes, such as work orders, work order tasks, and dispatch locations.

Note: If you have tasks defined with assignment groups of the type Vendor, additional gauges will display useful information for tracking vendor tasks and performance.


The Overview module is a type of homepage. For details about editing gauges on homepages, see Adding Existing Gauges to a Homepage.

2 Roles

Only users with certain roles have access to the Overview module. The different levels of access are:

  • View: can view the overview page and refresh gauges.
  • Customize: can refresh, add, delete, and rearrange gauges.

Access is limited for users with non-administrative roles starting with the Dublin release.

3 Using the Work Management Overview Module

To use the Work Management Overview module, navigate to Work Management > Overview. The map view depends on whether or not the Work Management with Geolocation plugin is activated (for versions at Eureka or later) or if you are using a version prior to Eureka. You can click elements within the gauges to obtain more information without leaving the overview page. For more information about the dispatcher's view of the map, see Dispatcher View. For information about the agent's map view, see Agent Task Map.

The dispatch map view with geolocation enabled for Eureka groups tasks and agents by proximity and displays related records in pop-ups on the map.



The dispatch map view without geolocation or for versions prior to Eureka shows individual tasks by state and displays related records in a separate window.
