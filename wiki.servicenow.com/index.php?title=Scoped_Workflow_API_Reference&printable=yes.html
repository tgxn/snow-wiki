Scripting in Scoped Applications

1 Overview

The scoped Workflow API provides methods that can be used in an activity definition script.

The scoped Workflow API is available starting with the Fuji release.

2 Method Summary

Return Type Method
Object inputs()
Returns the workflow variables.
Object scratchpad()
Returns the scratchpad.
String name()
Returns the workflow name.
String info(String message, Object args)
Adds an info message to the log. Returns the message with arguments.
String debug(String message, Object args)
Adds a debug message to the log. Returns the message with arguments.
String error(String message, Object args)
Adds an error message to the log. Returns the message with arguments.
String warn(String message, Object args)
Adds a warning message to the log. Returns the message with arguments.
void setVariable(String name, Object value)
Sets the specified variable to the specified value.
Object getVariable(String name)
Returns the specified variable's value.
void removeVariable(String name)
Deletes the variable from the activity.
String result()
Returns the workflow's result.
void setResult(String result)
Sets the workflow's result.

3 Constructors

There are no constructors for creating an instance of a scoped workflow object. Instead, use the workflow global object available in activity scripts.

4 Method Details

4.1 inputs()

Returns the workflow variables.

Parameters:

  • None

Returns:

Object - Contains the workflow variables as name value pairs.

Example: 

var variables = workflow.inputs();


4.2 scratchpad()

Returns the scratch pad.

Parameters:

  • None

Returns:

Object - The scratch pad.

Example: 

var scratchpad = workflow.scratchpad();


4.3 name()

Returns the workflow name.

Parameters:

  • None

Returns:

String - The workflow name.

Example: 

var name = workflow.name();


4.4 info(String message, Object args)

Adds an informational message to the log. Returns the message text that was logged.

Parameters:

  • message - (String) The message to add to the log.
  • args - (Object) Arguments to add to the message.

Returns:

String - The message that is logged.

Example: 

var loggedMessage = workflow.info("All is well");


4.5 debug(String message, Object args)

Adds a debug message to the log. Returns the message text that was logged.

Parameters:

  • message - (String) The message to add to the log.
  • args - (Object) Arguments to add to the message.

Returns:

String - The message that is logged.

Example: 

var loggedMessage = workflow.debug("All is well");

4.6 error(String message, Object args)

Adds an error message to the log. Returns the message text that was logged.

Parameters:

  • message - (String) The message to add to the log.
  • args - (Object) Arguments to add to the message.

Returns:

String - The message that is logged.

Example: 

var loggedMessage = workflow.error("An error has occurred. ");

4.7 warn(String message, Object args)

Adds a warning message to the log. Returns the message text that was logged.

Parameters:

  • message - (String) The message to add to the log.
  • args - (Object) Arguments to add to the message.

Returns:

String - The message that is logged.

Example: 

var loggedMessage = workflow.warn("Check your permissions.");


4.8 setVariable(String name, Object value)

Sets the specified variable to the specified value.

Parameters:

  • name - (String) The variable name.
  • value - (Object) The value to be assigned to the variable.

Returns:

void

Example: 

workflow.setVariable("task", "terrible");

4.9 getVariable(String name)

Returns the specified variable's value.

Parameters:

  • name - (String) The variable name.

Returns:

Object - The variables's value.

Example: 

var value = workflow.getVariable("task");

4.10 removeVariable(String name)

Remove the specified variable from the workflow.

Parameters:

  • name - (String) The variable name.

Returns:

void

Example: 

var value = workflow.removeVariable("task");

4.11 result()

Returns the workflow's result.

Parameters:

  • None

Returns:

String - The workflow's result.

Example: 

var value = workflow.removeVariable("task");

4.12 setResult(String result)

Set the workflow's result.

Parameters:

  • result - (String) The workflow's result.

Returns:

void

Example: 

workflow.setResult("Success");
