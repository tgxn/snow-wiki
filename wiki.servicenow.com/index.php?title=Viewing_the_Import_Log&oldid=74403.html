Overview

The Import Log is the general log where information about the internal processing occurring behind the scenes of an import operation can be found. It includes information outputted by the system during the every step of the import operation from initialization of the Import Set table from a data source to transformation onto a target table. It is also possible to customize log outputs during the transformation by using log.info, log.warn, log.error statements.
