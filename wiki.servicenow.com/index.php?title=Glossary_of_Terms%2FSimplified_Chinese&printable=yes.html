Localized Term Localized Definition English Term English Definition
帐户 在销售能力自动化插件中， 帐户表示作为销售客户或潜在销售客户的公司。 account In the Sales Force Automation plugin, accounts represent companies that are sales prospects or customers.
ACL（访问控制列表） 访问控制列表通过一系列 IP 地址来定义使用 Discovery 可以访问和扫描的设备。 这些地址可用于建立或绕过特定系统的访问限制。 ACL (Access Control List) An access control list defines via a range of IP address which devices can be accessed and scanned using Discovery. These can be used to establish or bypass access restrictions on certain systems.
活动 在销售能力自动化插件中， 活动追踪与销售相关的操作，例如电子邮件、电话或会议。 活动可能与联系人、帐户或机会相关。 activity In the Sales Force Automation plugin, activities track sales-related actions, such as email messages, phone calls, or meetings. Activities may be associated with a contact, account, or opportunity.
相关性 相关性是指由 ServiceNow Discovery 或 Orchestration 自动化所建立起来的计算机或网络设备与其凭据之间的关联性。 设备与其凭据建立匹配关系后，此关系将被写入数据库表中，并且这些凭据将用于在后续发现和工作流程活动中查询设备 affinity Affinity is the association of a computer or network device to its credentials as established by ServiceNow Discovery or Orchestration. Once a device is matched with its credentials, the relationship is written to a database table, and those credentials are used to query the device in subsequent discoveries and Workflow activities
代理 代理是指将信息写入队列 [ecc_queue] 表的外部系统（集成）的名称。 例如，MID 服务器的代理名称的格式为 mid.server.xxx，其中xxx是特定 MID 服务器的名称。 agent An agent is the name of an external system (integration) that writes messages to the Queue [ecc_queue] table. For example, the agent name for a MID Server is in the format mid.server.xxx, where \xxx\ is the name of a particular MID Server.
分配规则 分配规则定义如何处理每个支出行，以生成一条或多条支出分配记录。 规则条件是根据支出行源的数据进行定义的。 allocation rule Allocation rules define how to process each expense line to generate one or more expense allocation records. Rule conditions are defined based on data from the expense line source.
分配单元 分配单元追踪业务服务总容量以及由 CI 成本中心关系记录定义的消耗单元数量 allocation unit An allocation unit tracks the total capacity of a business service as well as the number of units being consumed as defined by the CI Cost Center Relationship records
应用程序导航器 应用程序导航器是指页面左侧用于显示应用程序和模块列表的导航栏。 application navigator The application navigator is a navigation bar on the left side of the page that displays a list of applications and modules.
审批 审批是指在工作流程继续前，工作流程内一项请求用户或用户组进行某一操作（审批或拒绝）的任务。 approval An approval is a task within a workflow that requests an action (approval or denial) from a user or user group before the workflow continues.
审批规则 审批规则是指审批一项服务或目录请求所需的配置步骤。 approval rule Approval rules are the configured steps required to approve a service or catalog request.
资产 资产是指在业务环境中发现的任何硬件、软件或易耗品。 服务提供商的资产包括任何有助于提供服务的事物。 asset An asset is any hardware, software, or consumable found in the business environment. Assets of a service provider include anything that could contribute to the delivery of a service.
资产分类 资产分类是指全部存储在同一个数据库表（资产 [alm_asset] 表或其扩展表）中的一组资产。 在 ServiceNow 中，默认的资产分类是硬件、软件和易耗品。 如果默认分类不适用于特定资产组，可以创建自定义的资产分类。 asset class An asset class is a group of assets all stored in the same database table (the Asset [alm_asset] table or a table that extends it). In ServiceNow, the default asset classes are hardware, software, and consumables. If the default classes are not appropriate for a specific group of assets, a custom asset class can be created.
订货 Scrum 敏捷开发环境中的 订货是指团队成员同意将在冲刺中处理的有序情景列表。 Scrum 过程使用三种订货： 产品、版本和冲刺。 backlog A backlog in the Scrum agile development environment is an ordered list of stories that team members agree will be worked on in a sprint. The Scrum process uses three backlogs: product, release, and sprint.
横幅框架 横幅框架是指横贯每个页面顶部的包含平台标志和基本全局导航选项的部分。 banner frame The banner frame is the portion across the top of every page that contains the platform's logo and basic global navigation choices.
基类 基类是指本身已扩展但未扩展任何其他表的表。 base class A base class is a table which is extended but does not extend any other tables.
基线 CI 基线是指配置项目在系统中以其基本形式出现时的快照。 您在创建快照之后对该配置项目进行的所有更改将在 CI 表格中显示为基本的属性更改。 创建基线后，系统将捕获 CI 的属性以及 CI 的所有第一层级关系。 系统将捕获并显示对基础 CI 或任何相关 CI 进行的任何更改。 baseline A CI baseline is a snapshot of a configuration item as it appears in your system in its basic form. All the changes that you make to that configuration item since the snapshot was created are displayed on the CI form as basic attribute changes. When a baseline is created, the attributes of the CI are captured as well as all first level relationships for the CI. Any changes to the base CI or to any related CI will be captured and displayed.
基线更新设置 基线更新设置是指安装系统更新设置插件前捕获所有自定义的更新设置。 baseline update set A baseline update set is an update set that captures all of the customizations before the System Update Sets Plugin was installed.
中断窗口 中断窗口是指定义不执行任何更改的时段的时间表。 由更改管理冲突检测器插件用于检测冲突更改。 blackout window A blackout window is a schedule that defines a period of time in which no changes should be performed. Used by the Change Management Collision Detector Plugin to detect conflicting changes.
面包屑导航 面包屑导航是指在记录列表顶部显示为链接的一系列筛选器。 单击路径的各个元素以查看不同视图。 breadcrumbs Breadcrumbs are a series of filters displayed as links at the top of the record list. Click the individual elements of the path to see a different view.
捆绑模型 捆绑模型是指由各个模型组成的单个模型。 例如，笔记本电脑、打印机、键盘和鼠标可以组合成单个捆绑模型。 bundled model A bundled model is a single model comprised of individual models. For example, a laptop, printer, keyboard, and mouse can be combined in a single, bundled model.
燃尽图 燃尽图由敏捷开发环境中的 Scrum 主控用于根据理想疾跑进度查看实际疾跑进度。 burn down chart The burn down chart is used by Scrum masters in an agile development environment to view ideal sprint progress against actual sprint progress.
业务规则 业务规则是指配置在插入、更新或删除记录或查询表时运行的一段 JavaScript。 业务规则可以设置为在出现数据库操作之前或之后运行。 在执行查询的情况下，应在数据库操作之前运行业务规则，以便使返回的数据适合用户的系统权限（用户角色）。 典型的业务规则可以在用户更新事件或升级更改请求优先级之后执行脚本。 business rule A business rule is a piece of JavaScript configured to run when a record is inserted, updated, or deleted, or when a table is queried. A business rule can be set to run before or after the database action has occurred. In the case of a query, the business rule should run before the database operation, so that the data returned to the user is appropriate to his system privileges (user roles). A typical business rule might execute a script after a user updates an incident or escalates the priority of a change request.
业务服务 业务服务是指通过配置项目 (CI) 的 IT 基础架构所支持的业务单元向客户提供的一种服务（例如电子邮件或电子消息）。 business service A business service is a service (such as email or electronic messaging) delivered to customers by business units that is supported by an IT infrastructure of configuration items (CI).
业务服务管理 (BSM) 映射 BSM 映射是指系统内的配置项目 (CI) 及其相互关系的图形显示。 Business Service Management (BSM) map A BSM map is a graphical display of the configuration items (CI) within a system and their relationships to one another.
功能 MID 服务器功能定义 IP 地址范围内的 MID 服务器的特定功能。 默认功能使管理员能够为每个 MID 服务器启动的探测选择特定功能（例如，SSH、SNMP、PowerShell）。 capability MID Server capabilities define the specific functions of a MID Server within an IP address range. The default functionality enables an administrator to select specific capabilities (for example, SSH, SNMP, PowerShell) for the probes launched by each MID Server.
目录项 目录项是指可以由最终用户订购的服务目录中的独特项目或服务。 要指定采购目录项的方式、使用人员或提供方式，必需将该项目绑定到服务目录执行计划。 catalog item A catalog item is any unique item or service in the Service Catalog that can be ordered by the end user. To specify how a catalog item is procured, who works on it, or how it is delivered, the item must be tied to a Service Catalog execution plan.
更改 更改是指可能会对 IT 服务产生影响的任何添加、修改或删除操作。 change A change is the addition, modification or removal of anything that could have an effect on IT services.
更改管理冲突检测器插件 更改管理冲突检测器插件提供用于检测计划更改是否与其他更改相冲突或是否具有其他计划问题的功能。 Change Management Collision Detector plugin The Change Management Collision Detector plugin provides the ability to detect whether planned changes conflict with other changes or have other scheduling issues.
聊天操作 聊天操作是指聊天窗口菜单（需要聊天插件）中出现的其他项目。 chat action Chat actions are additional items that appear in the chat window menu (requires the Chat plugin).
聊天队列 聊天队列定义技术支持聊天（需要聊天插件）的时间安排、支持人员和系统消息。 chat queue Chat queues define the schedules, support staff, and system messages for help desk chat (requires the Chat plugin).
聊天队列代理 聊天队列代理是指针对技术支持聊天队列（需要聊天插件）提供支持的服务支持人员。 要对特定队列提供支持，用户必须是该队列分配组的成员。 chat queue agent Chat queue agents are Service Desk staff who provide support for a help desk chat queue (requires the Chat plugin). To support a particular queue, a user must be a member of the assignment group for the queue.
CI（配置项目） CI（配置项目）是指 CMDB 中的任何计算机、设备或软件。 CI 的记录包括所有相关数据，例如制造商、供应商或位置。 CI (configuration item) A CI (configuration item) is any computer, device, or piece of software in the CMDB. A CI's record includes all relevant data, such as manufacturer, vendor, or location.
CI 成本中心关系 CI 成本中心关系是指将成本中心与其消耗的业务服务以及所使用单元的数量进行关联的记录。 此记录用于计算分配单元的数量，以更新分配单元记录。 CI cost center relationship A CI cost center relationship is a record associating a cost center to a business service that it consumes and the number of units it uses. This is used to calculate the number of allocated units to update the allocation unit record.
CI 费率卡 CI 费率卡表示与配置项目集合相关联的一系列循环出现的 CI 成本。 费率卡通常是围绕特定的成本模型而设计的。 例如，纽约数据中心中运行的某种模型服务器。 相关联的成本可能不同于在伦敦数据中心中运行的同一个服务器模型。 每个模型都具有独立的费率卡。 CI rate card A CI rate card represents a collection of recurring CI Costs associated with a collection of configuration items. Rate cards are usually designed around a certain cost model. An example might be a certain model server running in the New York data center. The costs associated could be different than the same server model running in the London data center. Each model would have a separate rate card.
分类 分类是指本身进行扩展或由其他表进行扩展的表。 class A class is a table that extends or is extended by another table.
点击链接 点击链接是指允许用户打开当前窗口中的引用记录的属性。 默认情况下，点击引用图标可以打开链接。 管理员还可以启用弹出图标，以便在其他窗口中打开引用记录。 clickthrough Clickthrough is a property that allows users to open a referenced record in the current window. Reference icons allow clickthrough by default. Administrators may also enable pop-up icons, which open a referenced record in another window.
客户端脚本 客户端脚本是指您所创建的用于在客户端浏览器中运行的 JavaScript。 ServiceNow 支持下列脚本类型：\n* onLoad() -- 加载表格时运行\n* onChange() -- 特定小部件更改值时运行\n* onSubmit() -- 提交表格时运行 client script A client script is a JavaScript you create to run in the client browser. ServiceNow supports the following script types:\n* onLoad() -- Runs when a form is loaded\n* onChange() -- Runs when a particular widget changes value\n* onSubmit() -- Runs when a form is submitted
克隆 克隆是指用于刷新测试或开发环境的 ServiceNow 生产实例的快照。 clone A clone is a snapshot of your production ServiceNow instance that you use to refresh your test or development environments.
群集 群集是指由两个或多个节点组成的 ServiceNow 实例： 将所有用户请求从浏览器定向至的一个或多个接口节点，以及处理操作的工作节点，例如 Discovery 探测。 群集可以存在于同一台物理机器或不同机器中。 在群集中，所有节点指向同一数据库并且具有相同的实例 ID。 cluster A cluster is a ServiceNow instance comprised of two or more nodes: one or more interface nodes, to which all user requests from the browser are directed, and worker nodes that handle discrete operations, such as Discovery probes. A cluster can exist on the same physical machine or on separate machines. In a cluster, all the nodes point to the same database and have the same instance ID.
CMDB（配置管理数据库） CMDB 是指包含企业网络内的配置项目 (CI) 及其关系的数据库。 CMDB (Configuration Management Database ) A CMDB is a database of the configuration items (CI) and their relationships within an enterprise's network.
并用 并用是指导入设置用以更新目标生产表中现有记录（而不是简单地插入新记录）的方式。 为某个字段配置并用后，导入设置应用程序会尝试将源值匹配到生产表上当前现有的目标值，以进行更新。 如果源值不匹配，则创建新记录。 coalescence Coalescence is the method by which import sets may update existing records in a destination production table, rather than simply inserting new records. When coalescence is configured for a field, the import set application attempts to match the source value to the currently existing target value on a production table for updating. If the source value does not match, a new record is created.
冲突 当本地客户更新和传入客户更新适用于相同的对象时，会出现 冲突。 系统更新设置预览确定是否应用传入客户更新。 collision A collision is when an incoming customer update in an update set applies to the same object as a newer local customer update. The System Update Sets Preview determines whether or not the incoming customer update is applied.
条件生成器 条件生成器使用一系列根据语境生成的字段构造条件语句。 条件生成器在许多应用程序中使用，例如筛选器、调查管理和访问控制管理。 condition builder A condition builder constructs a condition statement with a series of contextually generated fields. Condition builders are used in many applications, such as filters, survey administration, and access control administration.
配置页面 配置页面是指内容管理系统中的模块，用于设置内容页面和网站的全局默认设置。 configuration page The configuration page is the module within the Content Management System that sets the global defaults for content pages and sites.
冲突 冲突是指在计划更改与其他更改发生重叠或计划不当时由更改管理冲突检测器插件生成的记录。 冲突记录详细说明冲突的类型和冲突的项目或时间安排。 conflict A conflict is a record generated by the Change Management Collision Detector Plugin when a scheduled change overlaps with other changes, or is otherwise improperly scheduled. Conflict records detail the type of conflict and what items or schedules were in conflict.
易耗品资产 易耗品资产是指受到定性跟踪并可以大于零的数量进行转移的资产。 不会跟踪一些详细信息，例如安装易耗品的所有位置。 通常作为易耗品进行管理的资产包括铅笔、椅子和显示器支架。 consumable asset A consumable asset is an asset that is tracked qualitatively and can be transferred in quantities greater than zero. Some details are not tracked, such as all locations where the consumable is installed. Examples of assets that are often managed as consumables include pencils, chairs, and monitor stands.
联系人 在销售能力自动化插件中， 联系人表示公司的销售联系人。 联系人与帐户关联，因此会被分配给该帐户的销售代表。 contact In the Sales Force Automation plugin, contacts represent sales contacts at companies. A contact is linked to an account and hence is assigned to the sales representative for the account.
内容块 内容块是指可以添加到主页或内容页面，并且包含规定格式内容（例如静态的 HTML、报告或列表）的项目。 content block A content block is an item that can be added to a homepage or a content page, and which contains a defined form of content such as static HTML, a report, or a list.
内容框架 内容框架是指主要用户界面的主要区域。 内容框架显示用户导航到的页面。 content frame The content frame is the main area of the primary user interface. The content frame displays pages to which a user navigates.
内容链接 内容链接是指内容链接 [content_link] 表上含 URL 的记录，用于生成链接列表而非记录列表的。 content link A content link is a record on the Content Link [content_link] table that contains a URL, used by list blocks to generate a list of links rather than a list of records.
内容管理系统 内容管理系统是指为创建实例品牌前端提供功能的插件。 Content Management System The Content Management System is a plugin that provides functionality for creating branded front-ends for an instance.
内容页面 内容页面是指由内容管理系统生成的网页。 content page A content page is a web page generated by the content management system.
内容网站 内容网站是指通过相同的基础 URL 组合在一起、目标为同一个用户组的一系列内容页面。 content site A content site is a series of content pages grouped together with the same base URL, aimed at the same group of users.
内容类型 内容类型是指控制清单块的一系列定义。 content type A content type is a series of definitions that control list blocks.
约定 在销售能力自动化插件中， 约定管理产品或服务的销售记录。 约定可以具有有限的时间框架，跟踪特定的售后详细信息，并识别续订机会。 contract In the Sales Force Automation plugin, contracts manage sales records for products or services. Contracts may have a limited time frame, track specific post-sale details, and identify renewal opportunities.
CRM（客户关系管理） CRM 是指一种用于在最大限度地降低成本和利用可用技术的同时确保牢固的客户-提供商关系的管理策略。 这种策略在 ServiceNow 中通过销售能力自动化插件进行支持。 CRM (Customer Relationship Management) CRM is a management strategy to ensure a strong client-provider relationship while minimizing costs and taking advantage of available technology. This is supported in ServiceNow through the Sales Force Automation plugin.
当前版本 客户可以将实例升级到当前的 ServiceNow 版本。 current release Customers can upgrade instances to a current ServiceNow release.
当前更新设置 当前更新设置是指使用更新设置选取器选择的更新设置。 用户进行的自定义设置将会添加到用户当前的更新设置中。 current update set A current update set is an update set that has been selected using the update set picker. Customizations made by a user will be added to the user's current update set.
客户更新 客户更新是指客户更新 [sys_update_xml] 表上保存各个客户自定义更改的 XML的记录。 客户更新用于在更新流程中保护自定义，且系统更新设置允许客户将更新划分到更新设置组中并在实例之间进行移动。 customer update A customer update is a record on the Customer Updates [sys_update_xml] table which stores an individual customer customization as an XML storing the change. Customer updates are used to protect customizations during the update process, and the System Update Sets allows grouping customer updates into update sets and moving them from instance to instance.
仪表板 仪表板是指自动刷新的列表、图形、图表或其他内容项目。 dashboard A dashboard is a collection of lists, graphs, charts, or other content items that automatically refresh.
数据源 数据源可能来自文件或 Java 数据库连接 (JDBC) 的连接。 数据源用于创建导入设置，以便在将数据映射到生产表之前对其进行处理（如有需要）。 数据源支持下列远程文件检索方式： FTP、FTPS（Auth SSL、Auth TLS、隐式 SSL、隐式 TLS）、HTTP、HTTPS、SCP。 data source A data source can be from a file or a Java Database Connectivity (JDBC) connection. Data sources are used to create an import set so that data can be processed, if necessary, prior to being mapped onto a production table. Data sources support the following remote file retrieval methods: FTP, FTPS (Auth SSL, Auth TLS, Implicit SSL, Implicit TLS), HTTP, HTTPS, SCP.
默认更新设置 默认更新设置是指可以用于存储不打算转移的自定义的更新设置。 default update set The default update set is an update set which can be used to store customizations that are not meant for transfer.
执行计划 执行计划描述如何采购、配置和安装目录项。 每个计划都由执行进程中每个离散步骤的一个或多个执行计划任务组成。 例如，PDA 可能会经历桌面软件的采购、激活和安装。 delivery plan A delivery plan describes how a catalog item is procured, configured, and installed. Each plan consists of one or more delivery plan tasks which are discrete steps in the delivery process. For example, a PDA might go through procurement, activation, and installation of the desktop software.
派生的字段 派生的字段不是正在显示的主表中的字段，而是通过遵循主表中对其他某个表的引用而获取的字段。 derived field A derived field is a field that is not in the primary table that is being displayed, but is instead a field that is obtained by following a reference from the primary table to some other table.
详细信息块 详细信息块是指显示指定表中的列表、表格、目录项或知识文章的内容块。 detail block A detail block is a content block that displays either the lists, forms, catalog items, or knowledge articles of a given table.
详细页面 详细页面是指内容管理系统内包含详细信息块并用于显示 ServiceNow 表中内容的页面。 detail page A detail page is a page within the Content Management System that contains a detail block, and is used to display content from ServiceNow tables.
设备 ServiceNow 中的 设备是指用于接收平台内生成的消息的通知交付端点对象，例如电子邮件或手机。 device A device in ServiceNow is a notification delivery endpoint object, such as email or a cell phone, used to receive messages generated from within the platform.
字典 系统 字典 [sys_dictionary] 表描述实例中的表和字段。 它存储字段数据类型、字符限制、默认值、从属性和其他属性。 dictionary The system Dictionary [sys_dictionary] table describes the tables and fields in an instance. It stores field data types, character limits, default values, dependencies, and other attributes.

服务目录 服务目录是指您的 IT 技术支持为其用户提供商品和服务的中央存储库。 用户可以直接从服务目录请求这些服务和项目。 Service Catalog The Service Catalog is the central repository of goods and services that your IT help desk provides for its users. Users can directly request these services and items from the Service Catalog.
服务级别协议 (SLA) 在平台内， 服务级别协议 (SLA) 是指衡量开放任务所用时间的定义规则。 在设置的一段时间之后，SLA 将开放任务提升一个级别，从正常到中度、中度到高度以及高度到逾期。 Service Level Agreement (SLA) Within the platform, a Service Level Agreement (SLA) is a defined rule that measures the elapsed time of an open task. After a set period of time, the SLA promotes the open task an escalation level, from Normal to Moderate, Moderate to High, and High to Overdue.
服务级别约定 在平台内， 服务级别约定是指控制一个或更多 SLA 的定义主控分类。 将服务级别约定分配给特定事件将有助于找到最合适的 SLA。 Service Level Contract Within the platform, a Service Level Contract is a defined master class that controls one or more SLAs. Assigning a Service Level Contract to a particular incident will help find the most appropriate SLA.
短轮询 轮询是指聊天时浏览器从服务器获取信息以发送即时消息的方式。 通过短轮询，浏览器以 glide.short_poll_delay 属性定义的固定时间间隔将请求发送到服务器。 例如，每隔 1000 毫秒打开新的请求、接收即时（可能为空）响应并关闭连接。 short polling Polling is the method by which the browser gets information from the server to send instant messages in Chat. With short polling, the browser sends a request to the server in fixed intervals as defined by the glide.short_poll_delay property. For example, open a new request every 1000 milliseconds, receive an immediate (possibly empty) response, and close the connection.
网站首页 网站首页是指当用户首次到达内容网站时在向其显示的内容页面。 site entry page A site entry page is a content page designed to be displayed to the user upon first arriving at a content site.
多项双选列 多项双选列是指用于从可用项目库中选择多个选项的双列界面。 slushbucket A slushbucket is a double column interface for choosing multiple selections from an available pool of items.
社交 IT 社交 IT 是指使用社交媒体技术（例如即时消息和微博）提高 IT 操作。 social IT Social IT is the use of social media technology (such as instant messaging and microblogging) to enhance IT operations.
电子表格编辑模式 电子表格编辑模式，或网格编辑，允许用户使用键盘导航在列表视图中修改数据，与在电子表格中输入数据相似。 管理员可以通过将 glide.ui.list_edit_grid 属性设置为 true 来启用列表的电子表格编辑模式。 spreadsheet edit mode Spreadsheet edit mode, or grid editing, allows users to modify data in list view using keyboard navigation, similar to entering data in a spreadsheet. Administrators can enable spreadsheet edit mode for lists by setting the glide.ui.list_edit_grid property to true.
疾跑 在 Scrum 中， 疾跑是指具有可调节范围、旨在满足迅速变化的开发需要的、较短的且周期固定的日程安排。 sprint In Scrum, a sprint is a short, fixed schedule of cycles with adjustable scope, designed to address rapidly changing development needs.
静态 HTML 块 静态 HTML 块是指内容页面中提供 HTML 的内容块。 static HTML block A static HTML block is a content block that renders HTML within a content page.
库存规则 库存规则是一项定义标准，陈述当库存室中某项特定资产的库存达到指定阈值时，应从其他库存室转移或从供应商处订购一定数量。 stock rule Stock rules are defined criteria stating that when inventory of a particular asset in a stockroom reaches a specified threshold, a certain number should either be transferred from another stockroom or ordered from a vendor.
库存室 库存室是指存储资产的地方。 stockroom A stockroom is a place where assets are stored.
非索引字 非索引字是指由于对搜索结果无意义而未编索引的常见字。 冠词、连词、人称代词和介词是可能配置为 Zing 文本搜索非索引字的非索引字示例。 stop word Stop words are common words that are not indexed because they are not meaningful in search results. Articles, conjunctions, personal pronouns, and prepositions are examples that may be configured as stop words for Zing text search.
情景 情景，或用户情景，是指用于敏捷软件开发的 Scrum 方式的产品要求或客户业务案例的简要说明。 通常，情景采用通俗易懂的语言表达，以帮助读者了解软件应达到的目标。 story A story, or user story, is a brief statement of a product requirement or a customer business case that is used in the Scrum method of agile software development. Typically, stories are expressed in plain language to help the reader understand what the software should accomplish.
情景点 情景点，在 Scrum 中是指基于 Scrum 团队成员的评估对完成情景所需工作（不一定是时间）的估量。 story point Story points in Scrum are arbitrary measurements of the effort (not necessarily the time) required to complete a story, based on the estimates of Scrum team members.
Summary Set Summary Set [sys_report_summary] 表临时保存必要的信息以提供图。 此表由自定义表插件启用。 Summary Set The Summary Set [sys_report_summary] table temporarily holds the necessary information for rendering a chart. This table is enabled by the Custom Charts plugin.
支持版本 所有 ServiceNow 版本都支持两个功能版本周期的补丁和修补程序。 supported release All ServiceNow releases are supported with patches and hotfixes for two feature release cycles.
标记 标记是指实时源消息中使用 hash (#) 符号标记的字。 标记是按照关键字或标题对消息分类以获得更好的搜索结果的方式。 tag Tags are words marked with a hash (#) symbol in Live Feed messages. Tags are a way to categorize messages by keyword or topic for improved search results.
任务 任务 [task] 是指基础系统中所提供的内核表之一。 任务提供扩展任务的每个表上使用的一系列标准字段，例如事件和问题表。 task Task [task] is one of the core tables provided in the base system. It provides a series of standard fields used on each of the tables that extend it, such as the Incident and Problem tables.
主题 主题是指内容管理系统中的一组样式表。 theme A theme is a grouping of style sheets in the Content Management System.
跟踪自定义 跟踪自定义是指对记录到更新设置中的表自定义。 tracked customization Tracked customizations are customizations to tables which are recorded into update sets.
转换映射 转换映射将导入设置中的字段与单个目标表中的字段匹配。 单个导入设置字段可以映射到目标表中的多个字段。 当创建转换映射时，自动映射功能可以用于匹配字段，或者脚本可以用于高级逻辑。 定义之后，转换映射可以重新使用，并且多个转换映射（每个针对特定表）可以按您指定的顺序应用于导入设置。 transform map A transform map matches fields from an import set to fields in a single destination table. A single import set field can be mapped to multiple fields in the destination table. When creating a transform map, the automap feature can be used to match fields, or scripting can be used for advanced logic. Once defined, a transform map can be reused, and multiple transform maps (each for a specific table) can be applied to an import set in the order you specify.
树目录选取器 树目录选取器是指从分层列表中进行选择的导航工具。 tree picker A tree picker is a navigational tool for selecting from hierarchical lists.
UAC（用户帐户控制） UAC 是指限制非管理员用户对特定软件权限的安全功能。 此功能包含在现代的 Microsoft Windows 操作系统中并且可能影响 MID 的服务器插件的行为。 UAC (User Account Control) UAC is a security feature that limits the permissions of a particular software for non-administrative users. This feature is included in modern Microsoft Windows operating systems and can affect the behavior of the MID Server plugin.
UI 操作 UI 操作是指使您能够根据其他表记录中的信息在表中创建记录的弹出菜单选择。 例如，您可以直接从现有的问题记录中创建新的更改记录。 UI action A UI action is a pop-up menu selection that enables you to create a record in a table based on the information from a record in another table. For example, you can create a new Change record directly from an existing Problem record.
UI 宏 UI 宏是指可以从 UI 页面调用的可重复使用的 Jelly 脚本。 UI macro A UI macro is a reusable Jelly script that can be called from a UI page.
进程 更新设置是指客户对实例进行的、也可转移到其他实例的一系列自定义。 当对实例进行更新时，更改会自动添加到当前的更新设置中。 每次只能有一个当前的更新设置。 update set An update set is a collection of customizations made by a customer to their instance that can be transferred to another instance. When an update is made to the instance, the change is added automatically to the current update set. There can be only one current update set at a time.
更新设置选取器 更新设置选取器是指实例标题栏中的下拉选择列表，选择当前用户进行的哪项更新设置自定义将被纳入分组。 update set picker The update set picker is a drop-down choice list in the instance header bar that selects which update set customizations made by the current user will be grouped in.
更新源 更新源是指当前实例可以转移更新设置的 ServiceNow 实例。 update source An update source is a ServiceNow instance that the current instance can pull update sets from.
URL 后缀 URL 后缀定义内容管理系统中内容页面的 URL。 内容网站和内容页面均有其自己的 URL 后缀，并以下列格式生成： <实例路径> + /<网站后缀> + /<页面后缀> + .do URL suffix A URL suffix defines the URL of a content page within the Content Management System. Both Content Sites and Content Pages have their own URL suffixes, and are generated in the format: <path to your instance> + /<site suffix> + /<page suffix> + .do
验证脚本 验证脚本控制用户输入到数据字段的格式，并显示语法错误的错误消息。 例如，您可以创建脚本以强制执行用户输入电子邮件地址的正确格式。 validation script A validation script controls the formatting of user input into data fields and displays an error message for syntax violations. For example, you can create a script that enforces the proper formatting of an email address entered by a user.
变量设置 变量设置是指可以在目录项之间共享的变量模块单位。 定义一次变量设置，可以在多个地方使用。 variable set A variable set is a modular unit of variables that can be shared between catalog items. Define a variable set once and use it in multiple places.
速度图 在 Scrum 中， 速度图显示团队在一次疾跑中可以处理的工作量（单位为点），随时间推移衡量。 此计算有助于 Scrum 主控预测团队在将来的疾跑中可以处理的负载量。 velocity chart In Scrum, a velocity chart shows how much effort (in points) a team can handle in one sprint, measured over time. This calculation helps the Scrum master predict how much load a team can handle in future sprints.
视图 视图是指个性化表格或列表的保存版本。 视图使用户能够以多种方式快速显示同一表格或列表。 管理员可以创建自定义视图。 view A view is a saved version of a personalized form or list. Views enable users to quickly display the same form or list in multiple ways. Administrators can create custom views.
工作流程活动 工作流程活动是指工作流程运行时执行的各个操作。 这可以包括（但不限于）执行脚本、操作记录、等待一段设置的时间或记录事件。 workflow activity Workflow activities are the individual actions the workflow performs as it runs. This can include (but is not limited to) performing scripts, manipulating records, waiting for a set period of time, or logging an event.
工作流程字段 任务记录中的 工作流程字段存储和显示工作流程中任务记录的位置。 workflow field A workflow field in a task record stores and displays the location of the task record in the workflow.
