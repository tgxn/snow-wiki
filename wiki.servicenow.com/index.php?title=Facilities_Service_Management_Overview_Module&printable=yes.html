1 Overview

The Facilities Overview module displays various facilities request gauges. These gauges help facilities administrators and staff members track and manage aspects of the facilities request fulfillment processes.

Note: If you have tasks defined with assignment groups of the type Vendor, additional gauges will display useful information for tracking vendor tasks and performance.


The Overview module is a type of homepage. For details about editing gauges on homepages, see Adding Existing Gauges to a Homepage.

2 Accessing Facilities Reports

  1. Navigate to Facilities > Overview.
    A list of active facilities requests appears at the top, and several built-in charts appear at the bottom.
  2. Click a facilities request number to open that request.
  3. Click elements within the gauges to obtain more information.
    For example, in the Active Facilities Requests by Building bar chart, click the building name. All of the active requests for that building appear in a list.
The Facilities Overview Homepage.

2.1 Creating Facilities Reports

Users with one of the appropriate reporting roles can create and customize facilities-related reports.

Note: The facilities roles do not inherit any of the reporting roles. To grant facilities administrators or staff the rights to create and customize facilities reports, assign the appropriate reporting role.


To create a facilities report:

  1. Navigate to Reports > View / Run.
  2. Scroll down to the Facilities Request section.
  3. Click a report to open it, or click Edit to make modifications.
  4. To create your own report, click New beside Global reports at the top of the page and use any of the Facilities application tables to create the report.
