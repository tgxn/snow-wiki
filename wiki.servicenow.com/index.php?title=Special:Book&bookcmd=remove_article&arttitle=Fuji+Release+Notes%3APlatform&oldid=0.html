Platform Enhancements

The Fuji release provides new and enhanced features for configuring and managing a ServiceNow instance.


Click to sort the features list:

Domain Separation - Domain Data Scope

Automatically limits data access based on the domain of the record, and works across multiple browser tabs simultaneously.


Requires the Domain Support - MSP Extensions plugin.

Domain Troubleshooting

Improves logging for domain path query method installation.


Requires the Domain Support - MSP Extensions plugin.

Homepages - Enhancements

Allows homepages to be exported as PDF files. Finds and loads gauges more quickly.


Automatic for all instances.

Improved workflow editor

Allows citizen developers to build a simple workflow for their service in terms they understand. Building the workflow becomes more like telling the story of what happens when the request comes in, who it goes to, and who has to approve it.


Automatic for all instances.

Live Feed - Improvements

Provides an improved user experience and adds greater support for record feeds, company feed, and user feeds.


Automatic for new instances. Requires the Live Feed v2 plugin on upgraded instances.

Mobile UI

Enables users to view form attachments in the smartphone interface.


Automatic for new instances. Requires the Mobile UI plugin on upgraded instances.

OAuth - OAuth Auth Provider

Registers applications with an instance through OAuth 2.0 instead of locally-stored access credentials.


Requires the OAuth plugin.

OAuth - OAuth Resource Server

Provides OAuth support for REST APIs.


Requires the OAuth plugin.

Personalized Forms

Enables users to personalize forms without changing what other users see.


Automatic for new instances. Requires the Form Personalization plugin on upgraded instances.

Read-Only Role

Restricts a user or group of users to read-only access to all tables and fields.


Automatic for all instances.

Reporting New Chart Types

Provides additional chart types, including area, donut, semi-donut, column, pyramid, funnel, spline, speedometer, and dial.


Automatic for all instances.

Reporting - Report Designer

Offers options for configuring additional chart elements, such as the legend, title, and axis.


Automatic for all instances.

Reporting - Report Overview Homepage

Provides a redesigned reports list to filter for unused, heavily used, or performance-intensive reports.


Automatic for all instances.

Reporting - Report Sources

Allows users to create reusable and shareable report sources containing conditions that define the information to be included in reports.


Automatic for all instances.

Social Collaboration

Provides a chat service optimized to facilitate collaborative work on tasks. Supports standard chat functionality as well as the ability to track updates to comments and work notes on records.


Requires the Collaboration plugin.

Subscription Usage

Enables developers to view purchased subscriptions, allocate appropriate roles to per-user subscriptions, monitor subscription usage and allocation, and monitor application usage.


Automatic for all instances.

User Interface

Improves usability with a streamlined look and additional system settings.


Automatic for all instances.
