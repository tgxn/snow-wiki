Our Spring 2009 release was completed May 29, 2009. It contains the following new or improved functionality.

1 Spring 2009 Feature Matrix

The following table outlines the availability of major new features included in the Spring 2009 release. Most will be automatically available for new and/or existing instances. Others will be available as plugins that require system administrator or Technical Support activation.

2 Spring 2009 Description of Functionality


Our Spring 2009 release will contain the following new or improved functionality (content subject to change):

  • Map Page Marker Labels
  • Translate Large Text or HTML Fields
  • Time component added to date picker
  • Import Sets Performance Improvements
  • Dependent Field Support for Mobile Clients
  • Mobile client enhancements
  • Knowledge Base Internationalization
  • Admin has access to more plugins
  • Formatters can be overridden by macros
  • Enhanced Group by for over-loaded choice lists
  • Limit Query Complexity
  • View specific UI Policy
  • Improved Cascade Delete Performance
  • Improved Text Search Performance
  • Store and reference images in the database
  • Optionally display attachment links on knowledge articles
  • Customize mobile device lists with labels and splits
  • Client Script Access to Variable Fields on Task Records

2.1 Preview webinar

A preview webinar of Spring 2009 release functionality was held Wednesday May 13, 2009.
