1 Overview

Software license contracts contain the details of the agreement with a vendor for the licensing of a software package, including account information, license duration, and location. This functionality requires the Software License Management plugin.

A software license contract lists all the licenses defined for the software purchased and enables you to create new licenses that are related to a contract. A license is a record that displays the requirements of the agreement and the details of how the license is being used. License records calculate the expiration level and the compliance status of the software package to which the license contract applies. It is not necessary to create licenses through a Software License Contract. Licenses can be created independently without being related to a contract.

2 Contract Form

To create a software license contract and a license record:

  1. Navigate to Software License Management > Software License Contracts.
  2. Click New.
  3. Complete the form with the vendor information.
    Make sure to include the name of the software covered by the contract in the Short description field. It is good practice to use the actual name of the software package here to differentiate it from other software by the same vendor.
  4. Right-click in the header bar and select Save from the context menu.
    The Software License Bases Related List appears on the form.
  5. Click New in this list.
    A wizard appears, allowing you to select the type of license base to create:
    • Complex Software Licenses
    • Infrastructure Software Licenses
    • Usage Software Licenses
    The contract number is inherited by the license record, joining the contract to license.


  6. This form provides at-a-glance information about where and how these licenses are being used.
  7. Click on a license to reconcile compliance issues.
