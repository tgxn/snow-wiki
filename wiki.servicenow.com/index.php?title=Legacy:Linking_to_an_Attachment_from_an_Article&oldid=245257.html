1 Overview

This article details how to link directly to an attachment from a Knowledge Base article. There are two methods outlined below:

  • Linking to an attachment to view
  • Adding an image link which, when clicked, will open the image in a new window.

2 Linking Directly to the Attachment

To link directly to the attachment, create a link to the attachment record and append the parameter &view=true to the end. For example: 

<a title="My Link" href="sys_attachment.do?sys_id=43f614050a0a0b846fc33c7021c00455&view=true" target="_blank">My Link</a>

The link automatically opens the attachment. This method will only work for attachments that can be read in the browser, for instance image files, for example .jpg, .png, .gif, or document files such as .txt, .html.

3 Opening an Image Link in a New Window

This method inserts an image into an article, which when clicked will open the link in a new window of specified size rather than with default or previously set browser options.

  • In an existing Knowledge base article or a new article select Toggle HTML Source from the toolbar.


  • Add the following script to the article in your desired location:


 <SCRIPT LANGUAGE="JavaScript"> 
 function openindex() { 
 var OpenWindow=window.open("", "newwin","height=450,width=500"); 
 OpenWindow.document.write("<img src='http://www.service-now.com/your_instance/image_name.jpgx' height='450' width='500'>") 
 OpenWindow.document.write("<br>") 
 OpenWindow.document.write("<center><a href='javascript:self.close()' target='_self'>Close</a></center>")} 
 </SCRIPT> 
 <a href="javascript:openindex()" target="_self">
 <img src='http://www.service-now.com/your_instance/image_name.jpgx' border='no' ></a>


  • To change the size of the new open window you change the height and width attributes in the window.open() method. You can also change the image size that appears in the new open window from within the first <img> tag by modifying the height and width attributes and change the size of the image that appears in the knowledge article from within the second <img> tag.
  • From the left navigation pane, select Knowledge Bases --> View, and then select the knowledge article.  Click the image to open it in the new window.


4 Limitations

  1. Going back into the knowledge article to edit it and selecting Toggle HTML Source from the toolbar will cause the script to be returned with special characters modified. As a suggestion, you can keep a local copy of your script and make changes to the copy and past it back in upon edit.
  2. Only one attachment can be downloaded from a knowledge article. If a user attaches more than one attachments to an article, when the attachment link is clicked only one of the attachments will be downloaded, not both.
