1 Overview

The My Amazon Assets and My Team Amazon Assets portals, accessed from the Self-Service application, provide a view of the Amazon-specific assets issued to an individual user or for a user group which the logged in user belongs to. These assets include Amazon resources such as EC2 instances, CloudFormation stack requests, VPCs, and elastic load balancers. The base ServiceNow system provides views of the logged-in user's software entitlements, subscription contracts, and all requests the user has made for company assets. The Amazon resources portals are constructed like ServiceNow homepages and contains familiar controls for moving, adding, or deleting available gauges.

2 Required Roles

You must have the cloud_user role to access the My Amazon Assets and My Team Amazon Assets portals.

3 Using the My Amazon Assets Portal

To access the portal, navigate to Self-Service > My Amazon Assets. The following gauges appear by default:

  • My Amazon Resources: This list report shows all discovered Amazon resources, including EC2 instance, AWS VPC, AWS subnet, AWS Elastic Load Balancer, AWS Elastic Load Balancer listener and AWS Auto Scaling Group.
  • AWS Console Access: Access to the Amazon Web Services console, or a link to request a token to access the console.
  • My Amazon Virtual Resources Metrics: This list shows the number of resources, such as instances and security groups.
  • Resources by Class: This pie chart shows all discovered Amazon resources by different classes.
  • Stacks by Status: This pie chart shows CloudFormation stacks visible to the user by different stack status.
  • Resource Optimization: This bar chart shows how underutilized resources are.
  • My CloudFormation Stacks: This list report shows CloudFormation stacks info assigned to the login user or its assignment group is where the user belong to. The info includes the stack name, updated time, stack status and description.
  • My CloudFormation Stack Requests: This list report shows CloudFormation stack requests info requested by or opened by the login user.

4 Using the My Team Amazon Assets Portal

To access the portal, navigate to Self-Service > My Team Amazon Assets. The same gauges in the My Amazon Assets Portal are available, except that they are for the user's team.
