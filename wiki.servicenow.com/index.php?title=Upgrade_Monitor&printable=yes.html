1 Overview

Upgrade Monitor is a user interface for monitoring the progress of upgrades.

2 Using the Upgrade Monitor

To view the status of the current upgrade, navigate to System Diagnostics > Upgrade Monitor.

The Upgrade Monitor will display whether the upgrade is Complete, Running, or Failed, as well as the previous version and the upgraded version.

If the upgrade is still running, it will display information about its current progress, including the current step information and a progress bar:
