1 Overview

The Dublin Patch 1 Hot Fix 5 release provides fixes for the Dublin release. For more information about how to upgrade your instance, see Requesting an Upgrade.

For the Dublin Patch 1 Hotfix 5 release:

Build date: 11-05-2014_1158
Build tag: glide-dublin-09-13-2013__patch1-hotfix5-11-04-2014
 Note: See how to determine the version on your instance.

// Add the 'Blackberry' catalog item to the cart. 
// OOB record sc_cat_item.do?sys_id=e2132865c0a8016500108d9cee411699

// Submit the Service Request.

Step 3: Consume these two web services concurrently

  • Expected Results: One request with one requested item per each of the web services calls made.
  • Actual Results: Two requests with two requested items each.
