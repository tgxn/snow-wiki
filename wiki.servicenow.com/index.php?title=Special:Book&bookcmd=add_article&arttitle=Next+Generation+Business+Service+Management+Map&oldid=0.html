1 Overview

A ServiceNow business service is work or goods that are supported by an IT infrastructure. For example, delivering email service to an employee can require services such as email servers, web servers, and the work to configure the user's account. A Next Generation Business Service Management (NG-BSM) map graphically displays the configuration items (CI) that support a business service and the relationships between the configuration items.

Many of the relationships in a BSM map are created through the discovery process. You can also create, define, and delete CI relationships in the map. You can display the map from different perspectives and open specific records that relate to configuration items. The system refreshes the map automatically to reflect changes to the CMDB.

The NG-BSM Map application is active in all instances, and includes demo data.

Note: If you are using a version of the ServiceNow platform other than Fuji, see the related documentation in Business Service Management Map.


2 Internet Browser Requirements

The following versions of internet browsers are required for viewing and manipulating NG-BSM maps. If you try to view an NG-BSM map using an older version, Eureka BSM is used instead.

  • Firefox, Firefox/ESR, version 20 or higher (latest version recommended).
  • Chrome version 25 or higher (latest version recommended).
  • Safari version 6 or higher.
  • Microsoft Internet Explorer (IE), with these requirements and limitations:
    • IE version 9 or higher.
    • If you are using Internet Explorer 7 or 8, use the Legacy version of BSM map with Adobe Flash Player 10.1 or higher.
    • You can not export images from NG-BSM map using Internet Explorer as your browser.
    • When you save a map view, NG-BSM will not make a thumbnail image. You may navigate to the saved map view using the version number.

3 Roles

Users with the itil and ecmdb_admin roles can view NG-BSM maps and perform all actions in the map, which includes accessing the map views and filters they have saved, both from the lists in the map and from the Saved Filters module.

4 Menus and Modules

The Next Generation BSM Map application menu contains the following modules.

  • View Map in a New Tab: view the map in a new, full screen tab without the application navigator.
  • View Map: view the BSM map in the content pane of the current tab.
  • Saved Filters: view saved filters for the map.
  • Saved Views: view saved configurations of the map.
  • Map Menu Actions: edit and create context menu options for the map.
  • Map Indicators: configure highlight colors for map nodes that have tasks and outages.
  • Map Related Items: relate referenced CIs to one another.
  • Map Edge Colors: define a map edge color to associate a color with a relationship.
  • Map Icons: replace default icons with custom icons.
  • Map Properties: set properties that control the appearance of a map, including the display colors, level depth, and task thresholds.

5 Enhancements

5.1 Fuji

The Next Generation BSM Map application has been rebuilt using new technology and tools to provide better performance and scalability. When you click the BSM map icon on CI records or on task records that identify a CI, the Next Generation Business Service Management Map opens. You can change the Link to Eureka BSM when navigating via CI Reference Field or Related Items map property to have the Eureka Business Service Management map open instead.
