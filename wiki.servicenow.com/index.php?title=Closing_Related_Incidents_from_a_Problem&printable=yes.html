Functionality described here requires the ITIL role.

1 Overview

Once a Problem is considered resolved, all of the incidents related to the problem with the state Awaiting Problem can be closed using a UI action from the Problem. form.

2 Closing Related Incidents from a Problem

To close related incidents with state Awaiting Problem, right-click the header and select Close Incidents:

3 Customizing Behavior

To customize the behavior of the Close Incidents UI Action:

  1. Navigate to System UI > UI Actions.
  2. Select Close Incidents on the Problem table.
  3. Alter the script as desired.

For instance, removing the following line will close all incidents related to the current problem: 

incident.addQuery("incident_state", "=", 3);
